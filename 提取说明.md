## 🚀 快速使用

### 方式一：使用独立代码包

```bash
# 1. 复制代码包到你的项目
cp -r llm_reference_code/ /path/to/your/project/

# 2. 安装依赖
cd /path/to/your/project/llm_reference_code
pip install openai>=1.0.0 pyyaml>=6.0

# 3. 配置 API
cp example_config.yaml config.yaml
# 编辑 config.yaml，填入你的 API 配置

# 4. 运行示例
python example_usage.py
```

### 方式二：直接使用代码

```python
from llm_reference_code.config import Config
from llm_reference_code.ai_client import AIClient

# 加载配置
config = Config("config.yaml")
client = AIClient(config)

# 同步调用
response = client.chat("你好，请介绍一下人工智能", stream=False)
print(response)

# 流式调用
for chunk in client.chat("请列举3个AI应用场景", stream=True):
    print(chunk, end='', flush=True)
```
