# 2.2.2 标准化入户核查流程（“看、问、查、核”四步法）

#### 2.2.2 标准化入户核查流程（“看、问、查、核”四步法）

一、目标与原则
（一）目标
1. 通过标准化的“四步法”实现对入户对象家庭经济状况的全面、真实、可核验的掌握，确保核查信息的准确性与完整性，形成可追溯的证据链。
2. 实现核查工作在覆盖率、时效性与质量之间的最佳平衡，确保对核心人群的全覆盖与典型案例的深度核验。
3. 以信息化手段支撑核查过程，确保数据在采集、存储、传输、分析各环节的完整性和可追溯性，便于后续审计与监督。

（二）工作原则
1. 法规遵循原则：严格执行国家与地方关于社会救助数据管理、数据共享、个人信息保护等相关法规及地方性实施办法。
2. 数据最小化与隐私保护：仅采集实现核查目的所必需的信息，敏感信息进行脱敏处理，建立数据访问分级与审计追踪。
3. 统一标准、同质化执行：采用统一问卷、统一表单、统一操作规程，确保不同人员、不同时间点核查的一致性与可比性。
4. 风险导向与可追溯性：在关键环节设置控制点，留痕、日志、痕迹链路，确保可追溯性与问题的快速定位。
5. 安全保密与责任到人：签署保密承诺，明确各环节责任人和处置流程，建立纠纷与违规处理的快速响应机制。

二、四步法总体设计与应用范围
（一）总体设计原则
1. 看、问、查、核四步法为核心化流程，覆盖入户核查全生命周期，形成闭环管理。
2. 以现场为基础、以证据为支撑、以信息系统为载体，确保数据的时效性与一致性。
3. 对特殊情形设立备用流程，包含拒访、暴力干扰、无证件等情况，确保安全可控下的继续核查能力。

（二）适用范围
1. 适用于社会救助对象入户核查、动态台账建立、变动核查等情境的标准化执行。
2. 适用于数据比对、上级审核、跨部门协同等前置工作，确保数据的一致性与可比性。

三、看、问、查、核四步的具体操作要点
（一）看 - 现场观察要点
1. 目的与要点：通过直观观察获取家庭居住环境、生活条件、人员构成、日常自理能力等信息，为后续访谈提供背景与佐证。
2. 操作要点：现场安全评估、环境要素记录、覆盖区域与关键区域的照片与清单、与核查目标相关的客观证据留存。
3. 注意事项：遵循隐私保护原则，不在非必要区域进行拍照；遇到安全风险或突发状况，立即停止并按规定上报。
4. 产出物：现场观察要点记录、影像证据清单、现场安全评估记录。

（二）问 - 结构化访谈要点
1. 结构设计：以收入、支出、资产、负债、就业、家庭结构、保障需求等核心维度为框架，设置必答项与必要追问。
2. 访谈技巧与合规：统一话术、语言温和、避免引导和偏见，确保被访者信息真实表达，并把关键表述逐项记录。
3. 取证与佐证：在访谈过程中同步核对并记录原始凭证清单，确保口供与证据的一致性。
4. 产出物：结构化访谈记录、关键表述摘录、证据清单对照表。

（三）查 - 证据核验要点
1. 原始资料核验：对收入凭证、支出票据、资产证明等进行原件对比、真实性核验、逻辑一致性检查。
2. 交叉验证：从多源获取信息，对同一事实进行横向比对，如银行流水、工资单、税务资料、社保信息等。
3. 现场留存：对关键证据在现场留存原件/影印件并进行编号，确保可随时调用与复核。
4. 产出物：证据清单、核验结果表、异常情况记录表。

（四）核 - 结果比对与动态台账更新要点
1. 系统录入与校验：将核查结果准确录入信息系统，进行字段与逻辑校验，触发异常规则并生成整改清单。
2. 动态台账更新：对照历史记录更新家庭基本信息、收支变动、资产负债变动等，形成可追溯的动态台账。
3. 异常处理与复核：对识别出的异常情况，制定整改/复核计划，必要时上级复核或再次现场核验。
4. 产出物：入户核查记录、动态台账、异常处理单、系统日志。

表格：标准化入户核查四步法要点对照表
| 步骤 | 核心要点 | 执行要点 | 产出物 | 责任人 |
| ---- | -------- | -------- | ------ | ------ |
| 看 | 现场环境与居住条件初步判断 | 现场照片清单、环境证据留存、风险提示记录 | 现场观察记录、照片清单 | 入户调查员/现场主管 |
| 问 | 结构化访谈提纲执行 | 使用统一问卷、避免引导、获取关键表述 | 结构化访谈记录、关键表述摘录 | 入户调查员/质控员 |
| 查 | 原始证据真实性核验 | 与多源数据比对、原件/影印件留存 | 证据清单、核验结果表、异常记录 | 入户调查员/资料管理员/复核员 |
| 核 | 结果比对与台账更新 | 系统录入、字段校验、异常处置 | 入户核查记录、动态台账、日志 | 系统管理员/复核员/项目负责人 |

四、数据质量控制与信息系统支撑
（一）数据标准与字段定义
1. 统一数据字典：对收入、支出、资产、负债等字段建立统一定义、取值范围与单位口径，确保跨时间、跨人员的一致性。
2. 数据完整性与一致性检查：对必填字段、格式、逻辑关系设定校验规则，缺失时提供填充策略与预警机制，确保数据完整性。

（二）信息系统支撑
1. 数据录入与校验模块：实现字段必填、格式统一、下拉选项规范、输入控制等，减少人工偏差。
2. 审核与留痕模块：对每次核查操作进行日志留痕，时间戳、操作人、变更记录等可追溯。
3. 动态台账与数据分析模块：建立家庭动态台账、趋势分析、数据可视化和异常告警，支持管理层决策与后续核查。

五、现场操作规范与培训
（一）现场操作规范
1. 着装、仪容、举止：符合职业规范，尊重对象隐私，防护用品按规定使用。
2. 拍照与录音/录像：明确权限范围、取得同意、仅在必要范围内使用，确保隐私保护与合规。
3. 安全与风险管理：现场风险识别、安排行为路线、建立应急处置清单，必要时终止现场访问并上报。
4. 输出物品留痕：对照表单、证据清单、照片清单等清晰标注编号，形成完整留痕链。

（二）培训要求
1. 岗前培训、在岗培训与定期复训，覆盖政策法规、核查流程、数据保护、沟通技巧等内容。
2. 实操演练、情景模拟、案例教学，提升现场应变能力与合规意识。
3. 数据保密与信息保护培训，强调个人信息保护、数据脱敏与权限管理。
4. 培训考核与合格证书发放，建立可追溯的培训档案与资质核验。

六、风险控制与合规性
（一）风险识别
1. 拒访、暴力干扰、证据不足、隐私泄露等情形，需提前制定应对策略与沟通方案。
（二）应对措施
1. 安全保障与风险沟通：设定替代方案、引导性对话路径、必要时请当地相关部门协助。
2. 流程调整与协同：对异常情况启用备用流程，确保核查工作的连续性与合规性。
3. 记录留痕与追溯：完善日志、证据链、系统变更记录，便于事后审计与追责。

八、章节小结
本章围绕标准化入户核查流程的“看、问、查、核”四步法展开，聚焦于现场观察、结构化访谈、证据核验与结果汇总四大环节的统一标准与操作要点，强调在确保合规与安全的前提下，通过信息系统支撑实现数据的完整性、可追溯性与时效性。通过设定统一的表单、明确的证据链与完善的培训机制，确保对入户对象的核查工作具有高度的一致性、可比性和可监控性，降低废标风险并提升评标委员会对方案执行能力的认可度。
