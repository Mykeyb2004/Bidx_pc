# 2.1.3 职能分组设置

为确保项目运行的专业化、精细化和高效化，我们将打破传统的“大锅饭”混岗模式，依据“专业人做专业事”的原则，将驻点项目部划分为三个核心职能小组：**统筹管理组、入户调查组、数据质控组**。各组分工明确，职责清晰，既各司其职又协同作战，形成流水线式的作业闭环。

## 一、 统筹管理组——项目的“大脑中枢”

统筹管理组是整个项目部的指挥中心，负责对内管理、对外协调和后勤保障，确保项目航船方向正确、动力充足。

### 1. 人员配置
*   **组长**：由**项目经理**兼任，拥有最终决策权。
*   **成员**：配备**行政主管**1名，**财务专员**1名。
*   **素质要求**：具备较强的沟通协调能力、公文写作能力和突发事件应对能力。

### 2. 主要职责
1.  **对外联络协调**：作为与通山县民政局及各乡镇民政办对接的唯一官方出口，负责接收采购人的指令、任务清单，并反馈工作进度和重难点问题；协调解决入户过程中遇到的“门难进、脸难看”等外部阻力。
2.  **制度建设与监督**：制定和修订项目部的各项管理制度（考勤、保密、绩效等），并监督各组执行情况。定期召开项目例会，通报问题，部署工作。
3.  **后勤保障服务**：负责办公场所的日常运维、车辆调度、办公用品采购、食宿安排等，为一线调查员解除后顾之忧。
4.  **财务资金管理**：负责项目资金的独立核算、工资奖金发放、费用报销审核，配合第三方审计。
5.  **应急处突指挥**：在发生车辆趴窝、人员受伤、暴力抗法等突发事件时，第一时间启动应急预案，赶赴现场处置。

## 二、 入户调查组——项目的“铁脚板”

入户调查组是项目部的主力军和核心力量，直接面对困难群众，其工作质量直接决定了基础数据的真实性和准确性。

### 1. 人员配置与分组
*   **规模**：根据通山县乡镇分布和工作量，拟组建**5个入户调查小分队**。
*   **结构**：每小分队**2-3人**。为方便与女性申请人沟通及降低防备心理，坚持**“男女搭配”**原则，每组至少配备1名女性调查员。
*   **网格化包保**：实行“分片包干”制，每个小分队固定负责2-3个乡镇。这样有利于调查员熟悉当地地形、风土人情和村干部情况，提高核查效率。

### 2. 主要职责
1.  **实地入户核查**：严格按照“四步法”（看、问、查、核）开展入户调查。
    *   **看**：看居住环境、家用电器、生活用品、精神面貌。
    *   **问**：问健康状况、就业情况、子女上学、赡养能力。
    *   **查**：查身份证、户口本、残疾证、病历单、务工证明。
    *   **核**：核算家庭人均纯收入，核对家庭财产填报是否属实。
2.  **邻里外围走访**：对每一户申请家庭，必须走访至少2户邻居或村民小组长，侧面了解其真实生活水平和隐形收入情况，并在调查表上记录旁证人联系方式。
3.  **政策宣传讲解**：在入户过程中，向群众发放宣传单，耐心解答低保、特困等救助政策疑问，当好“政策宣讲员”。
4.  **现场数据采集**：使用移动终端（平板/手机）现场拍摄人脸照片、房屋全景、证件原件，录制访谈录音，并引导申请人进行电子签名确认。

## 三、 数据质控组——项目的“守门员”

数据质控组是确保数据“零差错”的关键防线，在这个“大数据”时代，他们负责将前方采集的繁杂信息转化为规范、精准的数据资产。

### 1. 人员配置
*   **规模**：配备**3名**专职质控人员（可根据不同阶段工作量，由内勤人员统筹调配）。
*   **素质要求**：具备极强的责任心、细致耐心，熟练掌握Office办公软件、PhotoShop及社会救助信息系统操作。

### 2. 主要职责
1.  **100%逻辑复核**：对入户调查组每天交回的纸质表格进行全覆盖审核。重点审查：
    *   **完整性**：必填项是否有空缺，签字、手印是否齐全。
    *   **逻辑性**：家庭人口数与户口本是否一致，收入计算公式是否正确，支出与收入是否严重背离（如收入极低但有高额刚性支出说明可能有隐形收入）。
    *   **一致性**：纸质表记录与现场录音、照片反映的情况是否吻合。
2.  **系统录入与维护**：将审核无误的数据准确录入“湖北省社会救助信息系统”或县自建系统。确保录入速度跟上调查进度，杜绝积压。
3.  **档案归档管理**：按照“一户一档”标准，将《入户调查表》、《授权书》、《核对报告》、证件复印件等资料分类整理、打码、装订、入柜。建立电子档案目录索引，确保存放有序、查找方便。
4.  **数据分析报告**：定期（月度/季度）对核查数据进行统计分析，通过图表形式展示各乡镇的核查率、通过率、主要致贫原因分布等，为民政局决策提供数据支撑。

## 四、 组间协作与流转机制

三个小组虽然职能不同，但在业务流程上是紧密咬合的齿轮。

1.  **任务分发**：统筹管理组接收任务 -> 按片区派发给入户调查组。
2.  **实地作业**：入户调查组完成现场核查 -> 每日下班前移交原始资料给数据质控组。
3.  **质控反馈**：数据质控组当天/次日完成审核 -> 合格的归档录入；**不合格的（信息缺失、逻辑错误）填写《退回补正单》，退回入户调查组重新核实**。
4.  **闭环汇报**：统筹管理组汇总各组数据 -> 向采购人提交正式核查报告。

通过这种“流水作业、层层把关、相互制衡”的职能分组设置，彻底改变过去“一人包干到底”、既当运动员又当裁判员的弊端，最大限度地降低廉政风险，提升服务质量。
