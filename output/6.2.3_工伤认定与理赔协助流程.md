# 6.2.3 工伤认定与理赔协助流程

#### 6.2.3 工伤认定与理赔协助流程

在社会救助服务项目的实际运营中，一线社工与核查人员需高频次深入街道、乡镇及村居进行入户调查、政策宣传与个案服务，面临着交通意外、意外伤害（如入户过程中的犬只咬伤、跌倒）、突发疾病以及潜在的服务对象情绪激化冲突等职业风险。杭州菲尔德经济信息咨询有限公司（以下简称“菲尔德咨询”）始终秉持“员工至上、依法合规、高效保障”的原则，依据《工伤保险条例》、《社会保险法》及地方性法规，建立了一套严密、规范、充满人文关怀的工伤认定与理赔协助全流程体系。本流程旨在确保当员工不幸遭遇工伤事故时，能够得到最快速的法律认定、最充分的经济赔偿与最温暖的组织关怀，消除员工后顾之忧，保障项目团队的稳定性与战斗力。

**一、 总体原则与组织保障**

**（一） 核心处置原则**
1.  **生命优先原则：** 在工伤事故发生的第一时间，首要任务是救治伤员，理赔取证工作绝不应干扰医疗救治。
2.  **依法合规原则：** 所有认定申报材料、理赔流程严格遵循国家及项目所在地人力资源和社会保障部门（以下简称“人社局”）的规定，确保程序的合法性与有效性。
3.  **时效控制原则：** 严格遵守工伤认定申报的“黄金时效”（事故发生后30日内），杜绝因拖延导致认定权益受损。
4.  **全程代办原则：** 菲尔德咨询承诺实行“保姆式”理赔服务，由公司专职人员负责跑腿、填表、沟通，最大限度减少受伤员工及其家属的事务性负担。

**（二） 专项工作小组职责**
公司成立由项目负责人任组长，行政人事部经理任副组长，法务专员及项目主管为成员的“工伤处理专项工作小组”。
1.  **项目负责人：** 统筹工伤处理全局，负责重大工伤事故的应急指挥与资金调度。
2.  **人事专员（工伤专办员）：** 具体负责工伤认定材料的收集、整理、申报，与社保中心及商业保险公司的对接。
3.  **法务专员：** 提供法律支持，审核申报材料的严谨性，处理可能出现的行政复议或诉讼，保障公司与员工的合法权益。
4.  **部门主管：** 负责现场证据保全、目击证人寻找及协助家属进行医疗陪护对接。

**二、 工伤事故的界定与前期响应**

为确保理赔协助的精准性，菲尔德咨询首先明确社会救助服务场景下的工伤界定范围，并实施快速响应。

**（一） 社会救助项目常见工伤情形界定**
依据《工伤保险条例》第十四条、第十五条，结合本项目特点，重点涵盖以下情形：
1.  **工作场所内的伤害：** 在街道办事处、民政局驻点办公室或档案室工作期间，因工作原因受到的事故伤害（如搬运物资砸伤、办公设备漏电击伤）。
2.  **工作时间前后的预备性工作伤害：** 在工作场所内，从事与工作有关的预备性或者收尾性工作受到事故伤害的。
3.  **因工外出期间的伤害：** 员工进行低保核查、特困探访、街面巡查期间，由于工作原因受到伤害（如入户途中摔伤、被服务对象家养宠物咬伤、遭遇意外暴力抗法等）。
4.  **上下班途中的交通事故：** 在上下班的规定时间与必经路线上，受到非本人主要责任的交通事故或城市轨道交通、客运轮渡、火车事故伤害的。
5.  **突发疾病死亡或抢救无效：** 在工作时间和工作岗位，突发疾病死亡或者在48小时之内经抢救无效死亡的。

**（二） 事故发生后的即时响应机制**
1.  **现场急救与送医：** 事故发生后，现场同事或主管立即拨打120，将伤者送往就近的工伤定点医疗机构（情况紧急可送往最近医院，待病情稳定后转入定点医院）。
2.  **事故上报（1小时内）：** 现场人员在确保安全的前提下，通过电话向项目负责人及人事部门口头汇报事故简况（时间、地点、人物、受伤程度、初步原因）。
3.  **证据保全（24小时内）：**
    *   **影像留存：** 拍摄事故现场照片、视频，重点记录致害物、周围环境、伤者位置。
    *   **证人锁定：** 登记目击者（如同行社工、村居干部、周边群众）的姓名与联系方式，必要时录音取证。
    *   **官方记录：** 涉及交通事故的，必须立即报警并要求出具《道路交通事故责任认定书》；涉及暴力伤害的，必须报警并要求出具《接处警记录》或《验伤报告》。

**三、 工伤认定行政申报流程（社保端）**

这是工伤理赔的核心环节，菲尔德咨询将按照“严谨、细致、快速”的标准执行。

**（一） 申报材料准备阶段（事故后3个工作日内）**
工伤专办员协助伤者及家属准备以下核心材料，并进行合规性审核：
1.  **《工伤认定申请表》：** 由公司如实填写，加盖公章。填写内容需与医疗记录、事故证明高度一致，避免描述性矛盾。
2.  **劳动关系证明：** 提供盖章的《劳动合同》复印件。若合同未在身边，提供工资流水、社保缴纳记录、工作证、考勤记录等作为辅助证据。
3.  **医疗诊断证明：** 收集初诊病历、急诊病历、住院病历首页、出院小结、医疗诊断书、X光/CT报告单等。**特别注意：** 病历中关于受伤原因的描述必须是“工作中受伤”或“车祸”，严禁出现“在家摔倒”等与事实不符的描述，以免造成认定障碍。
4.  **事故证明材料：**
    *   **交通事故：** 提供交警部门出具的《道路交通事故认定书》（需注明非本人主要责任）。
    *   **因工外出：** 提供公司出具的《因工外出证明》、派工单、核查任务记录表、签到表等，证明其当时处于工作状态。
    *   **暴力伤害：** 提供公安机关出具的证明材料或法院判决书。
    *   **证人证言：** 两名以上目击证人的书面证言及身份证复印件。
5.  **身份证明：** 受伤员工身份证复印件。

**（二） 正式申报与受理阶段（事故后30日内）**
1.  **窗口递交：** 工伤专办员携带全套材料前往项目所在地的人社局工伤保险科窗口进行申报。菲尔德咨询承诺，除非特殊不可抗力，所有工伤事故必须在事故发生之日起30日内完成申报，绝不因公司原因导致申报逾期。
2.  **补正材料：** 若社保部门提出材料补正要求，专办员需在《补正材料通知书》规定的时限内（通常为15日）补齐所有材料。
3.  **取得受理通知：** 确保拿到社保部门出具的《工伤认定申请受理通知书》，标志着工伤认定程序正式启动。

**（三） 配合调查与认定决定阶段**
1.  **配合核实：** 在社保部门调查人员进行实地核查或电话回访时，公司相关人员（目击者、主管）需实事求是地配合调查，提供真实口供。公司法务专员将提前对相关人员进行流程辅导，确保陈述的准确性与一致性。
2.  **领取决定书：** 一般情况下，社保部门自受理工伤认定申请之日起60日内作出工伤认定决定。专办员领取《认定工伤决定书》后，第一时间复印一份交予员工或家属，原件归档备用。

**四、 商业保险理赔协助流程（补充保障）**

为构建多层次保障体系，菲尔德咨询为本项目所有员工投保了雇主责任险和团体意外伤害险。商业保险理赔与工伤社保理赔并行处理，互为补充。

**（一） 报案与立案**
1.  **即时报案：** 事故发生后24小时内，工伤专办员拨打保险公司客服电话进行报案，获取报案号。
2.  **信息备案：** 将事故经过、伤者信息、初步诊断发送给保险公司专属客户经理，建立理赔档案。

**（二） 单证收集与提交**
商业保险理赔通常需要等待工伤认定结果或治疗终结后进行，但材料收集需同步进行：
1.  **医疗票据：** 收集所有医疗费发票原件（若社保已报销，则提供分割单原件）、费用清单、处方笺。
2.  **社保结算单：** 经医保或工伤保险基金结算后的结算单据。
3.  **认定文件：** 《认定工伤决定书》复印件。
4.  **其他材料：** 银行卡复印件、身份证复印件、索赔申请书。

**（三） 理赔款项支付**
1.  **跟踪进度：** 专办员每周跟进保险公司理赔审核进度，解答保险公司关于事故性质或用药合理性的质询。
2.  **款项到账：** 确保理赔款项直接打入员工个人账户，或由公司垫付后打入公司账户再转付员工。公司承诺不截留、不挪用任何商业保险理赔款。

**五、 医疗救治与停工留薪期管理**

在员工治疗与康复期间，菲尔德咨询将严格落实法律规定的待遇保障，确保员工安心养伤。

**（一） 医疗救治管理**
1.  **定点就医：** 引导员工在工伤定点医疗机构进行治疗。
2.  **费用垫付：** 对于工伤认定决定下达前的医疗费用，若员工家庭经济困难，公司可启动“员工关爱基金”先行垫付，待工伤报销后冲抵。
3.  **用药合规提醒：** 提醒员工及家属，尽量使用工伤保险目录内的药品和诊疗项目，避免产生高额自费项目。对于必须使用的自费药，需签署知情同意书，并尝试通过商业保险覆盖。

**（二） 停工留薪期待遇**
1.  **工资照发：** 依据《工伤保险条例》第三十三条，员工因工作遭受事故伤害或者患职业病需要暂停工作接受工伤医疗的，在停工留薪期内，原工资福利待遇不变，由菲尔德咨询按月支付。
2.  **期限确定：** 停工留薪期一般不超过12个月。具体期限根据医疗机构出具的诊断证明和休假建议确定。伤情严重或者情况特殊，经设区的市级劳动能力鉴定委员会确认，可以适当延长，但延长不得超过12个月。
3.  **护理费支付：** 生活不能自理的工伤员工在停工留薪期需要护理的，由公司负责。公司可指派专人护理，或按照当地护工市场价格支付护理费。

**六、 劳动能力鉴定与待遇申领流程**

对于伤情较重、可能存在功能障碍的员工，待病情相对稳定后，启动劳动能力鉴定程序。

**（一） 劳动能力鉴定申请**
1.  **时机把握：** 停工留薪期满或伤情稳定后，由工伤专办员协助员工向市级劳动能力鉴定委员会提出申请。
2.  **材料提交：** 提交《劳动能力鉴定申请表》、《认定工伤决定书》、完整的病历资料、检查报告等。
3.  **现场鉴定：** 通知员工按时到达指定地点接受专家鉴定。对于行动不便的员工，公司派车接送。

**（二） 伤残待遇申领与发放**
根据鉴定出的伤残等级（一级至十级），协助员工申领相应待遇：
1.  **一次性伤残补助金：** 由工伤保险基金支付，标准根据伤残等级和本人工资计算。公司专办员负责填报《工伤保险待遇申领表》并跟进到账情况。
2.  **一次性工伤医疗补助金：** 解除劳动关系时由工伤保险基金支付（如适用）。
3.  **一次性伤残就业补助金：** 解除劳动关系时由公司支付（如适用）。
4.  **按月领取的伤残津贴：** 针对一至四级伤残员工，按月发放伤残津贴，保留劳动关系，退出工作岗位。

**七、 人文关怀与心理疏导机制**

作为民政领域的服务机构，菲尔德咨询深知工伤事故不仅带来身体创伤，更可能带来心理冲击（如遭遇暴力抗法后的PTSD）。因此，我们将心理干预纳入理赔协助流程。

**（一） 情感支持与探视**
1.  **第一时间慰问：** 事故发生后24小时内，项目负责人代表公司前往医院或家中探视，送去慰问金及慰问品，安抚家属情绪，表明公司负责到底的态度。
2.  **定期回访：** 住院期间每周一次电话问候，每月一次实地探视，了解治疗进度与实际困难，解决家属在陪护期间的食宿等后顾之忧。

**（二） 心理危机干预**
1.  **专业评估：** 针对因暴力抗法、严重车祸导致工伤的员工，公司邀请具备国家二级心理咨询师资质的督导进行心理创伤评估。
2.  **个案辅导：** 对于出现焦虑、恐慌、失眠等应激反应的员工，提供免费的心理咨询服务，帮助其重建安全感，走出心理阴影。

**（三） 返岗复工支持**
1.  **岗位调整：** 对于因工伤导致身体机能下降、无法胜任原岗位的员工（如无法承担高强度入户核查），坚持“不抛弃、不放弃”原则，在征求员工意见后，转岗至档案管理、电话客服、数据录入等体力负荷较轻的岗位。
2.  **复工适应：** 员工返岗首月设为“适应期”，适当降低工作考核指标，安排资深同事进行“一对一”帮扶，帮助其平稳回归职场。

**八、 档案管理与案例复盘**

**（一） “一案一档”管理**
建立严格的工伤案件档案管理制度。每个工伤案件均建立独立档案，内容包含：
1.  事故报告与现场证据资料；
2.  工伤认定申请与决定文件；
3.  医疗病历与费用单据复印件；
4.  劳动能力鉴定结论；
5.  待遇赔付凭证与银行回单；
6.  员工签收确认函。
档案保存期限不少于10年，确保随时可查，规避法律风险。

**（二） 事故复盘与预防改进**
工伤处理完毕后10个工作日内，由项目负责人主持召开“事故复盘分析会”：
1.  **原因分析：** 深入剖析事故发生的根本原因（是流程漏洞、培训不足、防护装备缺失还是个人疏忽）。
2.  **整改措施：** 针对性制定整改方案。例如，若是入户被狗咬伤，则需采购驱狗器并加强防犬培训；若是交通事故，则加强交通安全教育或调整出行方式。
3.  **制度优化：** 将案例脱敏后作为内部培训教材，警钟长鸣，从源头上降低工伤事故发生率，实现从“事后理赔”向“事前预防”的跨越。

通过上述八大板块的精细化运作，菲尔德咨询构建了一个既有“法度”又有“温度”的工伤认定与理赔协助体系。我们承诺，在本项目执行期间，将以最专业的态度、最高效的执行力，守护每一位为社会救助事业奉献的员工，确保项目团队始终保持旺盛的生命力与凝聚力，为采购人交出一份满意的答卷。
