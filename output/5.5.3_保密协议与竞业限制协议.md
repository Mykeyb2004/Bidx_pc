# 5.5.3 保密协议与竞业限制协议

#### 5.5.3 保密协议与竞业限制协议

在社会救助服务项目中，数据安全与隐私保护不仅是法律底线，更是政治红线。作为专业的社会救助第三方服务机构，杭州菲尔德经济信息咨询有限公司（以下简称“菲尔德咨询”）深知，我们接触的是全区（县）最困难群体的核心隐私数据（包括但不限于身份证号、银行账户、残疾状况、重大疾病史、家庭收入结构等），以及民政部门尚未公开的政策研判数据。因此，建立严苛的保密协议与竞业限制协议体系，是保障项目安全运行、维护政府公信力、保护困难群众尊严的基石。

本章节详细阐述菲尔德咨询在本项目中实施的保密与竞业限制管理制度，通过法律文本约束、全员签署、全周期管理及技术防范，构建起一道坚不可摧的信息安全“防火墙”。

一、 总体原则与法律依据

（一） 管理目标
菲尔德咨询坚持“涉密必保、违规必究、全员覆盖、终身追责”的十六字方针。我们的目标是实现项目执行期间“零泄密、零滥用、零流失”，确保每一份困难群众的档案都在受控状态下流转，确保每一位核心员工都严守职业操守。

（二） 法律与政策依据
本制度及相关协议文本严格依据《中华人民共和国民法典》、《中华人民共和国数据安全法》、《中华人民共和国个人信息保护法》、《社会救助暂行办法》以及民政部关于社会救助家庭经济状况核对的相关保密规定制定。同时，结合采购单位对数据安全的具体要求，形成具有法律效力的内部管控文件。

二、 保密协议体系设计

保密协议并非一张简单的承诺书，而是涵盖数据全生命周期的法律契约。菲尔德咨询针对社会救助项目的特殊性，实行分级分类的保密协议签署机制。

（一） 保密协议的分级签署
考虑到不同岗位接触数据的深度与广度不同，我们将保密协议细分为三个等级，确保责权对等。

1. **核心涉密人员保密协议**
适用对象：项目负责人、数据管理员、档案管理员、系统维护工程师。
协议重点：此类人员掌握全量数据库及系统后台权限。协议条款涵盖最高级别的保密义务，包括严禁将数据带离涉密场所、严禁私自备份、严禁在非涉密网络传输数据等。违约金设定为最高档次，并附带严厉的法律追诉条款。

2. **一线服务人员保密协议**
适用对象：入户调查员、街道/乡镇驻点社工、窗口受理人员。
协议重点：此类人员直接接触服务对象及其纸质档案。协议重点约束“入户过程中的隐私保护”（如不得在该户邻居面前大声谈论病情）、“纸质资料的保管”（如不得遗失、拍照上传朋友圈）、“口头保密”（如不得将服务对象隐私作为谈资）。

3. **外部协作人员保密协议**
适用对象：临时聘用的专家、志愿者、实习生及第三方系统供应商。
协议重点：强调“单次授权、用完即毁”原则，明确其仅对特定任务期间接触的信息负有保密义务，且必须在任务结束后立即销毁相关数据副本。

（二） 保密协议的核心条款
我们的保密协议文本经过专业法务团队与民政领域专家的双重审核，确保条款严密、无懈可击。

1. **保密信息的定义**
明确界定本项目中的保密信息范围，包括但不限于：
（1）服务对象及其家庭成员的个人身份信息、生物识别信息、健康生理信息、财产状况信息；
（2）采购单位提供的社会救助底册、政策草案、会议纪要、内部统计数据；
（3）菲尔德咨询在项目执行过程中形成的调查报告、评估结果、分析图表、工作台账；
（4）项目相关的技术方案、算法逻辑、管理软件账号密码等。

2. **保密义务与行为规范**
协议详细列明了“十不准”行为规范：
（1）不准在私人交往和通信中泄露涉密内容；
（2）不准在公共场所谈论涉密事项；
（3）不准通过普通邮寄或非加密网络传递涉密载体；
（4）不准私自复制、保存和销毁涉密载体；
（5）不准在非涉密计算机上处理涉密信息；
（6）不准将涉密存储介质带出工作区域；
（7）不准在未采取保密措施的情况下对涉密内容进行数字化处理；
（8）不准向任何无关人员（包括公司内部非项目组成员）透露涉密信息；
（9）不准利用职务之便查询与工作无关的家庭经济状况信息；
（10）不准利用掌握的困难群众信息进行任何形式的商业推销或诈骗活动。

3. **保密期限**
协议明确规定，保密义务不因员工离职、项目结束或合同终止而解除。对于一般性工作秘密，保密期限为项目结束后5年；对于涉及个人核心隐私及国家秘密的信息，实行“终身保密制”。

三、 竞业限制协议的深度应用

在社会救助领域，竞业限制不仅是为了保护公司的商业机密，更是为了防止因人员流动导致政府数据外泄或服务中断。菲尔德咨询针对关键岗位实施严格的竞业限制制度。

（一） 竞业限制的适用范围
并非全员签署，而是聚焦于掌握核心资源与技术的关键少数：
1. **高级管理人员**：掌握公司整体运营策略及全区民政业务布局的人员。
2. **高级技术人员**：掌握社会救助信息化平台架构、核对算法模型、数据接口标准的人员。
3. **关键业务骨干**：掌握全区困难群众详细底册及特殊个案处理逻辑的项目主管。

（二） 竞业限制的具体内容
1. **限制行为**
在劳动关系存续期间及解除劳动合同后的一定期限内，相关人员不得：
（1）自营或为他人经营与菲尔德咨询及本项目同类的业务；
（2）到与菲尔德咨询有竞争关系的机构任职；
（3）利用在本项目中积累的客户渠道（即民政局及街道乡镇关系）和数据资源进行个人牟利；
（4）诱导公司其他员工离职或向竞争对手提供服务。

2. **限制地域与期限**
根据项目特性，竞业限制地域通常设定为本项目所在的行政区划及周边关联区域。限制期限严格遵守劳动合同法规定，一般为员工离职后两年。在此期间，公司将按月支付竞业限制经济补偿金，确保协议的法律效力。

3. **违约责任**
若员工违反竞业限制义务，除必须退还已领取的经济补偿金外，还需支付高额违约金（通常为离职前12个月总收入的2-3倍），并赔偿因其违约行为给公司及采购单位造成的全部直接和间接损失。

四、 协议签署与执行流程

制度的生命力在于执行。菲尔德咨询建立了一套标准化的协议签署与管理流程，确保每一份协议都合法、有效、可追溯。

（一） 入职即签，先签后岗
所有拟参与本项目的员工，在办理入职手续的第一环节即为签署《保密协议》。对于关键岗位，同步签署《竞业限制协议》。人力资源部负责对协议条款进行逐条解读，确保员工充分理解其法律后果。未签署协议者，一律不得进入项目组，不得开通系统账号，不得接触任何业务档案。

（二） 全程留痕，动态管理
1. **档案专管**：所有签署的协议原件统一归档至公司机要室，建立电子台账，实行双人双锁管理。
2. **变更续签**：当员工岗位发生调整（如从普通社工晋升为数据主管），涉密等级提高时，必须在调岗当日补签更高等级的保密协议。
3. **定期复核**：每半年对项目组全员的协议签署情况进行一次自查，确保无遗漏、无过期。

（三） 离职脱密，闭环管控
员工提出离职申请后，立即启动脱密程序：
1. **权限回收**：IT部门在1小时内冻结其所有系统账号、VPN访问权限及门禁权限。
2. **设备清查**：收回其使用的电脑、U盘、录音笔等设备，并由技术人员进行专业的数据擦除与格式化处理。
3. **资料移交**：在督导人员监交下，移交所有纸质档案、工作笔记及电子文档，签署《资料移交清单》。
4. **离职谈话**：项目负责人与离职员工进行保密谈话，重申离职后的保密义务及竞业限制要求，并签署《离职后保密承诺书》。

五、 配套保障措施与技术防范

仅靠一纸协议是不够的，菲尔德咨询通过“技术+管理”的双重手段，将保密与竞业限制要求落实到日常工作的每一个细节。

（一） 物理环境安全管控
1. **档案室管理**：项目档案室按照国家秘密载体管理标准建设，安装防盗门、铁窗、监控探头及红外报警系统。实行“双人双锁”制度，进出必须登记，严禁携带手机、相机等摄录设备进入。
2. **办公区隔离**：设立独立的社会救助项目办公专区，与公司其他业务物理隔离。非项目组成员未经授权不得进入，访客接待必须在公共会议室进行。

（二） 信息系统技术防范
1. **数据加密**：所有存储困难群众信息的数据库均采用国密算法进行加密存储，敏感字段（如身份证号、手机号）实行脱敏展示。
2. **网络隔离**：项目业务网络与互联网逻辑隔离，严禁在连接互联网的电脑上处理涉密数据。
3. **行为审计**：部署终端安全管理系统（DLP），对员工电脑的USB接口进行管控（仅允许使用公司配发的加密U盘）。系统自动记录所有文件的创建、复制、打印、外发行为，一旦发现违规操作（如尝试将大批量数据通过邮件发送），系统自动拦截并向管理员报警。
4. **数字水印**：内部流转的电子文档及系统界面均强制加载含有员工姓名及工号的隐形数字水印。一旦发生截屏泄露，可立即溯源定位到责任人。

（三） 培训与文化建设
1. **岗前培训**：将保密教育作为新员工入职培训的第一课，通过剖析社会救助领域的泄密案例（如某地低保名单泄露引发的舆情事件），敲响警钟。
2. **常态化宣教**：每月召开一次保密工作例会，学习最新的数据安全法规；在办公区域张贴保密警示标语；每季度组织全员进行保密知识闭卷考试，考试不合格者暂停工作资格。

六、 违约处理与责任追究

菲尔德咨询对违反保密及竞业限制协议的行为持“零容忍”态度，建立了严厉的追责机制。

（一） 内部行政处罚
对于轻微违规行为（如未锁屏离开工位、在公共办公区大声讨论个案），虽未造成实质性泄密，但存在安全隐患的，给予口头警告、通报批评、扣发当月绩效奖金等处罚，并强制重修保密课程。

（二） 解除劳动合同
对于严重违规行为（如私自拷贝数据回家、将账号借给他人使用、违反竞业限制条款），公司将依据《劳动合同法》第三十九条及内部规章制度，立即解除劳动合同，不予支付任何经济补偿，并列入行业黑名单。

（三） 法律追诉
对于造成泄密事故、给采购单位或服务对象造成损失的，公司将启动法律程序：
1. **民事赔偿**：通过仲裁或诉讼，追讨违约金及全额赔偿金。
2. **刑事报案**：对于情节严重、涉嫌侵犯公民个人信息罪或故意泄露国家秘密罪的，公司将固定证据，直接移送公安机关依法追究刑事责任。

七、 针对本项目的特别承诺

鉴于本项目作为民政社会救助服务的特殊性质，菲尔德咨询在此郑重承诺：

1. **数据所有权归属**：本项目产生的所有数据、档案、报表及系统成果，其所有权完全归属于采购单位（民政局）。菲尔德咨询仅拥有在合同期内的使用权与管理权。项目结束后，我们将完整移交所有数据，并彻底清除公司内部留存的任何副本，绝不留存“私房数据”。
2. **第三方连带责任**：若因我方聘请的第三方（如系统开发商、审计机构）原因导致泄密，菲尔德咨询自愿承担全部连带赔偿责任，先行赔付采购单位及受害群众，再向第三方追偿。
3. **主动接受监管**：我们无条件接受采购单位及相关保密行政管理部门的定期检查与突击抽查。对于检查中发现的任何保密隐患，承诺在24小时内整改到位。

通过上述严密的保密协议与竞业限制管理体系，菲尔德咨询将为本项目打造一个“铜墙铁壁”般的信息安全环境，让政府放心，让群众安心，确保社会救助工作在阳光、安全、规范的轨道上高效运行。
