# 1.2.4 数据保密承诺

#### 1.2.4 数据保密承诺

**一、 总体保密原则与安全观**

社会救助工作涉及大量困难群众的个人隐私信息（包括但不限于身份证号、家庭住址、联系方式、健康状况、残疾类别）以及家庭经济状况信息（包括银行存款、房产车辆、社保纳税等敏感数据）。作为本次项目的投标人，**杭州菲尔德经济信息咨询有限公司**（以下简称“我司”）深刻认识到，数据安全与隐私保护不仅是技术问题，更是严肃的政治问题和民生底线。

我司郑重承诺：在本项目全生命周期内，将严格遵守《中华人民共和国数据安全法》、《中华人民共和国个人信息保护法》、《社会救助暂行办法》及采购人所在地的相关保密规定，坚持“**预防为主、分级管控、全程留痕、责任到人**”的原则，构建“**制度+技术+人员**”三位一体的立体化保密防御体系，确保项目数据**不泄露、不滥用、不丢失、不被非法篡改**，坚决筑牢民政社会救助领域的信息安全防火墙。

**二、 保密管理组织架构与责任体系**

为确保保密工作落到实处，我司将成立本项目专属的“数据安全与保密工作领导小组”，实行垂直化管理，确保责任链条无断点。

**（一） 组织架构设置**
1.  **保密工作组组长**：由本项目负责人担任，作为数据安全的第一责任人，对项目整体保密工作负总责，负责签署保密责任书，审批数据使用权限。
2.  **保密专员（安全管理员）**：由项目团队核心成员担任，负责日常保密检查、账号权限管理、日志审计及保密教育培训的组织实施。
3.  **技术防护员**：负责办公网络环境的安全维护、防病毒软件更新、漏洞修复及数据备份工作。
4.  **资料管理员**：负责纸质档案的入库、借阅、归还及销毁登记管理，确保“账物相符”。

**（二） 责任落实机制**
1.  **全员签署保密协议**：项目启动前，我司将与所有参与项目的人员（包括项目经理、社工、调查员、数据录入员、驾驶员等）逐一签署《个人保密承诺书》及《竞业限制协议》，明确法律责任。
2.  **连带责任制**：实行“谁主管、谁负责；谁使用、谁负责；谁经手、谁负责”的责任追究机制。若发生数据泄露事件，除追究直接责任人法律责任外，项目负责人承担连带管理责任。
3.  **离职脱密管理**：项目人员离职或调岗前，必须进行脱密处理，回收所有账号权限、门禁卡及相关纸质、电子资料，并签署《离职保密承诺书》，承诺离职后仍对知悉的项目秘密承担保密义务。

**三、 全生命周期数据安全管理方案**

我司将针对社会救助业务特点，对数据采集、传输、存储、处理、交换、销毁等六个环节实施全流程闭环管控。

**（一） 数据采集安全管控**
1.  **最小化采集原则**：严格按照采购人及社会救助政策规定的指标进行信息采集，不收集与项目无关的个人隐私信息。
2.  **入户设备管控**：入户调查员使用的移动终端（平板电脑或手机）必须安装指定的安全防护软件。采集的电子数据（如照片、录音）必须存储在加密区域，禁止通过微信、QQ等社交软件直接传输原始档案图片。
3.  **纸质表单保护**：入户调查填写的《家庭经济状况核对授权书》、《入户调查表》等纸质材料，必须放入专用密封袋携带，严禁随意放置在车辆显眼处或公共场所，确保当日采集、当日归档。

**（二） 数据传输安全管控**
1.  **内网传输优先**：所有敏感数据的上传下达，原则上通过采购人指定的政务内网或专用业务系统进行。
2.  **加密传输通道**：若必须通过互联网传输数据，必须使用VPN（虚拟专用网络）或加密压缩包形式。严禁在互联网网盘、未加密的电子邮件中存储或发送含有公民个人信息的明文文档。
3.  **介质传递规范**：确需使用移动存储介质（U盘、移动硬盘）拷贝数据的，必须使用经我司登记备案并经过病毒查杀的专用加密介质。严禁在涉密计算机与互联网计算机之间交叉使用移动存储介质。

**（三） 数据存储安全管控**
1.  **分级分类存储**：将数据分为“核心敏感数据”（如身份证号、银行账号、残疾类型）、“一般业务数据”和“公开数据”。核心敏感数据实行加密存储，数据库须部署审计系统。
2.  **物理环境安全**：项目组办公区域设置独立的档案室或保密柜，实行门禁管理，安装24小时高清视频监控，监控录像保存不少于90天。
3.  **备份恢复机制**：建立“本地+异地”双重备份机制。每日增量备份，每周全量备份。每季度进行一次数据恢复演练，确保在系统故障或勒索病毒攻击下，数据能快速恢复，保障业务连续性。

**（四） 数据处理与使用管控**
1.  **权限最小化**：依据岗位职责严格分配系统权限。普通录入人员仅能查看和编辑自己负责区域的数据；审核人员可查看完整档案；系统管理员不得直接接触业务数据内容（实行“三权分立”）。
2.  **数据脱敏使用**：在进行项目汇报、案例分析、督导培训或接受外部检查时，涉及的具体服务对象信息必须进行脱敏处理（如隐去姓名、身份证号中间位数、模糊具体住址），严禁在PPT演示或公开文档中展示未经脱敏的敏感信息。
3.  **屏幕水印技术**：项目组办公电脑部署屏幕水印系统，显示操作员的姓名及时间戳，一旦发生拍照泄露，可立即溯源追责。
4.  **禁止非法复制**：办公电脑禁用USB写入功能或部署终端安全管理软件，阻断未经授权的数据导出行为。

**（五） 数据销毁安全管控**
1.  **电子数据销毁**：项目结束或设备报废时，对存储过项目数据的硬盘、存储卡进行多次覆写清除或消磁处理，确保数据不可恢复。严禁直接将未处理的存储设备作为二手设备转售或丢弃。
2.  **纸质档案销毁**：对于废弃的草稿、打印错误的表单、不再需要的复印件等含有个人信息的纸质材料，必须使用碎纸机进行粉碎处理，严禁直接丢入垃圾桶。需批量销毁的，应送至指定保密销毁机构进行监销，并留存销毁记录。

**四、 物理与环境安全防护措施**

针对社会救助项目存在大量纸质档案（如低保申请书、核对报告、民主评议记录等）的特点，我司将采取严格的物理防护措施。

**（一） 办公场所封闭管理**
1.  **独立办公区**：我司将为本项目设立独立的办公区域，与公司其他项目物理隔离。
2.  **门禁考勤**：办公区入口设置门禁系统，非项目组人员未经许可严禁进入。外来人员（如快递、访客）必须在前台登记，且不得进入核心数据处理区。
3.  **视频监控全覆盖**：在档案柜存放区、数据录入区、服务器存放区等关键位置安装无死角视频监控，杜绝非法翻阅、拍摄档案的行为。

**（二） 档案实体安全**
1.  **“三铁”要求**：档案存放严格执行“铁门、铁窗、铁柜”标准。所有纸质档案必须存放在带锁的铁皮文件柜中，钥匙由专人保管。
2.  **防火防潮防虫**：档案室配备灭火器、温湿度计、除湿机及防虫防鼠设施，确保档案实体安全，防止因自然灾害导致档案损毁。
3.  **借阅登记制度**：建立严格的档案借阅登记台账。项目组成员因工作需要查阅档案的，须经主管审批，在指定区域阅览，严禁将原件带出档案室，严禁私自复印。

**五、 人员保密教育与培训**

人是保密工作的核心变量。我司将把保密教育贯穿于项目执行的全过程，提升全员保密意识和技能。

**（一） 岗前保密培训**
所有拟投入本项目的人员，在正式上岗前必须接受不少于4学时的保密专题培训。培训内容包括：
1.  国家保密法律法规及刑事责任解析。
2.  采购人及我司的数据安全管理制度。
3.  社会救助工作中常见的泄密风险点（如入户拍照、微信群传输、闲聊泄密等）。
4.  办公设备及软件的安全操作规范。
培训结束后进行闭卷考试，成绩合格者方可上岗。

**（二） 定期保密教育**
1.  **月度例会提醒**：每月的项目工作例会中，设立专门的“保密时刻”，通报近期的信息安全检查情况，强调保密纪律。
2.  **警示教育**：每季度组织一次典型泄密案例警示教育活动，通过剖析行业内发生的违规泄密案例，以案说法，警钟长鸣。

**（三） 专项技能培训**
针对系统管理员、档案管理员等关键岗位，开展数据防泄漏、病毒查杀、应急响应等专业技能培训，提升技术防护能力。

**六、 应急响应与违规处置机制**

**（一） 应急响应预案**
我司制定了详细的《数据安全事件应急响应预案》，将安全事件分为特别重大、重大、较大和一般四级。
1.  **监测预警**：一旦发现数据泄露嫌疑、系统被入侵或档案丢失等情况，发现人应在15分钟内向项目保密组长报告。
2.  **立即止损**：项目组立即启动应急预案，采取断网、封存设备、修改密码、物理隔离等措施，防止事态扩大。
3.  **及时上报**：在事故发生后1小时内，向采购人及相关监管部门报告事件概况，不瞒报、不迟报。
4.  **溯源整改**：配合相关部门进行调查取证，查明原因，追究责任，并在3个工作日内提交详细的事故分析报告及整改措施。

**（二） 违规惩处承诺**
若因我司管理不善或人员违规导致数据泄露，我司承诺承担以下责任：
1.  **内部惩处**：对直接责任人给予开除处理，并扣发全部绩效奖金；对项目负责人给予降级或撤职处理。
2.  **法律责任**：若构成犯罪，我司将主动移交司法机关，依法追究刑事责任。
3.  **违约赔偿**：完全接受采购人依据合同条款提出的违约金处罚。若给采购人或服务对象造成经济损失或名誉损害的，我司承诺全额赔偿，并公开道歉，消除影响。

**七、 针对本项目的特别保密承诺**

基于对民政社会救助工作的深刻理解，**杭州菲尔德经济信息咨询有限公司**向采购人做出以下十项特别承诺：

1.  **承诺不倒卖数据**：绝不以任何形式将项目获取的低保户、特困人员等信息出售给第三方机构（如保险公司、推销机构、放贷机构等）牟利。
2.  **承诺不私自留存**：项目结束后，除按合同约定移交的数据外，彻底清除我司所有设备上的过程数据，不留备份。
3.  **承诺不跨境传输**：本项目所有数据存储、处理均在中国境内完成，绝不向境外机构或服务器传输任何项目数据。
4.  **承诺不擅自发布**：未经采购人书面授权，绝不擅自接受媒体采访，不在互联网、自媒体上发布任何与项目对象相关的文字、图片或视频。
5.  **承诺配合审计**：无条件接受采购人及上级主管部门对保密工作的飞行检查和审计，对发现的问题立行立改。
6.  **承诺技术合规**：所使用的软件系统（如有）均经过安全检测，不含后门程序、木马病毒或非法收集用户信息的插件。
7.  **承诺背景审查**：对所有项目人员进行严格的背景调查，确保无犯罪记录、无不良征信记录，政治立场坚定。
8.  **承诺第三方管控**：若项目中涉及第三方协作（如印刷服务、车辆租赁），我司将与第三方签署严格的保密协议，并承担连带保密责任。
9.  **承诺7×24小时响应**：建立全天候的安全联络机制，确保在任何时间发生数据安全紧急情况时，均能联系到我司负责人并立即处置。
10. **承诺终身追责**：即便合同履行完毕，我司对项目期间知悉的国家秘密、商业秘密及个人隐私仍承担无限期的保密义务。

**八、 保密工作自查与审计计划**

为确保上述措施落到实处，我司将执行严格的自查与审计计划：

1.  **日检查**：资料管理员每日下班前检查档案柜是否上锁、电脑是否关机、桌面是否有敏感纸质材料（“清桌锁屏”）。
2.  **周抽查**：保密专员每周随机抽查入户调查员的手机图库、微信聊天记录（工作机），检查是否存在违规传输敏感信息行为。
3.  **月审计**：每月对系统访问日志、数据导出记录进行审计，分析是否存在异常操作行为。
4.  **季评估**：每季度组织一次全面的保密风险评估，识别新出现的安全隐患，更新防护策略，并向采购人提交《季度数据安全与保密工作报告》。
