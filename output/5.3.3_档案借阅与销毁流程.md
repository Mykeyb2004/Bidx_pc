# 5.3.3 档案借阅与销毁流程

#### 5.3.3 档案借阅与销毁流程

为规范杭州菲尔德经济信息咨询有限公司（以下简称“菲尔德咨询”或“我司”）承接社会救助项目过程中的档案借阅与销毁工作，确保困难群众家庭信息的安全性、保密性与完整性，根据《中华人民共和国档案法》、《中华人民共和国数据安全法》、《社会救助暂行办法》及民政部门关于社会救助档案管理的具体规定，特制定本流程。本流程旨在建立一套权限明确、审批严格、流转可溯、销毁合规的闭环管理体系，杜绝档案丢失、损毁及信息泄露风险。

**一、 档案借阅与销毁总体原则**

我司在社会救助档案管理中，始终秉持“安全第一、利用受控、程序合规、责任到人”的核心原则。

**（一） 分级授权原则**
档案借阅严格实行分级授权管理制度。根据档案的密级（如涉及特困人员隐私、未成年人保护个案等）和借阅人员的职级、岗位职责，设定不同的查阅权限。严禁越级查阅、跨部门随意调阅，确保档案信息仅在必要的工作范围内流动。

**（二） 审批留痕原则**
所有档案的借出、查阅、复制及销毁行为，必须经过严格的书面或电子审批流程。每一个环节都必须有完整的记录，包括申请人、审批人、经办人、时间、事由及具体档案卷宗号，做到“凡动必批、凡批必留痕”，确立全生命周期的可追溯机制。

**（三） 物理与信息双重安全原则**
对于实体纸质档案，实行物理隔离与监控下的借阅；对于电子档案，实行加密传输与阅后即焚（或权限回收）机制。销毁过程必须确保实体彻底粉碎、数据彻底擦除，不可复原。

**（四） 终身责任追究原则**
档案借阅人及销毁监销人对档案安全负有直接责任。一旦发生档案泄露、违规销毁或私自留存等情况，无论当事人是否在职，我司都将依据相关法律法规及公司制度严厉追责。

**二、 档案借阅管理流程**

档案借阅是档案利用的主要形式，也是风险管控的关键环节。我司将借阅分为“馆内查阅”与“外借使用”两类，原则上以馆内查阅为主，严格限制实体档案外借。

**（一） 借阅权限与范围界定**

1.  **内部工作人员借阅**
    *   **项目经理/主管：** 因工作汇报、迎检审计、绩效评估等需要，有权申请查阅本项目全量档案。
    *   **一线社工/核查员：** 仅限查阅其负责片区或具体个案对象的档案资料，严禁跨片区查阅非关联对象的敏感信息。
    *   **质控/督导人员：** 因抽查案卷质量、开展个案督导需要，可申请查阅相关批次或特定类型的档案。

2.  **外部单位借阅**
    *   **委托方（民政局/街道）：** 因行政复议、信访调查、大数据比对核实等公务需要，凭单位介绍信或正式公函，经项目负责人审批后提供查阅。
    *   **公检法纪机关：** 因案件侦查、取证需要，须持两名工作人员执法证件及单位专用介绍信，经公司法务审核及项目经理批准后，配合查阅或复印。
    *   **其他第三方：** 严禁向商业机构、非关联个人提供任何形式的社会救助档案借阅服务。

**（二） 借阅具体操作规程**

1.  **借阅申请**
    *   借阅人需登录我司“社会救助档案管理系统”或填写纸质版《档案借阅申请单》。
    *   申请内容必须详实，包括：借阅档案的案卷号、救助对象姓名、所属街道/社区、借阅具体用途（如：低保复核、审计整改、信访回复）、拟归还日期等。
    *   **严禁模糊申请：** 不得使用“工作需要”等笼统词汇，必须具体到事由。

2.  **审批流转**
    *   **一级审批（部门负责人）：** 审核借阅事由的真实性与必要性。确认该员工是否有权限接触此类敏感信息。
    *   **二级审批（档案管理员）：** 核对档案状态（是否在库、是否处于封存期），确认无误后签署意见。
    *   **三级审批（项目经理）：** 对于涉及特困供养人员财产信息、未成年人救助个案等高敏感度档案，或需将档案带出档案室的情况，必须由项目经理进行终审。

3.  **档案调取与清点**
    *   审批通过后，档案管理员进入库房调取档案。
    *   **双人清点机制：** 管理员与借阅人当面核对案卷内容。重点检查：卷内目录与实物是否相符、页码是否连续、是否有破损或涂改痕迹。
    *   确认无误后，双方在《档案借阅登记簿》上签字确认，并注明借出时的档案状况（如“第5页右下角有轻微折痕”等细节）。

4.  **阅览管理**
    *   **阅览环境：** 原则上仅限在档案室专设的“阅览区”进行查阅，该区域配备24小时高清监控，无死角覆盖。
    *   **行为规范：** 借阅人查阅时，严禁吸烟、喝水、进食，严禁使用钢笔、圆珠笔在档案上涂改、圈划或批注。仅允许使用铅笔进行必要的摘录，或使用公司专用复印机进行脱敏复印。
    *   **禁止行为：** 未经特别授权，严禁使用手机、相机对档案内容进行拍照、扫描或OCR识别。

5.  **外借特殊规定**
    *   社会救助原始档案原则上不外借。确因法院出庭、政府紧急会议等特殊情况需带出档案室的，须办理“特许外借手续”。
    *   外借档案必须装入专用的防水、防火、带锁的“机要文件传输箱”。
    *   借阅期限严格控制在24小时以内，必须当天借出、当天归还。无法当天归还的，必须存放在符合保密要求的保险柜中，严禁带回家中或公共场所存放。

6.  **归还与注销**
    *   档案归还时，实行严格的“逆向清点”程序。档案管理员须逐页检查案卷，确认无缺页、无新增污损、无涂改痕迹。
    *   重点核查案卷内的证件复印件（如身份证、户口本、残疾证复印件）是否短少。
    *   检查无误后，管理员在《档案借阅登记簿》上填写归还日期，双方签字注销借阅记录。
    *   若发现档案受损或缺失，立即启动《档案安全事故应急预案》，暂缓注销，并向项目经理报告，追查责任。

**（三） 电子档案借阅的特殊管控**

随着数字化民政建设，电子档案借阅日益频繁，我司采取以下技术手段保障安全：

1.  **数字水印技术：** 所有借阅展示的电子档案页面，均自动生成包含“借阅人姓名+工号+时间+IP地址”的动态半透明水印，防止截屏外泄。
2.  **浏览权限控制：** 系统设定“只读”权限，禁止下载、另存为、打印功能（除非获得下载/打印的专项审批）。
3.  **时效自动熔断：** 借阅授权设定精确到分钟的有效期（如2小时）。超时后，系统自动收回查看权限，页面强制关闭。
4.  **操作日志审计：** 系统后台自动记录借阅人的每一次点击、翻页、搜索行为，形成不可篡改的审计日志，定期由技术部门导出分析，排查异常访问行为。

**三、 档案销毁管理流程**

社会救助档案涉及困难群众切身利益，其销毁工作具有极高的法律风险。我司严格遵循“鉴定先行、多级审批、物理粉碎、监销到位”的流程，严禁随意丢弃或非正规销毁。

**（一） 销毁鉴定机制**

1.  **成立鉴定小组**
    我司组建由项目经理任组长，档案主管、法务专员、资深社工师及业主方代表（视合同要求邀请）组成的“档案价值鉴定小组”。

2.  **定期排查**
    每年年底，档案管理员对库房内档案进行全面盘点，筛选出已达到保管期限的档案。
    *   **保管期限标准：** 严格执行《社会救助档案管理办法》及当地民政局规定。通常情况下，低保、特困供养等审批类档案保管期限为长期（或永久）；短期救助、咨询类记录保管期限一般不少于10年。
    *   **延期保留原则：** 对于涉及未结信访案件、正在进行审计调查、存在权属争议或具有历史研究价值的档案，即使达到保管期限，也应提出“延期销毁”意见，继续妥善保管。

3.  **价值评估**
    鉴定小组对初选出的拟销毁档案进行逐卷审查。
    *   审查内容：档案是否确已失效？是否涉及后续追溯？是否有留存样本价值？
    *   鉴定结论：分为“准予销毁”、“延长保管期限”、“提升保管级别”三类，并形成《档案销毁鉴定报告》。

**（二） 销毁申报与审批**

1.  **编制销毁清册**
    对于鉴定为“准予销毁”的档案，由档案管理员编制详细的《档案销毁清册》。
    *   清册内容包括：案卷号、题名（如某某年度某街道低保档案）、起止年度、所属类别、应保管期限、已保管年限、销毁原因、卷数、页数等。
    *   清册必须做到账物相符，严禁笼统登记。

2.  **内部审批**
    *   档案主管复核清册，确认无误后签字。
    *   项目经理审核鉴定报告及清册，签署执行意见。
    *   公司总经理或分管副总进行终审批准。

3.  **报备业主方**
    鉴于社会救助档案的政务属性，我司在实施销毁前，将《档案销毁申请书》及《档案销毁清册》正式发函报送至委托方（民政局或街道办事处）。经委托方书面批复同意后，方可启动物理销毁程序。未经业主方同意，我司无权单方面销毁任何项目档案。

**（三） 物理销毁实施规程**

1.  **打包与转运**
    *   待销毁档案在出库前，再次进行清点核对。
    *   使用专用密封袋或箱体进行封装，贴上“待销毁”封条，注明数量及重量。
    *   转运过程中，必须由我司两名以上正式员工全程押运，严禁委托第三方快递或物流公司单独运输，确保途中不遗失、不被窃取。

2.  **销毁方式与场所**
    *   **少量销毁：** 对于数量较少的纸质档案，在我司内部使用高保密级别的碎纸机（达到德国DIN 66399标准P-5级以上，即碎纸颗粒面积小于30mm²）进行粉碎。
    *   **批量销毁：** 对于大批量档案，送至国家保密局指定的涉密载体销毁中心进行集中销毁。采用制浆（化浆）或焚烧方式，确保信息载体彻底毁灭。

3.  **双人监销制度**
    *   销毁现场必须实行“双人监销”。监销人通常由档案主管和纪检/行政专员担任。
    *   监销职责：
        *   监督销毁过程，防止档案被中途截留或私自藏匿。
        *   检查销毁结果，确保纸张完全化为纸浆或灰烬，无任何可识别字迹残留。
        *   对于电子存储介质（硬盘、光盘、U盘），必须进行物理粉碎或消磁处理，严禁仅做格式化处理。

4.  **销毁记录归档**
    *   销毁结束后，监销人和具体执行人在《档案销毁清册》上共同签名确认，并注明销毁地点、时间、方式。
    *   若在指定销毁中心处理，需索取销毁中心出具的《销毁证明》。
    *   将《档案销毁申请书》、《档案销毁清册》、《销毁证明》及现场销毁照片/视频资料整理立卷，作为永久档案保存。该卷宗即为“档案的档案”，用于日后查证档案去向。

**四、 档案借阅与销毁的风险防控**

为了确保流程的刚性执行，我司建立了多维度的风险防控体系。

**（一） 借阅逾期预警机制**
档案管理系统设置自动预警功能。
*   **临期提醒：** 借阅期满前4小时，系统自动向借阅人发送短信或邮件提醒，提示尽快归还。
*   **逾期锁定：** 一旦逾期未还，系统自动锁定该员工的借阅权限，禁止其发起新的借阅申请，并同步通知其部门负责人进行催办。
*   **通报处罚：** 对于恶意逾期、长期霸占档案不还的行为，纳入员工月度绩效考核，扣除相应分值。

**（二） 敏感信息脱敏机制**
在提供借阅（尤其是复印件或电子件）时，对非必要显示的敏感信息实施强制脱敏。
*   **部分遮盖：** 对身份证号中间8位、银行卡号中间8位、手机号中间4位进行星号遮盖。
*   **隐私保护：** 对于涉及艾滋病、精神障碍、吸毒史等高度敏感隐私的医学证明材料，除特定审核人员外，对普通查阅人员实行物理封存或技术屏蔽。

**（三） 应急处置预案**
针对档案借阅与销毁过程中可能出现的突发状况，制定专项预案：
1.  **档案丢失：** 立即封锁现场，调取监控录像，排查接触人员。同时在24小时内向业主方报告，并尝试通过电子备份或源头数据进行补救重建。
2.  **档案损毁：** 对受损档案进行抢救性修复（如裱糊、去污）。无法修复的，注明损毁原因及程度，经审批后用复印件或电子打印件替代，并加盖“与原件核对无误”印章。
3.  **违规销毁：** 若发现误销毁未到期档案，立即启动倒查程序，严肃追究责任人法律责任，并评估对项目的影响，制定补救措施。

**五、 档案管理配套表单与工具**

为确保流程落地，菲尔德咨询设计并使用以下标准化表单和工具：

**（一） 标准化表单**
1.  **《社会救助档案查阅/借阅申请单》：** 包含申请人信息、借阅清单、用途说明、保密承诺书、三级审批栏。
2.  **《档案借阅登记簿（台账）》：** 包含序号、借阅日期、案卷号、借阅人签字、归还日期、经手人签字、备注（档案状况）。
3.  **《档案销毁鉴定报告》：** 包含鉴定小组成员名单、鉴定时间、鉴定标准、鉴定结论。
4.  **《档案销毁清册》：** 详细列明销毁档案的元数据。
5.  **《档案销毁监销记录表》：** 包含销毁过程描述、监销人签字、照片粘贴处。

**（二） 管理工具**
1.  **智能档案柜：** 配备人脸识别或指纹开锁功能，自动记录开柜人员和时间。
2.  **RFID射频识别标签：** 在每卷档案上粘贴RFID标签，配合门禁系统，实现档案违规带出库房自动报警。
3.  **高拍仪与水印系统：** 用于档案的快速数字化采集及输出时的强制水印添加。

**六、 结语**

档案借阅与销毁是社会救助档案管理中风险最高的环节。杭州菲尔德经济信息咨询有限公司郑重承诺：我们将以最严谨的态度、最规范的流程、最先进的技术手段，守护每一份社会救助档案。我们深知，这些档案不仅是纸张与数据，更是困难群众获得政府帮扶的凭证与希望。通过上述严格的借阅审批与销毁监销制度，确保档案“管得住、用得好、销得准”，为政府社会救助工作的阳光、公正、透明提供坚实的档案管理保障。
