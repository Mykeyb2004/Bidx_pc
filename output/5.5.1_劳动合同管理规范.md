# 5.5.1 劳动合同管理规范

#### 5.5.1 劳动合同管理规范

**一、 总则**

**(一) 制定目的**
为规范杭州菲尔德经济信息咨询有限公司（以下简称“菲尔德咨询”或“公司”）的劳动用工行为，构建和谐稳定的劳动关系，保障公司与员工双方的合法权益，确保社会救助服务项目的长期性、稳定性和专业性，根据《中华人民共和国劳动法》、《中华人民共和国劳动合同法》及相关法律法规，结合民政社会救助工作的特殊性质及公司实际情况，特制定本管理规范。

**(二) 适用范围**
本规范适用于公司所有以全日制用工形式建立劳动关系的员工，包括但不限于项目管理人员、一线社会工作者、家庭经济状况核对员、入户调查员及行政后勤人员。对于非全日制用工、劳务派遣人员及实习生，参照本规范相关条款或另行签订相应协议执行。

**(三) 管理原则**
1.  **依法合规原则：** 所有劳动合同的订立、履行、变更、解除和终止，必须严格遵守国家法律法规及地方政策规定，确保零法律风险。
2.  **平等自愿原则：** 劳动合同的签订建立在双方地位平等、意志自由的基础上，严禁强迫或欺诈行为。
3.  **协商一致原则：** 合同条款的确定及后续变更，须经公司与员工充分沟通并达成共识。
4.  **诚实信用原则：** 双方应如实告知与劳动合同直接相关的基本情况，信守承诺，履行义务。
5.  **保障服务原则：** 劳动合同管理应服务于社会救助项目的整体目标，通过规范化管理降低人员流失率，保障对困难群众服务的连续性。

**(四) 管理机构**
公司人力资源部是劳动合同归口管理部门，负责劳动合同制度的制定、修订、解释及执行监督。各项目组（包括驻街道/乡镇工作站）负责人协助人力资源部做好本项目内员工的劳动合同日常管理工作。

**二、 劳动合同的订立**

**(一) 录用前提与资格审查**
鉴于社会救助工作的严肃性与涉密性（涉及大量困难群众隐私及家庭经济数据），公司在订立劳动合同前，必须对拟录用人员进行严格的资格审查：
1.  **背景调查：** 核实拟录用人员的身份信息、学历学位、职业资格（如社会工作者职业水平证书）、过往工作经历及无犯罪记录证明。
2.  **入职体检：** 拟录用人员需提供指定医疗机构出具的合格体检报告，确保身体状况能适应入户调查、基层走访等高强度工作。
3.  **利益冲突审查：** 核实拟录用人员是否与服务辖区内的低保、特困等救助对象存在近亲属关系，若存在，需在合同中签署回避承诺书。

**(二) 订立时限**
公司自用工之日起即与员工建立劳动关系。人力资源部应在员工入职之日起**一个工作日内**，最迟不超过**一个月内**，与员工签订书面劳动合同。若因员工个人原因拒绝签订，公司将依法终止用工并书面通知。

**(三) 合同类型与期限**
根据社会救助项目的周期特点及岗位性质，公司实行多元化的合同期限管理：
1.  **固定期限劳动合同：** 适用于大多数项目制员工。合同期限原则上与服务采购项目周期保持一致或略长，一般为1年至3年。
2.  **无固定期限劳动合同：** 对于在公司连续工作满十年，或连续订立二次固定期限劳动合同且无相关法律规定情形的骨干员工（如资深督导、项目总监），公司依法与其订立无固定期限劳动合同，以稳定核心团队。
3.  **以完成一定工作任务为期限的劳动合同：** 适用于临时性、突发性或专项性的调查评估任务（如年度专项复核、突发灾害临时救助摸排）。

**(四) 合同内容**
劳动合同必须具备法律规定的必备条款，并结合民政服务特色增加约定条款：
1.  **必备条款：** 用人单位信息、劳动者信息、合同期限、工作内容与地点、工作时间与休息休假、劳动报酬、社会保险、劳动保护与职业危害防护等。
2.  **约定条款：**
    *   **试用期约定：** 劳动合同期限三个月以上不满一年的，试用期不超过一个月；一年以上不满三年的，试用期不超过二个月；三年以上固定期限和无固定期限的，试用期不超过六个月。试用期包含在劳动合同期限内。
    *   **保密协议：** 针对接触社会救助信息系统、掌握救助对象家庭隐私数据的岗位，必须在合同中约定严格的保密义务及违约责任。
    *   **服务质量承诺：** 将“入户调查准确率”、“档案管理规范率”、“群众满意度”等关键绩效指标作为岗位职责的附件，纳入合同管理范畴。
    *   **廉洁从业约定：** 明确禁止员工在服务过程中索取或收受救助对象的财物，禁止利用职务之便谋取私利，一旦违反，视为严重违反公司规章制度，公司有权解除合同。

**(五) 签订程序**
1.  **文本准备：** 人力资源部根据岗位类别准备相应的标准合同文本及附件（如岗位说明书、廉洁承诺书）。
2.  **条款告知：** 人力资源部专员向员工详细解读合同条款，特别是薪酬结构、绩效考核方式、录用条件及违纪责任，确保员工充分理解。
3.  **签字盖章：** 双方确认无误后，由员工本人签字（按手印），公司法定代表人或授权委托人签字并加盖公司公章。
4.  **文本留存：** 劳动合同一式两份，双方各执一份，具有同等法律效力。人力资源部应建立《劳动合同签收台账》，由员工签收确认已领取合同文本。

**三、 劳动合同的履行**

**(一) 全面履行**
公司与员工应按照劳动合同的约定，全面履行各自的义务。
1.  **公司义务：** 按时足额支付劳动报酬，缴纳社会保险及住房公积金，提供符合国家规定的劳动安全卫生条件，提供必要的劳动防护用品（如入户调查时的防疫物资、交通补贴等），开展岗前培训及定期专业技能培训。
2.  **员工义务：** 遵守公司考勤制度、档案管理制度、入户调查规范等各项规章制度，按质按量完成工作任务，维护公司声誉，保守工作秘密。

**(二) 试用期管理**
1.  **考核标准：** 试用期内，公司将依据《新员工试用期考核办法》对员工进行全方位考核，重点考察其政策理解能力、群众沟通能力、文书撰写能力及职业道德。
2.  **转正流程：** 试用期满前一周，由直属主管及项目负责人进行评估。考核合格者，按期转正；考核不合格者，公司将向其说明理由及依据，并依法解除劳动合同。

**(三) 劳动报酬与福利**
1.  **薪酬结构：** 公司实行“基本工资+绩效工资+津贴补贴”的薪酬结构。绩效工资与项目履约情况、个人工作量（如核查户数）及服务质量直接挂钩，体现多劳多得、优绩优酬。
2.  **支付方式：** 每月固定日期通过银行转账支付，并向员工提供工资清单。
3.  **调整机制：** 建立正常的工资增长机制，根据公司经营状况、项目预算调整及员工年度绩效考核结果，适时调整员工薪酬水平。

**四、 劳动合同的变更**

**(一) 变更情形**
在合同履行期间，出现下列情形之一的，双方可以协商变更劳动合同相关内容：
1.  订立劳动合同所依据的法律、法规、规章或政策发生变化（如最低工资标准调整、社保政策调整）。
2.  订立劳动合同所依据的客观情况发生重大变化，致使劳动合同无法履行（如政府购买服务项目内容发生重大调整、服务区域变更）。
3.  员工因个人原因（如身体健康、家庭变故、进修学习）申请调整工作岗位或时间。
4.  员工经考核不能胜任原工作岗位，需要进行岗位调整的。
5.  员工因工作表现优异，获得晋升或转岗的。

**(二) 变更程序**
1.  **提出申请：** 一方提出变更要求的，应当以书面形式通知对方，并说明变更的理由、内容及期限。
2.  **协商谈判：** 双方在收到通知后，应在平等自愿的基础上进行协商。公司应充分听取员工意见，并做好解释工作。
3.  **签订协议：** 协商一致后，双方签订《劳动合同变更书》。变更书一式两份，双方各执一份，作为原劳动合同的附件。
4.  **未达成一致的处理：** 若双方未能就变更达成一致，原劳动合同继续履行。但若属于法律规定公司有权单方调整的情形（如脱密期管理、法定医疗期满后不能从事原工作），按相关法律规定执行。

**五、 劳动合同的解除与终止**

**(一) 双方协商解除**
经公司与员工协商一致，可以解除劳动合同。由公司提出解除的，公司依法支付经济补偿；由员工提出解除的，公司不支付经济补偿。

**(二) 员工单方解除（辞职）**
1.  **预告解除：** 员工提前三十日以书面形式通知公司（试用期内提前三日），可以解除劳动合同。员工应配合做好工作交接，确保项目档案、救助对象数据、办公设备等完好归还。
2.  **即时解除：** 若公司存在未及时足额支付劳动报酬、未依法缴纳社保、规章制度违法损害员工权益等情形，员工可以随时通知公司解除劳动合同，并有权要求经济补偿。

**(三) 公司单方解除**
1.  **过失性解除（无补偿）：** 员工有下列情形之一的，公司可以立即解除劳动合同，且不支付经济补偿：
    *   在试用期间被证明不符合录用条件的。
    *   严重违反公司规章制度的（如：伪造入户调查记录、泄露救助对象隐私造成严重后果、利用职务便利收受贿赂、无故旷工达到规定天数等）。
    *   严重失职，营私舞弊，给公司造成重大损害的。
    *   同时与其他用人单位建立劳动关系，对完成本公司工作任务造成严重影响，或者经公司提出，拒不改正的。
    *   被依法追究刑事责任的。
2.  **非过失性解除（有补偿）：** 有下列情形之一的，公司提前三十日以书面形式通知员工本人或者额外支付一个月工资后，可以解除劳动合同：
    *   员工患病或者非因工负伤，在规定的医疗期满后不能从事原工作，也不能从事由公司另行安排的工作的。
    *   员工不能胜任工作，经过培训或者调整工作岗位，仍不能胜任工作的。
    *   劳动合同订立时所依据的客观情况发生重大变化，致使劳动合同无法履行，经双方协商，未能就变更劳动合同内容达成协议的。

**(四) 经济性裁员**
若因公司依照企业破产法规定进行重整，或生产经营发生严重困难等法定情形需要裁减人员二十人以上或者裁减不足二十人但占企业职工总数百分之十以上的，公司将严格履行法定程序：
1.  提前三十日向工会或者全体职工说明情况。
2.  听取工会或者职工的意见。
3.  将裁减人员方案向劳动行政部门报告。
4.  在裁员时，优先留用与公司订立较长期限劳动合同的、家庭无其他就业人员且有需要扶养的老人或者未成年人的员工。

**(五) 劳动合同终止**
有下列情形之一的，劳动合同终止：
1.  劳动合同期满的。
2.  员工开始依法享受基本养老保险待遇的。
3.  员工死亡，或者被人民法院宣告死亡或者宣告失踪的。
4.  公司被依法宣告破产、被吊销营业执照、责令关闭、撤销或者提前解散的。
5.  法律、行政法规规定的其他情形。

**(六) 离职交接与证明**
1.  **交接义务：** 劳动合同解除或终止时，员工必须按照《员工离职交接管理办法》办理工作交接手续。特别是对于手中的困难群众个案档案、系统登录账号、加密狗（Key）、未办结的审批事项等，必须逐一核对移交，由监交人签字确认。
2.  **出具证明：** 公司应当在解除或者终止劳动合同时出具解除或者终止劳动合同的证明，并在十五日内为员工办理档案和社会保险关系转移手续。
3.  **经济补偿支付：** 涉及经济补偿的，公司在办结工作交接时支付。

**六、 特殊事项管理**

**(一) 保密与竞业限制**
1.  **保密义务延续：** 员工离职后，仍需遵守劳动合同中约定的保密义务，不得泄露在职期间掌握的救助对象隐私数据、政府内部文件及公司商业秘密。
2.  **竞业限制：** 对于掌握核心技术或商业秘密的高级管理人员、高级技术人员，公司可与其约定竞业限制条款。竞业限制期限不得超过两年。在竞业限制期限内，公司按月支付经济补偿；若员工违约，应支付违约金。

**(二) 服务期协议**
公司为员工提供专项培训费用，对其进行专业技术培训（如高级社会工作师考前培训、各类民政业务专项研修）的，可以与该员工订立协议，约定服务期。员工违反服务期约定的，应当按照约定向公司支付违约金。违约金的数额不得超过公司提供的培训费用，且不得超过服务期尚未履行部分所应分摊的培训费用。

**(三) 劳务派遣特别规定**
若项目涉及使用劳务派遣人员（如临时性辅助岗位），公司将严格审核劳务派遣单位的资质，并监督其与被派遣劳动者签订两年以上的固定期限劳动合同，确保同工同酬，防止非法用工风险。

**七、 劳动争议处理与法律责任**

**(一) 争议预防**
公司坚持“预防为主”的原则，通过建立健全职工代表大会制度、工会组织及畅通的内部沟通渠道（如总经理信箱、HR接待日），及时了解员工诉求，化解潜在矛盾，构建和谐劳动关系。

**(二) 争议处理程序**
发生劳动争议时，按照以下程序处理：
1.  **内部协商：** 双方优先通过友好协商解决。
2.  **调解：** 协商不成的，可以向公司劳动争议调解委员会或基层人民调解组织申请调解。
3.  **仲裁：** 不愿调解、调解不成或者达成调解协议后不履行的，可以向劳动合同履行地或者公司所在地的劳动争议仲裁委员会申请仲裁。
4.  **诉讼：** 对仲裁裁决不服的，除法律规定的终局裁决外，可以向人民法院提起诉讼。

**(三) 违规责任**
1.  **公司责任：** 若公司违反本规范或法律规定，损害员工合法权益的，应依法承担赔偿责任；相关责任管理人员将受到公司内部行政处分。
2.  **员工责任：** 若员工违反本规范或劳动合同约定，给公司造成经济损失的，应依法承担赔偿责任。

**八、 档案管理与信息化**

**(一) “一人一档”制度**
人力资源部建立严格的员工劳动关系档案，实行“一人一档”管理。
1.  **归档内容：** 包括但不限于入职申请表、简历、身份证复印件、学历证件、体检报告、劳动合同原件、保密协议、岗位说明书、历年绩效考核表、培训记录、奖惩记录、变更协议及离职证明存根等。
2.  **保管要求：** 档案室应具备防火、防潮、防盗、防虫条件，严格限制查阅权限，确保员工个人信息安全。

**(二) 电子化管理**
依托公司HRM（人力资源管理）系统，建立电子劳动合同台账。
1.  **预警机制：** 系统设定关键节点预警功能，如合同到期前45天自动提醒、试用期满前10天提醒、退休年龄提醒等，确保劳动合同管理的及时性与准确性，避免因漏签、迟签引发的法律风险。
2.  **数据同步：** 劳动合同信息应与工资发放、社保缴纳、项目人员备案系统保持实时同步，确保数据一致性。

**九、 附则**

1.  本规范作为公司《员工手册》的重要组成部分，与《员工手册》具有同等效力。
2.  本规范制定过程中，已依法经职工代表大会讨论通过，并向全体员工公示。
3.  本规范依据的国家法律、法规及政策发生变化的，按新规定执行。
4.  本规范由杭州菲尔德经济信息咨询有限公司人力资源部负责解释。
5.  本规范自发布之日起正式实施。
