# 3.3.2 疑难个案会商机制（复杂家庭经济状况认定）

#### 3.3.2 疑难个案会商机制（复杂家庭经济状况认定）

在社会救助服务项目的实际执行过程中，家庭经济状况认定是核心环节，也是最具挑战性的难点。面对日益复杂的社会阶层结构和家庭组成形式，传统的“一刀切”式认定标准往往难以覆盖所有特殊个案，如人户分离、隐性收入核查难、因病致贫家庭刚性支出扣减、赡养抚养义务履行不到位等问题。为确保社会救助的公平性、精准性与及时性，杭州菲尔德经济信息咨询有限公司（以下简称“菲尔德咨询”或“我方”）基于多年一线实务经验，特制定本疑难个案会商机制。

本机制旨在通过搭建我方项目组与采购人（民政局/街道办）、村（居）委会及第三方专业力量的沟通桥梁，针对复杂家庭经济状况认定问题，实行“一事一议、集体决策、全程留痕”，在确保证据链完整、程序合规的前提下，实现救助政策的“实质正义”与“温情救助”。

一、 机制构建原则与总体目标

（一） 构建原则

1.  **依法依规，严守底线：** 所有会商行为必须严格遵循《社会救助暂行办法》及省、市、区关于最低生活保障、特困人员供养、临时救助等相关政策文件精神，确保决策有法可依。
2.  **客观公正，实事求是：** 坚持以入户调查客观事实为依据，摒弃主观臆断，通过多维数据交叉验证，还原困难家庭真实的经济生活状况。
3.  **以人为本，兜底保障：** 在政策允许的裁量权范围内，坚持“就高不就低、就宽不就严”的原则（在合规前提下），最大限度保障困难群众的基本生存权益。
4.  **高效协同，权责清晰：** 明确菲尔德咨询作为第三方服务机构的调查建议权与采购人的行政审批权之间的界限，形成高效互补的工作合力。

（二） 总体目标

通过建立常态化的疑难个案会商机制，实现以下目标：一是将疑难个案的认定准确率提升至100%；二是有效化解因认定分歧引发的信访投诉风险；三是形成一批具有典型参考价值的案例库，为后续政策微调提供数据支撑；四是确保每一笔救助资金都用在刀刃上，实现精准救助。

二、 疑难个案的界定标准与分类

启动会商机制的前提是精准识别“疑难个案”。菲尔德咨询根据过往项目经验，将以下五类情形纳入重点会商范围：

（一） 收入核算类疑难
1.  **隐性就业与灵活就业：** 申请人从事网约车、外卖配送、家政服务或零散务工，无固定银行流水，且申报收入与行业平均水平或家庭实际生活水准明显不符的。
2.  **收入豁免与扣减争议：** 涉及因病、因残、因学等刚性支出扣减金额的认定，特别是对于自费药、康复辅具等非医保目录内费用的合理性判定。
3.  **共同生活家庭成员认定：** 存在事实抚养/赡养关系但无相关法律文书，或户籍在一起但实际长期分户生活（如离婚不离家、分家析产不清）的复杂家庭结构。

（二） 财产认定类疑难
1.  **车辆与房产特殊情形：** 申请人名下拥有车辆但仅用于维持基本生计（如运营货车、农用机械），或拥有房产但属于唯一住房且年久失修、无法变现，以及名下有注册企业但实际已倒闭注销无门的“僵尸企业”法人代表。
2.  **拆迁安置补偿：** 涉及拆迁过渡期安置费算作收入还是财产、拆迁款分割纠纷导致申请人实际未获得资金等情况。

（三） 支出型贫困类疑难
家庭收入虽略高于低保标准，但因家庭成员患重大疾病、遭遇突发意外事件导致家庭刚性支出巨大，实际生活水平低于当地低保标准的“边缘户”认定。

（四） 赡养/抚养义务履行类疑难
法定赡养/抚养义务人有收入能力，但因家庭矛盾、失联、服刑、重残等原因，实际上未履行或无法履行赡养/抚养义务，导致申请人生活陷入困境的情形。

（五） 历史遗留与信访类疑难
涉及多次重复申请、对认定结果存在持续异议、有越级上访倾向或历史档案资料缺失导致认定困难的特殊个案。

三、 会商组织架构与职责分工

为确保会商机制的权威性与专业性，菲尔德咨询建议组建“疑难个案认定联席会商小组”，具体架构如下：

（一） 组织架构

1.  **召集人：** 由采购人分管领导或社会救助科（科室）负责人担任，负责主持会商会议，拥有最终决策权（一票否决权）。
2.  **执行组长：** 由菲尔德咨询项目经理担任，负责筛选案例、准备材料、陈述调查事实、记录会议纪要。
3.  **核心成员：**
    *   **采购人代表：** 民政经办人员，负责政策解释与把关。
    *   **我方专业社工：** 负责个案入户调查的一线社工，对家庭状况最熟悉。
    *   **村（居）代表：** 熟悉申请人邻里关系和日常生活的村（居）干部。
4.  **特邀顾问（视情况邀请）：** 法律顾问（律师）、医疗专家、审计人员或街道纪工委代表，针对涉及法律纠纷、医疗鉴定等极度复杂的个案提供咨询意见。

（二） 菲尔德咨询的职责定位

在会商机制中，菲尔德咨询不具备行政审批权，但承担着至关重要的“事实还原者”与“政策参谋者”角色：
1.  **调查取证：** 利用入户核查、邻里访问、信函索证等手段，收集包括银行流水、医疗发票、就业证明、法律文书等在内的完整证据链。
2.  **初审研判：** 对收集到的信息进行初步整理，对照政策条款提出初步认定建议（建议批准、建议驳回或建议调整救助类别）。
3.  **方案设计：** 针对特殊个案，设计“物质+服务”的综合救助建议方案。
4.  **档案留存：** 负责会商全过程的文字、录音、影像记录，形成《疑难个案会商会议纪要》，确保全程可追溯。

四、 疑难个案会商全流程实施方案

菲尔德咨询将严格执行“五步闭环”会商流程，确保每一个疑难个案都能得到妥善处理。

（一） 第一步：个案筛查与预警（启动阶段）
1.  **主动发现：** 我方入户调查员在实地核查中发现符合上述“疑难界定标准”的个案，立即在工作系统中进行标记，并填写《疑难个案情况说明表》。
2.  **资料预审：** 项目主管对标记个案进行复盘，检查证据是否充分。若证据不足，立即启动“二次深查”，必要时引入侧面调查（如走访网格员、物业、孩子班主任等）。
3.  **提请会商：** 经项目组内部研判无法独立得出明确结论的，由项目经理汇总整理，向采购人提交《疑难个案会商申请单》，列明个案基本情况、疑难点及拟会商时间。

（二） 第二步：多维调查与证据固化（准备阶段）
在正式会商前，菲尔德咨询将对提请个案进行全方位的“画像”：
1.  **经济状况画像：** 整理家庭近12个月的银行流水，标注大额进出款项并要求申请人说明；核算家庭刚性支出凭证。
2.  **生活状况画像：** 拍摄居住环境、家用电器、衣着饮食等照片，通过“生活痕迹”佐证经济状况。
3.  **社会关系画像：** 梳理赡养抚养关系网，必要时通过法律援助中心发函询证赡养费给付情况。
4.  **形成调查报告：** 撰写详尽的《疑难个案深度调查报告》，包含家庭概况、核心矛盾、政策依据分析及初步建议，提前2个工作日发送给会商小组成员审阅。

（三） 第三步：联席会商与集体决策（核心阶段）
1.  **会议召开：** 按照约定时间召开会商会议。会议流程如下：
    *   **情况陈述：** 我方调查员利用PPT或纸质档案，客观陈述个案调查情况，展示证据材料，重点说明疑难点。
    *   **多方质询：** 采购人代表、村（居）代表就调查细节进行提问，我方人员负责答疑补充。
    *   **政策辩论：** 依据相关救助政策，对该个案是否符合救助条件进行讨论。对于政策边缘地带，探讨是否适用“支出型贫困”或“临时救助”等兜底政策。
    *   **集体表决：** 参会人员发表明确意见（同意、不同意、缓办或补充调查），最终由召集人（采购人领导）根据多数意见并结合政策法规做出最终决定。
2.  **会议纪要：** 我方记录员全程记录会议内容，形成《会议纪要》，详细记录争议焦点、各方观点、最终结论及政策依据。所有参会人员需在纪要上签字确认。

（四） 第四步：结果执行与公示反馈（落地阶段）
1.  **分类执行：**
    *   **认定符合条件：** 协助采购人办理审批手续，按规定进行公示，确保救助金及时发放。
    *   **认定不符合条件：** 由我方社工配合村（居）干部，上门向申请人送达《不予批准告知书》，并耐心做好政策解释与情绪疏导工作，避免激化矛盾。
    *   **需补充调查：** 根据会商意见，在规定时限内（通常为3个工作日）完成补充调查，再次提交审核。
2.  **动态监测：** 对通过会商认定的对象，列入“重点监测名单”，提高复核频次（如由半年一次改为季度一次），及时掌握其家庭经济状况变化。

（五） 第五步：档案归档与复盘优化（提升阶段）
1.  **一案一档：** 将《疑难个案深度调查报告》、《会议纪要》、签字表决单及相关证据材料单独建档，作为行政审批的补充依据，以备审计检查。
2.  **案例入库：** 每季度对会商案例进行复盘，提炼典型特征，更新《社会救助疑难个案操作指引》，持续提升项目组的业务能力。

五、 针对典型复杂场景的专项解决方案

为体现菲尔德咨询的专业深度，针对以下三类最高频的复杂场景，我们制定了专项会商策略：

（一） 场景一：子女有赡养能力但拒绝履行的“支出型”老人
*   **痛点：** 老人无收入，子女有房有车但因家庭矛盾不给赡养费，按政策核算子女收入后老人不符合低保，但实际生活极度困难。
*   **会商策略：**
    1.  **法律介入：** 建议引入法律援助，发律师函督促子女履行义务。
    2.  **分类施策：** 若子女确有困难（如也患重病），通过会商核减赡养费标准。
    3.  **权利让渡：** 依据相关地方政策，若老人同意将赡养费追索权转让给政府或司法机关，可视情先给予低保或临时救助，后续由相关部门代为追偿。
    4.  **事实认定：** 侧重调查老人的实际生活水准（如是否独自居住危房、是否长期吃药无钱购买），通过“事实无人赡养”的逻辑进行会商认定。

（二） 场景二：拥有车辆/房产但陷入困境的家庭
*   **痛点：** 申请人因车祸致残，名下有一辆车（车祸受损或作为谋生工具），按硬性规定“有车不得保”，但家庭无收入来源。
*   **会商策略：**
    1.  **车辆属性评估：** 调查车辆实际用途。若是唯一的经营性工具（如送货面包车）且价值较低，会商建议豁免。若是“僵尸车”，协助申请人办理报废注销手续后纳入救助。
    2.  **房产价值评估：** 对于拥有两套房但其中一套为无法居住的农村老宅或极小面积的非成套住房，通过实地拍照、村委会出具证明，会商认定为“基本居住需求”，不作超标处理。
    3.  **宽限期设置：** 给予一定期限的资产处置宽限期，期间先给予临时救助过渡。

（三） 场景三：收入虽高但医疗支出巨大的“因病致贫”
*   **痛点：** 家庭年收入超过低保标准2倍，但家庭成员患白血病，自费医疗支出远超家庭总收入。
*   **会商策略：**
    1.  **精准测算：** 建立详细的“收入-刚性支出”核算模型。扣除个人自负医疗费用、必要的康复费用及护理费用后，核算家庭人均月收入。
    2.  **低保边缘/支出型低保：** 根据核算结果，建议纳入“低保边缘家庭”或“支出型贫困家庭”，享受医疗救助专项政策，而非单纯追求低保金。
    3.  **资源链接：** 会商决议除了给予政府救助外，同步转介慈善总会、红十字会或水滴筹等社会力量进行综合帮扶。

六、 风险防控与合规保障机制

疑难个案会商涉及自由裁量权的运用，必须建立严格的风险防火墙，保护采购人与我方人员的安全。

（一） 廉政风险防控
1.  **利益回避制度：** 参与会商的人员若与申请人有亲属、利害关系，必须主动申请回避。
2.  **双人调查原则：** 所有疑难个案的入户调查必须由2名以上工作人员共同完成，且调查记录需经申请人签字按手印确认，严禁单人作业。
3.  **全过程留痕：** 会商会议必须录音或录像，关键决策过程有据可查，杜绝“人情保”、“关系保”。

（二） 审计合规保障
1.  **政策依据引用：** 在《会议纪要》和《调查报告》中，必须明确引用具体的文件名称、文号及条款（如“根据《XX省最低生活保障审核确认办法》第X条之规定”），确保每一个决定都有法理支撑。
2.  **专家背书：** 对于极度敏感或涉及金额巨大的个案，引入第三方法律顾问或审计师出具书面意见书，分担行政决策风险。
3.  **公示公开：** 会商结果严格按照规定在申请人所在村（居）务公开栏进行长期公示，接受群众监督，确保阳光透明。

（三） 纠错与退出机制
1.  **定期复核：** 对经会商认定的对象，实施“红黄绿”三色管理，疑难个案标记为“红色”，每月进行一次电话回访，每季度进行一次入户复核。
2.  **及时停保：** 一旦发现申请人家庭经济状况好转或不再符合疑难个案认定条件，立即启动退出程序，追回违规领取的资金（如有），并完善退出档案。
