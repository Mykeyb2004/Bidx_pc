# 5.4.2 试用期管理规定

#### 5.4.2 试用期管理规定

**一、 总则**

为规范杭州菲尔德经济信息咨询有限公司（以下简称“菲尔德咨询”或“公司”）承接民政社会救助项目的人力资源管理，确保新入职员工能够快速适应社会救助工作的高标准、严要求，构建一支政治素质过硬、业务能力精湛、服务意识深厚的专业化服务团队，特制定本试用期管理规定。

本规定依据《中华人民共和国劳动合同法》及公司《人力资源管理制度汇编》制定，并深度结合社会救助领域的工作特性，旨在通过“选、育、考、留”的全流程闭环管理，在试用期内准确甄别员工与岗位的匹配度，保障项目服务质量的连续性与稳定性。

**（一） 适用范围**
本规定适用于公司所有新入职员工（含社会招聘、校园招聘及劳务派遣转正人员）。针对社会救助项目的一线社工、核查专员、信息录入员及项目管理人员，需严格执行本规定中的专项考核指标。

**（二） 管理原则**
1.  **合法合规原则：** 试用期的设定、薪资支付、考核及解除劳动关系等环节，必须严格遵守国家法律法规，保障员工合法权益。
2.  **人岗匹配原则：** 以胜任力模型为核心，重点考察员工的政策理解能力、群众工作能力及抗压能力，确保“人适其岗”。
3.  **导师带教原则：** 实行“师徒制”传帮带，确保新员工在试用期内能够迅速掌握民政业务系统的操作规范与入户核查技巧。
4.  **全面考核原则：** 坚持定性与定量相结合，过程考核与结果考核相结合，全方位评估员工的综合素质。

**二、 试用期期限与薪酬设定**

**（一） 期限设定**
公司根据劳动合同期限及岗位层级，依法设定合理的试用期，具体标准如下：
1.  劳动合同期限三个月以上不满一年的，试用期不超过一个月；
2.  劳动合同期限一年以上不满三年的，试用期不超过二个月；
3.  劳动合同期限三年以上固定期限和无固定期限的，试用期不超过六个月。
鉴于社会救助项目的复杂性，对于项目核心岗位（如项目主管、资深个案管理师），原则上设定3至6个月的试用期，以充分考察其应对突发舆情及复杂个案的处理能力。

**（二） 薪酬福利**
1.  **工资标准：** 试用期工资不低于劳动合同约定工资的80%，且不低于项目所在地最低工资标准。为体现对人才的尊重，菲尔德咨询对于持有助理社会工作师、社会工作师职业资格证书的员工，试用期工资可按转正工资的90%-100%发放。
2.  **社保缴纳：** 公司自员工入职之日起，即依法为其缴纳基本养老保险、医疗保险、失业保险、工伤保险和生育保险，确保员工在试用期内的基本保障，消除后顾之忧。
3.  **福利待遇：** 试用期员工享受与正式员工同等的法定节假日、高温补贴（如适用）及项目组提供的劳保用品（如入户核查专用的马甲、工作证、录音笔等）。

**三、 试用期培养体系：双导师“传帮带”机制**

针对社会救助工作政策性强、实操难度大的特点，菲尔德咨询在试用期内推行特色的“双导师制”，确保新员工平稳着陆。

**（一） 导师选拔与职责**
1.  **业务导师：** 由拥有3年以上民政工作经验或中级社工师资质的骨干担任。
    *   **职责：** 负责传授低保、特困、临时救助等政策知识，指导入户核查技巧，演示“省大救助信息系统”操作流程，审核新员工撰写的入户调查报告。
2.  **思想导师：** 由项目负责人或党支部成员担任。
    *   **职责：** 负责企业文化宣导、职业道德教育（特别是保密意识与廉洁自律教育），关注新员工心理动态，帮助其建立“民政为民、民政爱民”的服务情怀。

**（二） 分阶段培养计划**
试用期培养划分为三个阶段，每个阶段均有明确的学习目标与输出要求：

1.  **融入期（入职第1-2周）：政策通识与保密教育**
    *   **学习内容：** 学习《社会救助暂行办法》、省市相关低保特困认定细则；签署《用户信息保密协议》；熟悉公司考勤、财务报销等行政制度。
    *   **考核方式：** 政策知识笔试（闭卷，80分合格）、保密制度通关测试。
    *   **输出成果：** 完成一份《新员工入职学习心得》。

2.  **成长期（入职第3周-第1个月）：跟岗学习与模拟实操**
    *   **学习内容：** 跟随导师进行入户核查，学习如何与困难群众沟通、如何查验家庭经济状况佐证材料；观摩民主评议会流程；学习档案整理规范。
    *   **实操要求：** 在导师指导下，尝试使用手持终端（APP）进行信息采集，但不直接提交系统。
    *   **考核方式：** 模拟入户情景演练、档案整理抽查。

3.  **冲刺期（入职第2个月-转正前）：独立作业与个案试炼**
    *   **学习内容：** 独立承担一般性个案的入户调查任务（导师陪同但不干预）；处理简单的系统预警信息；参与救助对象的动态管理工作。
    *   **实操要求：** 独立撰写不少于5份《入户调查报告》，并经导师审核合格。
    *   **考核方式：** 岗位胜任力综合评估、试用期转正述职。

**四、 试用期考核评估体系**

菲尔德咨询建立多维度、量化与质性相结合的试用期考核体系，考核结果作为员工转正、延期转正或解除劳动关系的唯一依据。

**（一） 考核维度与权重**
1.  **职业道德与工作态度（权重20%）：**
    *   **指标：** 考勤合规率、保密纪律遵守情况、服务意识（是否存在被服务对象投诉）、团队协作精神。
    *   **否决项：** 凡在试用期内发生泄露救助对象隐私、索拿卡要、态度恶劣引发重大舆情者，实行“一票否决”，直接解除劳动合同。
2.  **政策理论水平（权重30%）：**
    *   **指标：** 对“分层分类”社会救助体系的掌握程度，对低保边缘家庭、支出型贫困家庭认定标准的熟悉程度。
    *   **测评方式：** 月度闭卷考试或随机口头问答。
3.  **业务实操能力（权重50%）：**
    *   **入户核查能力：** 能否准确识别“人户分离”、“隐性就业”等复杂情况；沟通技巧是否得当。
    *   **文书写作能力：** 调查报告是否客观、详实，逻辑是否严密，佐证材料是否完整。
    *   **系统操作能力：** 信息录入准确率（要求100%无误），数据比对反馈时效性。

**（二） 过程考核机制**
1.  **月度面谈：** 每月月末，业务导师需与试用期员工进行绩效面谈，指出其在入户技巧、政策运用等方面的不足，并填写《试用期员工月度辅导记录表》，双方签字确认。
2.  **关键事件记录法（CIM）：** 建立员工试用期成长档案，记录其表现优异的典型案例（如成功发现一户隐瞒收入的骗保行为）或失误案例（如档案归档混乱），作为转正评审的佐证材料。

**（三） 试用期转正考核流程**
1.  **个人申请：** 试用期结束前10个工作日，员工提交《转正申请书》及《试用期工作总结报告》，重点阐述在社会救助工作中的感悟与实绩。
2.  **导师评价：** 业务导师与思想导师分别出具评价意见，明确是否推荐转正。
3.  **360度评估：** 由项目主管、同事（不少于2人）、服务对象代表（可选，通过电话回访形式）进行满意度评分。
4.  **部门审核：** 项目经理结合上述材料及平时考核成绩，签署审核意见。
5.  **公司审批：** 人力资源部复核程序合规性，报总经理审批。

**五、 考核结果应用**

根据综合考核分值（满分100分），将试用期结果分为三类：

**（一） 正常转正（90分及以上）**
1.  员工表现优异，完全胜任岗位要求，且无任何违规违纪行为。
2.  人力资源部在试用期期满前发出《转正通知书》，按期办理转正手续。
3.  薪资调整至转正标准，并补发试用期内暂扣的绩效工资（如有）。

**（二） 延期转正（75分-89分）**
1.  **适用情形：** 员工基本素质尚可，但在某些专项技能上（如特殊困难家庭的沟通技巧、复杂报表的制作）仍需提升，或因客观原因（如病假）导致考核期不足。
2.  **处理措施：** 经双方协商一致，可延长试用期，但总试用期不得超过法律规定的上限。
3.  **改进计划：** 导师需制定针对性的《延期转正辅导计划》，明确改进目标与期限（通常为1个月）。若延长期满仍未达标，则予以辞退。

**（三） 终止试用（75分以下或触犯否决项）**
1.  **适用情形：**
    *   经考核证实不符合录用条件（如无法掌握核心政策、频繁出现工作失误且屡教不改）。
    *   严重违反公司规章制度或项目管理规定（如迟到早退严重、泄露数据）。
    *   在入职时提供虚假学历、履历或隐瞒重大疾病、犯罪记录等欺诈行为。
2.  **处理程序：**
    *   **证据固化：** 人力资源部收集《试用期考核表》、《面谈记录》、《违纪证明》等证据材料，确保事实清楚、证据确凿。
    *   **告知送达：** 在试用期届满前，向员工送达《解除劳动合同通知书》，并当面说明解除理由。
    *   **工作交接：** 员工需在规定时间内完成手头个案资料、账号权限、物资装备的交接，严禁带走任何涉及救助对象隐私的电子或纸质文档。
    *   **结算离职：** 依法结算工资，办理社保减员及离职证明。

**六、 特殊情形管理**

**（一） 试用期内的调岗**
若员工在试用期间表现出与其他岗位更高的匹配度（例如：入户核查岗员工更擅长数据分析，不善言辞），在公司有空缺编制且员工同意的前提下，可进行内部调岗。调岗后，试用期考核指标随之调整，但试用期期限连续计算。

**（二） 试用期辞职**
员工在试用期内提出辞职的，需提前3日以书面形式通知公司。项目负责人应安排离职面谈，了解离职真实原因，以便优化招聘与留人策略。员工需严格履行工作交接义务，特别是对于正在跟进的低保申请个案，必须形成详细的《个案交接单》，确保群众申请事项不因人员变动而停滞。

**七、 档案与信息化管理**

**（一） 纸质档案归档**
所有试用期管理过程中的产生的表单、试卷、报告、协议等纸质材料，均应整理归入员工个人人事档案，由人力资源部统一保管，保存期限不少于员工离职后两年，以备劳动监察或法律诉讼之需。

**（二） 数字化留痕**
依托菲尔德咨询内部OA办公系统，实现试用期管理的数字化全流程留痕。从入职指引的发送、导师的分配、月度考核的评分到转正审批的流转，全部在线上完成，确保管理动作可追溯、可查询，提升管理效率与透明度。

**八、 附则**

1.  本规定作为劳动合同的附件，具有同等法律效力。新员工入职时，公司将组织专门培训并要求员工签字确认已阅读并理解本规定内容。
2.  本规定由杭州菲尔德经济信息咨询有限公司人力资源部负责解释、修订与监督执行。
3.  本规定未尽事宜，依照国家及项目所在地相关法律法规、政策文件执行。如遇法律法规调整，以最新法律法规为准。
