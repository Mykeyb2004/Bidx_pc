# 1.2.3 驻点服务承诺（常驻通山县团队配置承诺）

#### 1.2.3 驻点服务承诺（常驻通山县团队配置承诺）

针对本次通山县社会救助服务项目的特殊性与重要性，杭州菲尔德经济信息咨询有限公司（以下简称“菲尔德咨询”或“我方”）深刻理解“驻点服务”对于保障社会救助政策落地、提升核查精准度以及增强困难群众满意度的关键作用。社会救助工作点多、面广、线长，且通山县地形复杂、乡镇分散，唯有建立一支“扎根当地、反应迅速、业务精湛、作风过硬”的常驻服务团队，才能真正实现“精准救助、高效服务”的目标。

基于此，我方郑重承诺：若有幸中标，将严格按照招标文件要求及采购人指示，在通山县设立专门的项目驻点办公室，组建全职常驻服务团队，实行准军事化管理与专业化运作，确保项目服务“零时差响应、零距离沟通、零死角覆盖”。具体承诺内容如下：

一、常驻机构设立与硬件配套保障承诺

为确保项目工作的高效运转，我方承诺在中标通知书发出后3个工作日内完成通山县本地驻点办事机构的设立与挂牌，打造集办公、档案管理、会议培训、应急调度于一体的标准化服务阵地。

（一）驻点办公场所配置承诺
1. **选址原则：** 我方承诺驻点办公室选址将紧邻通山县民政局或采购人指定办公区域（距离不超过2公里），以便于日常工作的汇报沟通、资料交接及紧急任务的即时受领。
2. **功能分区：** 办公场所面积不少于100平方米，严格按照功能进行科学分区，至少包含：
    （1）项目经理室：用于统筹管理与各方协调；
    （2）综合办公区：用于入户核查员、信息录入员日常办公；
    （3）档案室：按照国家档案管理标准建设，配备防盗门、防盗窗、遮光窗帘，安装温湿度计、灭火器，配置密集架，确保“七防”（防火、防盗、防潮、防霉、防虫、防光、防尘）措施到位，专门用于存放低保、特困等社会救助纸质档案；
    （4）保密室：用于处理涉及群众隐私的敏感数据，实行物理隔离与专人管理；
    （5）会议培训室：用于开展内部例会、业务培训及个案研判会议。
3. **形象建设：** 办公场所将统一悬挂项目名称牌匾、组织架构图、岗位职责、服务流程图、廉洁自律“十不准”及工作人员去向牌，展现正规化、专业化的服务形象。

（二）办公设备与信息化设施投入承诺
为保障核查数据的实时录入与高效处理，我方承诺为驻点团队配置足额且高性能的办公设备：
1. **计算机设备：** 承诺人手配置一台高性能笔记本电脑（配置不低于i5处理器、16G内存），并额外配备2台台式机用于大数据比对与服务器连接。所有电脑均安装正版杀毒软件与防火墙，实行内网（业务专网）与外网物理隔离或逻辑隔离，确保数据安全。
2. **影像采集设备：** 为每组入户核查人员配备专业高清照相机、执法记录仪及录音笔。承诺入户核查全程开启执法记录仪，实现“痕迹化管理”，确保核查过程可追溯、有据可查，有效规避潜在的矛盾纠纷。
3. **办公外设：** 配置高速高拍仪2台（用于档案数字化扫描）、多功能彩色打印复印一体机2台、碎纸机2台（用于销毁废弃涉密文件）、固定电话及网络设备，确保办公自动化需求得到充分满足。
4. **车辆保障：** 考虑到通山县乡镇分布广泛且部分山区路况复杂，我方承诺常驻配备至少2辆项目专用查勘车辆（含越野性能车辆），并根据核查高峰期（如年度复核、专项排查）需求，随时从总公司调配增援车辆，确保“车随人动”，偏远村组入户率达到100%。

二、常驻团队人员配置与资质承诺

人才是社会救助服务的核心。菲尔德咨询承诺组建一支“1+1+N”的复合型常驻服务团队，即“1名项目负责人+1名技术督导+N名专业核查员/社工”，全员实行全职驻点办公模式，严禁兼职或挂靠。

（一）项目负责人（1名）配置承诺
1. **资质要求：** 承诺选派具有5年以上社会救助项目管理经验、持有中级及以上社会工作师职业资格证书或管理类相关学位的资深人员担任项目经理。
2. **职责定位：** 项目经理全权负责与采购人的对接、团队内部管理、重大疑难个案研判及突发事件处置。
3. **驻点承诺：** 项目经理每月在通山县驻点工作时间不少于22个工作日，手机24小时开机，确保随时响应采购人指令。

（二）专职工作人员配置承诺
我方承诺根据通山县社会救助对象数量及服务频次要求，配置足额的专职工作人员（具体人数严格响应或优于招标文件要求），并优化人员结构：
1. **本地化策略：** 承诺驻点团队中，通山县本地户籍或长期在通山居住、通晓当地通山话方言的人员比例不低于80%。这有利于消除入户核查时的语言隔阂，更准确地了解申请对象的真实生活状况，提升群众接纳度。
2. **专业背景：** 承诺团队成员中，具有社会工作、社会学、心理学、法学、财务会计等相关专业背景的人员比例不低于60%。
3. **持证上岗：** 承诺所有入户核查人员在上岗前必须通过我方组织的“社会救助政策与实务”专项考试。项目执行期内，持有助理社会工作师及以上资格证书的人员占比不低于40%，并承诺在服务期内通过激励机制逐年提升持证率。
4. **岗位细分：**
    （1）**入户核查专员：** 负责实地入户，采集影像资料，填写核查表格，具备敏锐的观察力与良好的沟通技巧。
    （2）**信息比对专员：** 负责核对家庭经济状况信息，具备较强的数据分析能力与保密意识。
    （3）**档案管理员：** 负责纸质档案的整理、归档与电子化扫描，具备耐心细致的工作作风。
    （4）**内勤联络员：** 负责电话回访、满意度调查及日常报表报送。

（三）专家督导团队支持承诺
除常驻人员外，我方承诺组建由高校教授、资深民政专家组成的“飞行督导团”，每季度至少赴通山县驻点现场开展一次业务督导与疑难案例会诊，为驻点团队提供强大的后台智力支持。

三、人员稳定性与薪酬福利保障承诺

人员流动性大是社会服务行业的痛点。为确保通山县项目的连续性与稳定性，菲尔德咨询承诺建立具有竞争力的薪酬福利体系与留人机制，确保项目执行期内核心骨干人员流失率低于5%，全员流失率低于10%。

（一）薪酬待遇承诺
1. **薪资标准：** 承诺驻点人员平均薪资水平高于通山县当地同行业平均水平15%以上，并设置工龄工资与绩效奖金，确保员工收入具有吸引力。
2. **五险一金：** 承诺严格遵守《劳动法》，为所有驻点人员足额缴纳社会保险（养老、医疗、失业、工伤、生育）及住房公积金，免除员工后顾之忧。
3. **专项补贴：** 针对入户核查工作的辛苦程度，额外设置“高温补贴”、“交通补贴”、“下乡误餐补贴”及“通讯补贴”。

（二）人员替补与应急机制
1. **AB角工作制：** 承诺所有关键岗位（如档案管理、数据统计）实行AB角互备制度，确保A角休假或离岗时，B角能立即补位，工作不断档。
2. **人才储备池：** 我方承诺在公司总部建立不少于项目编制10%的机动人才储备库。一旦通山驻点团队出现人员离职或因病无法上岗，承诺在48小时内从储备库中调派具备同等资质的人员到岗顶替，且交接过渡期内由项目经理亲自顶岗，确保服务质量不降。
3. **末位淘汰与补充：** 对年度考核不合格的人员实行末位淘汰，并承诺在淘汰前已完成新人的招聘与培训，实现无缝轮换。

四、驻点团队管理制度与纪律承诺

我方将把菲尔德咨询成熟的ISO9001质量管理体系植入通山项目部，实行规范化、制度化管理。

（一）考勤与行为规范
1. **打卡制度：** 实行“钉钉”或“企业微信”定位打卡制度，包括上下班打卡及入户点位签到打卡，生成考勤报表按月报送采购人备案。
2. **着装规范：** 承诺所有工作人员在工作期间统一着装（佩戴印有“社会救助核查”字样的工作牌或穿戴统一马甲），展现专业形象，接受群众监督。
3. **文明用语：** 制定《入户核查文明用语规范》，严禁态度生硬、推诿扯皮，承诺对待困难群众做到“一张笑脸相迎、一声问候暖心”。

（二）廉洁自律与保密承诺
1. **廉洁承诺：** 严禁工作人员在核查过程中接受申请对象的吃请、礼品、礼金或有价证券。全体驻点人员入职即签署《廉洁自律承诺书》，若发现违规行为，一律开除并追究法律责任。
2. **保密承诺：** 深刻认识社会救助信息涉及公民隐私（如家庭收入、存款、残疾状况等）。我方承诺建立最严格的保密制度，所有员工签署《保密协议》，严禁将服务对象的任何信息泄露给第三方或用于非工作用途。若因我方原因造成信息泄露，愿承担一切法律后果及经济赔偿。

（三）安全责任承诺
1. **人身安全：** 针对入户过程中可能遇到的恶犬伤人、精神障碍患者攻击、交通事故等风险，我方承诺为每一位驻点人员购买高额的人身意外伤害保险（保额不低于80万元）。
2. **行车安全：** 建立车辆管理制度，定期保养车辆，严禁酒后驾车、疲劳驾驶，确保下乡核查交通安全。

五、服务响应时效与工作质量承诺

驻点团队的核心价值在于“快”与“准”。我方对服务响应速度与产出质量做出如下量化承诺：

（一）响应时效承诺
1. **常规任务：** 对于采购人下达的常规核查名单，承诺在接收后24小时内制定核查计划，3个工作日内完成入户核查并反馈初核结果。
2. **急难个案：** 针对突发性困难家庭（如急难型临时救助）或信访投诉案件，承诺建立“绿色通道”，接到指令后30分钟内响应，2小时内（偏远乡镇4小时内）到达现场，24小时内出具核查报告。
3. **日常沟通：** 承诺项目经理保持24小时通讯畅通，对于采购人的微信、电话咨询，做到“秒回”或10分钟内回复；对于需要整改的问题，承诺24小时内出具整改方案。

（二）工作质量承诺
1. **入户率承诺：** 承诺对新增低保、特困、低保边缘家庭的入户核查率达到100%；对在册对象的年度复核入户率严格按照文件要求执行，绝不以电话问询代替实地走访。
2. **档案合格率：** 承诺整理归档的“一户一档”资料（含申请书、承诺书、核对报告、入户记录、公示照片等）要素齐全、逻辑严密，档案合格率达到98%以上，并在各级检查考核中不失分。
3. **数据准确率：** 承诺录入系统的各类信息（姓名、身份证号、银行卡号、家庭经济状况）准确无误，数据录入差错率控制在1‰以内。
4. **群众满意度：** 承诺在入户核查及服务过程中，尊重服务对象人格，保护其隐私，确保服务对象对核查工作的满意度达到95%以上，有效投诉处理率100%。

六、培训赋能与能力提升承诺

为打造一支懂政策、会沟通、能操作的专业队伍，菲尔德咨询承诺建立常态化的培训体系：

（一）岗前培训（不少于3天）
在项目进驻前，对所有拟派驻点人员进行全封闭式岗前培训，考核合格方可上岗。培训内容包括：
1. **政策法规：** 深入学习《社会救助暂行办法》、湖北省及咸宁市、通山县最新的低保、特困、临时救助政策文件，确保精准把握认定条件。
2. **实务技巧：** 培训入户核查沟通技巧、家庭经济状况推算方法（如养殖业、种植业收入核算）、邻里访问技巧及矛盾化解方法。
3. **系统操作：** 熟练掌握社会救助信息管理系统的录入、查询、审核流程。

（二）在岗培训（每月1次）
承诺每月组织一次内部业务学习会，由项目经理或外聘专家授课。
1. **案例复盘：** 选取当月典型的疑难复杂案例进行剖析，统一认定口径。
2. **政策更新：** 及时传达学习上级民政部门发布的最新政策补丁或解释性文件。
3. **心理疏导：** 针对一线核查员可能产生的职业倦怠或心理压力，开展心理调适培训。

（三）以赛代练与技能比武
承诺每半年组织一次“社会救助业务技能大比武”，通过政策知识竞答、模拟入户演练、档案整理竞赛等形式，激发驻点人员钻研业务的热情，提升整体专业素养。

七、项目进驻与交接方案承诺

若我方中标，将把项目启动期作为重中之重，承诺实现“无缝衔接、平稳过渡”。

1. **T+1日（中标次日）：** 成立“通山项目筹备组”，项目经理到位，主动对接采购人，领取相关资料，确认办公场所。
2. **T+3日：** 完成办公场所租赁（或入驻）、网络布线、设备采购与安装调试；完成核心团队人员的集结与劳动合同签订。
3. **T+5日：** 完成所有驻点人员的岗前强化培训与考试；建立项目管理制度汇编；完成与原服务单位（如有）的资料交接或历史档案盘点。
4. **T+7日：** 举行简短的开工仪式，全员正式进入工作状态，开始接收第一批核查任务，确保项目运行首周即达到满负荷工作能力。

八、特殊时期的应急支援承诺

针对可能出现的特殊情况，我方做出如下额外承诺：
1. **大规模排查支援：** 若遇上级部门要求开展大规模专项整治或全员复核行动，仅靠驻点团队无法在规定时间内完成时，我方承诺在不增加采购人费用的前提下，从总部或其他项目点抽调不少于10人的突击队支援通山，直至任务完成。
2. **不可抗力应对：** 若发生重大自然灾害或公共卫生事件，驻点团队将就地转化为“民政应急志愿服务队”，无条件服从通山县民政局的统一调度，协助开展物资发放、受灾群众安置及心理抚慰工作，彰显企业的社会责任感。
