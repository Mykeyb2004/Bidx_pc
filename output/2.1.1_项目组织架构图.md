# 2.1.1 项目组织架构图

#### 2.1.1 项目组织架构图

**一、 组织架构设计总纲与核心理念**

为确保本次社会救助服务项目的高效、规范、精准实施，杭州菲尔德经济信息咨询有限公司（以下简称“菲尔德咨询”或“我司”）依据《社会救助暂行办法》及相关部、省、市级民政部门关于政府购买社会救助服务的指导意见，结合本项目“政策性强、敏感度高、覆盖面广、时效性严”的特点，量身定制了“垂直指挥、分级负责、矩阵协同、全域覆盖”的项目组织架构。

本组织架构设计的核心理念基于“三位一体”的管控逻辑：

（一） 政治引领与行政服从
社会救助工作是保障基本民生的“最后一道防线”，具有极强的政治属性。我司在架构顶层特别设立“党建引领与政策指导组”，确保项目执行始终与党和国家的民政方针保持高度一致，确保在具体核查与服务过程中，既有“法度”又有“温度”，无条件服从采购人的行政指令与监管要求。

（二） 扁平化指挥与网格化执行
鉴于社会救助对象居住分散、情况复杂，传统的多层级金字塔结构极易导致信息失真与响应滞后。为此，我司采取“项目经理—片区主管—一线社工”的三级扁平化管理模式，缩短指挥链条。同时，引入“网格化”作业理念，将服务区域划分为若干作业网格，实行定人、定岗、定责，确保“事事有人管、户户有人查”。

（三） 专业分工与全程质控
针对社会救助工作中“入户核查、信息比对、档案管理、个案服务”等不同环节的专业需求，我们在架构中实行“流水线式”的专业分工。设立专门的质量控制部门，独立于执行团队之外，行使“一票否决权”，构建起“自查、互查、督查”的闭环风控体系，确保每一份调查报告都经得起历史检验和审计追溯。

**二、 项目组织架构图谱及层级说明**

基于上述设计理念，菲尔德咨询构建了严密的金字塔型组织架构，主要由决策层、管理层、执行层、支持层四大板块构成。

（一） 决策层：项目领导小组
位于架构顶端，由我司总经理挂帅，分管副总及社会救助领域资深专家组成。主要负责项目的宏观把控、重大资源调配、突发危机应对及与采购人高层的战略对接。

（二） 管理层：项目经理办公室（PMO）
作为项目的“中枢神经”，实行项目经理负责制。下设项目总监（项目经理）、项目副经理及技术负责人。负责将采购人的需求转化为具体的执行计划，监控项目进度，协调各职能组运作，对项目最终交付质量负全责。

（三） 执行层：五大职能作业组
这是项目的“躯干”与“四肢”，直接面向服务对象开展工作：
1. **入户调查组：** 负责实地走访、入户核查、邻里访问，是“铁脚板”精神的践行者。
2. **信息核对组：** 负责经济状况核对、大数据比对分析、隐性收入评估。
3. **档案管理组：** 负责纸质与电子档案的整理、归档、扫描及保密管理。
4. **社工服务组：** 针对特困、低保边缘等特殊群体提供个案管理、资源链接及心理疏导。
5. **政策咨询组：** 负责窗口接待、政策解答、信访回复协助及疑难个案研判。

（四） 支持层：综合保障与质控体系
1. **质量控制组：** 独立行使监督职能，对业务流程进行全节点稽核。
2. **行政财务组：** 负责后勤保障、车辆调度、物资采购及项目财务管理。
3. **专家顾问团：** 聘请高校学者、民政退休干部组成智囊团，提供理论支撑与疑难指导。

**三、 各层级职能部门详细职责界定**

为确保组织架构图不仅仅停留在纸面上，而是切实转化为战斗力，我司对架构中每个节点的职责进行了颗粒化拆解：

（一） 项目领导小组职责
1. **战略决策：** 审定项目整体实施方案、应急预案及阶段性工作计划，确保项目方向不偏航。
2. **资源统筹：** 在公司层面调动人力、物力、财力及技术资源，优先保障本项目的需求，特别是在全员核查、年度复核等高峰期，确保人员力量充足。
3. **高层对接：** 负责与采购人领导层保持定期沟通，汇报项目进展，接收指导意见，及时消除合作中的宏观障碍。

（二） 项目经理（项目总监）职责
作为项目第一责任人，项目经理需具备“懂政策、善管理、能抗压”的综合素质：
1. **全面统筹：** 对项目的进度、质量、安全、成本进行全方位管理，签署项目相关文件，对采购人负责。
2. **团队建设：** 负责项目团队的组建、培训、考核与激励，营造积极向上的团队文化，确保队伍稳定性。
3. **流程优化：** 根据实际运行情况，动态调整作业流程，解决执行过程中的“堵点”与“难点”。
4. **风险管控：** 建立风险预警机制，对廉政风险、人身安全风险、数据保密风险进行前置管理。
5. **绩效汇报：** 按月、季、年向采购人提交工作报告，组织召开阶段性总结会议。

（三） 技术负责人（督导）职责
1. **标准制定：** 制定统一的入户核查话术、调查表格填写规范、档案整理标准，确保“千人一面、万卷一律”。
2. **业务培训：** 建立常态化培训体系，组织岗前培训、政策更新培训及案例分析会，提升全员业务素质。
3. **疑难攻坚：** 牵头处理“人户分离、隐性就业、家庭财产认定”等疑难复杂个案，出具专业认定建议。

（四） 入户调查组职责
这是本项目人数最多、任务最重的团队，实行“双人成组、全员定位”管理：
1. **实地核查：** 严格执行“看、问、查、访”四步工作法，对申请对象及在册对象的家庭人口、收入、财产、健康状况进行全面核实。
2. **邻里走访：** 通过走访村（居）委会、邻居、网格员，侧面印证申请人家庭真实生活状况，排查是否存在隐瞒实情现象。
3. **轨迹留痕：** 利用移动终端设备，对入户过程进行拍照、定位打卡，确保“真入户、入真户”，杜绝闭门造车。
4. **主动发现：** 在走访过程中，敏锐捕捉辖区内未申请但在贫困边缘的“沉默少数”，及时上报，落实“主动发现”机制。

（五） 信息核对组职责
1. **数据采集：** 协助采购人采集、录入、清洗救助对象的基础信息，确保系统数据与纸质档案一致。
2. **报告分析：** 对居民家庭经济状况核对报告（核对单）进行深度解读，识别异常数据（如车辆、房产、社保缴纳记录等）。
3. **算账核实：** 根据赡养/抚养关系、劳动力状况、行业平均工资等要素，科学测算家庭人均收入，出具客观的收入核算清单。

（六） 质量控制组职责
作为项目的“纪委”与“审计”，直接向项目领导小组汇报：
1. **全量审核：** 对入户调查表、核对报告、档案资料进行100%形式审核，确保要素齐全、逻辑自洽。
2. **抽样复核：** 按不低于30%的比例进行电话回访或实地复核，检验入户调查的真实性与准确性。
3. **错案追究：** 发现弄虚作假、敷衍塞责行为，立即启动问责程序，对相关责任人进行处罚，并督促整改。
4. **满意度调查：** 定期对服务对象进行回访，收集对服务态度、廉洁自律方面的反馈。

（七） 档案管理组职责
1. **一户一档：** 按照“分类、分级、分年度”原则，建立规范的社会救助档案体系，确保每户资料索引清晰、查找便捷。
2. **电子化扫描：** 同步进行纸质档案的数字化扫描与挂接，实现档案管理的现代化与信息化。
3. **安全保密：** 严格执行档案库房管理制度（防火、防盗、防潮、防虫）及信息保密制度，严禁泄露救助对象隐私。

（八） 专家顾问团职责
1. **政策把脉：** 针对政策边缘地带或新兴问题（如网络直播收入认定、虚拟货币资产认定等），提供理论依据与政策解释建议。
2. **第三方评估：** 参与年度绩效评价，以第三方视角对项目执行效果进行客观评判，提升项目公信力。

**四、 组织架构的运行机制与协同逻辑**

一个优秀的组织架构不仅要有静态的“形”，更要有动态的“神”。我司通过以下四大机制，驱动各部门高效运转：

（一） 纵向指挥链：指令必达机制
1. **日例会制度：** 各小组每日晨会部署当日任务，晚会复盘当日进度与问题，确保“日事日毕”。
2. **周报月报制：** 每周五汇总本周工作数据与下周计划，报项目经理审阅；每月初向采购人提交月度服务专报。
3. **红黄灯预警：** 对即将超时的核查任务、未闭环的审批流程实行系统自动预警（黄灯催办、红灯督办），确保时效合规。

（二） 横向协同链：矩阵作业机制
1. **查核联动：** 入户调查组在发现疑点（如家中装修豪华但申报无收入）后，即时反馈给信息核对组进行定向数据深挖；反之，信息核对组发现大额存款记录后，指令入户组重点核实资金来源。
2. **档业融合：** 档案组实时跟踪业务进度，业务办理完毕即刻归档，避免“先干活、后补档”造成的资料缺失。
3. **社工介入：** 在核查中发现单纯物质救助无法解决困难的家庭（如重残无人照料、心理障碍等），立即转介社工服务组，启动“物质+服务”类救助。

（三） 应急响应链：快速动员机制
1. **AB角替补：** 关键岗位（如项目经理、数据专员）均设置AB角，A角休假或离岗时，B角无缝顶岗，确保工作不中断。
2. **潮汐用工：** 针对年度复核、政策扩面等业务高峰期，启动预备队机制，从公司人才库中紧急抽调经过培训的熟手支援，确保不积压、不延误。
3. **突发事件处置：** 遇到服务对象过激维权、突发重大舆情或自然灾害等情况，项目领导小组直接接管指挥，启动I级响应，2小时内到达现场，24小时内出具处置报告。

（四） 监督反馈链：自我净化机制
1. **廉政防火墙：** 所有项目人员入职前签署《廉洁自律承诺书》，架构中设立专职廉政监督员，定期开展廉政谈话。
2. **全过程留痕：** 所有业务操作均在系统中留痕，所有入户均有录音录像，确保权力在阳光下运行，任何违规操作皆可追溯。

**五、 组织架构的人员配置策略与资质要求**

为支撑上述架构，菲尔德咨询坚持“专业人做专业事”的配置原则，对关键节点人员资质设定了高于招标文件的标准：

（一） 项目经理配置标准
1. **学历与职称：** 必须具备本科及以上学历，持有中级及以上社会工作师职业资格证书。
2. **经验要求：** 拥有5年以上民政社会救助领域项目管理经验，累计管理服务对象不少于1万人次。
3. **能力画像：** 熟悉本地民政政策，具备优秀的沟通协调能力、公文写作能力及突发事件处置能力。

（二） 核心团队配置标准
1. **持证上岗：** 项目组核心成员（组长级）100%持有助理社会工作师或以上证书；一线核查员持证率不低于60%。
2. **专业背景：** 优先录用社会学、社会工作、公共管理、法律、财务等相关专业人员。
3. **本地化优势：** 一线入户调查员优先聘用懂本地方言、熟悉本地风土人情的人员，以消除沟通隔阂，提高入户成功率。
4. **政治面貌：** 党员比例不低于20%，关键岗位优先由党员担任，充分发挥党员先锋模范作用。

（三） 督导与顾问配置
聘请不少于2名省/市级社会救助领域专家作为外部督导，每月至少开展1次现场督导或案例研讨，确保服务技术的专业性与前瞻性。

**六、 组织架构的优势分析与保障承诺**

菲尔德咨询所设计的这套组织架构，相较于传统模式，具有显著的竞争优势，能够最大程度保障采购人的利益：

（一） 优势一：闭环风控，合规无忧
通过设立独立的“质量控制组”，实现了“执行”与“监督”的物理隔离。这种分权制衡的架构设计，能够有效规避“人情保”、“关系保”、“错保”、“漏保”等行业顽疾，最大限度降低采购人的履职风险与审计风险。

（二） 优势二：弹性伸缩，响应高效
“核心团队+预备梯队+专家外脑”的同心圆结构，使得项目具备极强的抗压能力。无论是日常的动态管理，还是突击性的专项排查，组织架构都能迅速调整队形，确保任务按时保质完成。

（三） 优势三：数智赋能，精准施策
架构中专门强化的“信息核对”与“数据分析”职能，顺应了“数字民政”的发展趋势。我们不仅提供人力服务，更提供数据智慧，通过精准画像，助力政府实现从“人找政策”向“政策找人”的转变。

（四） 优势四：五社联动，服务升级
通过“社工服务组”的嵌入，我们将传统的“入户核查”升级为“综合服务”。工作人员既是“核查员”，又是“政策宣传员”和“资源链接者”，有效推动社区、社会组织、社会工作者、社区志愿者、社会慈善资源的“五社联动”，提升社会救助的温度与厚度。
