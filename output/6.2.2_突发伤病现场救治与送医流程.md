# 6.2.2 突发伤病现场救治与送医流程

#### 6.2.2 突发伤病现场救治与送医流程

在社会救助项目执行过程中，一线调查员、社工及核查人员需频繁深入街道、乡镇、村居及特殊困难家庭进行入户走访与核查工作。面对复杂多变的交通环境、由于老旧小区或偏远农村地理环境带来的物理风险，以及高强度工作可能引发的突发健康问题，建立一套科学、严谨、高效的“突发伤病现场救治与送医流程”是保障员工生命安全、体现杭州菲尔德经济信息咨询有限公司（以下简称“菲尔德咨询”）人文关怀与风险控制能力的核心举措。

本方案依据《中华人民共和国安全生产法》、《工伤保险条例》及急救医疗相关规范，结合社会救助工作实际场景，制定了从现场应急响应、初步救治、紧急送医到院内对接的全链条闭环管理流程。

**一、 总体原则与响应机制**

菲尔德咨询在处理员工突发伤病时，始终遵循“生命至上、快速响应、科学施救、全程保障”的十六字方针。

（一） 核心处置原则
1. **生命第一原则：** 在任何情况下，员工的生命安全与身体健康高于一切项目进度与财产利益。一旦发生突发伤病，立即暂停当前工作，全力投入救治。
2. **就近救急原则：** 优先选择距离最近、具备相应救治能力的医疗机构进行抢救，待病情稳定后再根据医疗建议考虑转院。
3. **专业处置原则：** 现场急救必须由经过红十字会急救培训持证人员或在120调度员指导下进行，严禁盲目搬动或错误施救造成二次伤害。
4. **全额保障原则：** 公司设立“员工突发伤病应急备用金”，实行“先救治、后结算”制度，确保不因费用问题延误抢救时机。

（二） 黄金救援响应时效
为确保救援效率，项目组设定了严格的时间节点指标：
1. **1分钟响应：** 现场同伴或当事人在事故发生后1分钟内完成自我判断并呼叫救援（拨打120及上报项目经理）。
2. **5分钟决策：** 项目应急小组在接报后5分钟内完成情况研判，启动应急预案，划拨备用金，并指派专人赶赴现场或医院。
3. **全程陪护：** 从事故发生到家属抵达医院前，公司必须保证至少有一名管理人员全程陪同，不得让受伤员工处于无人照料状态。

**二、 现场急救组织体系与职责分工**

为确保救治流程有序进行，菲尔德咨询构建了“现场-项目部-总部”三级联动的急救组织体系。

（一） 现场第一响应人（通常为同组搭档）
作为距离伤病员最近的人员，其职责至关重要：
1. **环境评估：** 迅速确认现场环境是否安全（如是否存在漏电、交通危险、攻击性动物等），确保自身及伤员不再受二次伤害。
2. **初步判断：** 快速检查伤员意识、呼吸、脉搏及外伤出血情况。
3. **紧急呼救：** 立即拨打120急救电话，准确表述地点、伤情及联系方式；同时向项目经理汇报。
4. **基础急救：** 在专业医护人员到达前，利用随身携带的急救包进行止血、包扎或心肺复苏（CPR）等力所能及的救护。

（二） 项目应急处置小组
由项目负责人担任组长，负责统筹指挥：
1. **资源调配：** 协调车辆、人员赶赴现场支援。
2. **医疗对接：** 提前查询并联系最近的定点合作医院或三甲医院急诊科，请求开通绿色通道。
3. **家属联络：** 在掌握基本伤情后，第一时间通知员工家属，并做好安抚工作。

（三） 公司总部支持中心
由行政人事部与财务部组成：
1. **资金保障：** 立即通过即时转账方式向现场陪护人员支付急救备用金。
2. **保险报案：** 在事故发生24小时内向保险公司进行工伤或意外险报案。
3. **法律与舆情支持：** 处理可能涉及的第三方责任纠纷及对外信息发布。

**三、 突发伤病分级与标准化处置流程**

针对社会救助调查工作中可能遇到的不同程度伤病，菲尔德咨询制定了分级处置标准流程（SOP）。

（一） Ⅰ级：危重急症（生命体征不稳定）
**适用场景：** 突发心搏骤停、严重车祸导致的大出血/昏迷、高处坠落、重度中暑（热射病）、急性心肌梗死等。

1. **立即呼叫：** 现场人员必须在第一时间拨打120，并开启免提模式，接受急救中心调度员的在线指导。
2. **体位管理：**
   - 若无脊柱损伤嫌疑，将患者置于复苏体位（仰卧位）；
   - 若昏迷且有呼吸，置于侧卧位以防呕吐物窒息；
   - 若怀疑脊柱损伤，严禁随意搬动，需维持原状等待专业救援。
3. **心肺复苏（CPR）介入：** 若发现患者无意识、无呼吸（或仅有濒死叹息样呼吸），现场持证急救员应立即实施CPR，按压频率100-120次/分，按压深度5-6厘米，直至医护人员接手或患者恢复自主呼吸。
4. **AED使用：** 若事发地点（如社区服务中心、地铁站）配备有自动体外除颤器（AED），应立即获取并按照语音提示操作。
5. **协助转运：** 救护车到达后，协助医护人员搬运，并由项目组指派专人随车前往医院。

（二） Ⅱ级：严重伤病（生命体征暂时稳定但需紧急治疗）
**适用场景：** 骨折、大面积软组织挫伤、动物咬伤（出血较多）、突发急腹症、严重眩晕等。

1. **制动与固定：** 对于骨折患者，利用现场夹板或硬纸板进行简单固定，避免骨折端刺破血管神经；对于出血部位，进行加压包扎止血。
2. **动物咬伤专项处理：** 若在入户途中被犬只咬伤，应立即用肥皂水和流动清水交替冲洗伤口至少15分钟（若条件允许），严禁包扎伤口，随后立即送往最近的狂犬病疫苗接种点或医院。
3. **转运方式决策：** 根据伤情决定是否呼叫120。若伤情允许坐车且等待救护车时间过长，可由项目组安排专车或出租车迅速送医，但在转运途中必须保持与医院的联系。

（三） Ⅲ级：轻微伤病（不影响生命安全）
**适用场景：** 轻微擦伤、扭伤（未骨折）、轻度中暑、感冒发烧等。

1. **现场处理：** 使用随身携带的“菲尔德外勤急救包”中的碘伏、创可贴、云南白药气雾剂、防暑药品等进行自行处理。
2. **观察休息：** 停止当前高强度工作，在阴凉通风处休息观察30分钟。
3. **报备与就医：** 向项目经理报备，若症状未缓解，由同事陪同前往社区卫生服务中心或附近医院诊治。

**四、 送医转运与院内绿色通道对接**

送医过程并非简单的运输，而是救治链条的延伸。菲尔德咨询建立了完善的送医与院内对接机制。

（一） 最优送医路线规划
项目启动前，项目组将通过高德/百度地图大数据，绘制项目辖区内的“医疗救助地图”，标注所有三甲医院、二级医院及社区卫生服务中心的位置及急诊电话。
1. **路线选择：** 发生突发事件时，利用导航软件避开拥堵路段，选择耗时最短的路线。
2. **医院选择：**
   - 创伤类：优先送往具备骨科、外科优势的综合性医院。
   - 心脑血管类：优先送往具备胸痛中心、卒中中心资质的医院。
   - 传染病/特殊疾病：送往当地指定的定点医院。

（二） 随车陪护人员职责清单
随车陪护人员（通常为项目主管或资深员工）需携带员工身份证件、医保卡（如有）及应急备用金，并在途中完成以下工作：
1. **信息采集：** 记录伤病员发病时间、诱因、既往病史、过敏史，以便向医生提供准确信息。
2. **实时汇报：** 每隔10分钟向项目经理汇报位置及患者状态。
3. **心理安抚：** 若患者清醒，持续给予语言鼓励，缓解其恐惧心理。

（三） 院内“绿色通道”对接流程
1. **预检分诊：** 到达医院后，立即配合分诊护士进行生命体征测量，准确描述病情，争取最高优先级的诊疗序列。
2. **挂号与缴费：** 陪护人员利用公司应急备用金迅速完成建档、挂号及检查费用的缴纳。若遇费用不足，立即联系公司财务部进行紧急转账，或请求医院先行救治（由公司出具担保函）。
3. **协助检查：** 陪同伤员进行CT、X光、抽血等检查，保管好所有检查单据及影像资料。
4. **办理住院：** 如需住院，协助办理入院手续，购买必要的生活用品（脸盆、毛巾等），直至家属到达并完成交接。

**五、 典型场景下的专项救治预案**

结合民政社会救助工作的特殊性，菲尔德咨询针对三种高频风险场景制定了专项救治预案。

（一） 场景一：高温酷暑入户导致的热射病（重症中暑）
社会救助核查常需在夏日午后进行，极易发生中暑。
1. **识别：** 员工出现高热（体温>40℃）、无汗、意识模糊、抽搐。
2. **即刻降温：** 这是抢救成功的关键。立即将患者转移至空调房或阴凉处，脱去外衣，用湿毛巾擦拭全身，或用凉水喷洒，配合风扇吹风，力争在30分钟内将核心体温降至38℃以下。
3. **补水禁忌：** 若患者意识不清，严禁喂水，防止误吸入肺；若意识清醒，可少量多次饮用淡盐水。
4. **紧急送医：** 热射病死亡率高，必须呼叫120送往具备ICU救治能力的医院。

（二） 场景二：老旧小区楼道踩空或滑倒导致的骨折/外伤
低保户多居住在无电梯、光线昏暗的老旧小区。
1. **制动原则：** 严禁按摩、热敷或强行站立行走。
2. **冷敷处理：** 24小时内对患处进行冰敷（使用冰袋或冰镇饮料包裹毛巾），减少出血和肿胀。
3. **抬高患肢：** 将受伤肢体抬高至心脏水平以上。
4. **搬运注意：** 若怀疑脊柱或颈椎受损，必须使用硬担架或门板搬运，保持头颈躯干成一直线，严禁“一人抱头、一人抱脚”的错误搬运方式。

（三） 场景三：入户调查途中发生交通事故
1. **现场保护：** 在确保安全前提下，开启危险报警闪光灯，在车后设置警示标志，拍摄现场照片保留证据。
2. **伤情评估：** 重点检查是否有隐蔽性内脏损伤（如腹痛、便血）或脑震荡症状（恶心、呕吐、逆行性遗忘）。
3. **警医联动：** 同时拨打120和122，告知交警系工伤事故，请求快速处理。
4. **后续观察：** 即使表面无明显外伤，凡经历剧烈撞击的员工，必须强制送医进行全身体检（特别是头部CT和腹部B超），留院观察至少24小时。

**六、 费用保障与工伤申报闭环**

为解除员工救治的后顾之忧，菲尔德咨询实行完善的费用保障与行政闭环管理。

（一） 应急备用金管理
1. **额度设定：** 每个项目部设立不少于20,000元的突发伤病应急备用金专户。
2. **启用流程：** 遇紧急情况，项目经理有权直接支取使用，事后凭医院单据补办财务手续。
3. **支付范围：** 涵盖挂号费、检查费、治疗费、药费、救护车费、住院押金及必要的护理用品费。

（二） 工伤认定与保险理赔
1. **证据保全：** 现场陪同人员需收集并保存好门诊病历、诊断证明、检查报告、医疗发票原件及费用清单。
2. **事故调查：** 事故发生后24小时内，项目经理需填写《员工突发伤病/工伤事故报告单》，详述事发经过、时间、地点及原因，并附现场照片或证人证言（如入户对象的证明）。
3. **申报流程：**
   - **工伤认定：** 公司人事部在事故发生30日内向当地人社局提交工伤认定申请。
   - **商保理赔：** 同步启动雇主责任险或人身意外险的理赔程序，作为工伤保险的补充。

（三） 后续关怀与复工评估
1. **探视慰问：** 员工住院期间，公司领导及工会代表将前往医院探视慰问，了解治疗进展及家属困难。
2. **心理干预：** 针对经历严重创伤或目睹惨烈事故的员工，公司聘请专业心理咨询师提供心理疏导，预防创伤后应激障碍（PTSD）。
3. **复工评估：** 员工康复申请复工前，需提供医疗机构出具的康复证明。公司将根据其身体状况，实行“过渡期轻负荷工作制”，初期安排内业资料整理等低强度工作，待完全适应后再恢复外勤工作。

**七、 预案演练与技能培训**

“宁可备而不用，不可用而无备”。菲尔德咨询将急救能力作为项目团队的核心素质进行培养。

1. **全员急救培训：** 项目启动后，邀请红十字会或急救中心专家对所有外勤人员进行急救技能培训，重点讲授CPR（心肺复苏）、AED使用、海姆立克急救法及外伤包扎技术。确保项目组持证急救员比例不低于30%。
2. **急救物资配备：** 为每位外勤人员配备便携式急救包（含创可贴、酒精棉片、纱布、风油精、速效救心丸等），并要求每日出勤前检查补充。
3. **实战模拟演练：** 每季度组织一次“突发伤病应急演练”，模拟入户途中晕倒、狗咬伤等场景，考核从呼救、施救到送医的全流程反应速度，发现漏洞及时修订预案。

通过上述严密、专业的现场救治与送医流程，菲尔德咨询承诺：无论在繁华市区还是偏远乡村，我们都有能力、有机制、有资源为每一位为社会救助事业奉献的员工提供最坚实的安全屏障，确保项目平稳、安全、高效运行。
