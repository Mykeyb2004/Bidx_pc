# 3.3.1 联席会议制度（定期汇报与政策研讨）

#### 3.3.1 联席会议制度（定期汇报与政策研讨）

为确保本项目各项社会救助服务工作能够精准对接采购人需求，实现对项目进度的实时掌控、对服务质量的动态监测以及对救助政策的准确把握，杭州菲尔德经济信息咨询有限公司（以下简称“菲尔德咨询”或“我方”）将建立全方位、多层次、常态化的联席会议制度。该制度不仅是双方沟通的桥梁，更是项目风险防控、政策落地执行与服务效能提升的核心抓手。我方将本着“主动汇报、深度研讨、闭环管理、高效协同”的原则，通过周例会、月度分析会、季度总结会及政策专题研讨会等多种形式，构建起一套严密的沟通协作体系。

**一、 总体设计原则与组织架构**

**（一） 设计原则**

1.  **政治引领，政策为基。** 始终坚持将民政部门的最新指示精神作为联席会议的核心议题，确保项目执行不偏航、不走样，严格遵守社会救助相关法律法规。
2.  **问题导向，精准施策。** 联席会议不走过场，直面项目执行中的痛点、难点与堵点（如人户分离核查难、隐性收入认定难等），通过集体研判提出切实可行的解决方案。
3.  **数据驱动，科学决策。** 汇报内容不依靠主观臆断，而是基于入户核查率、档案合格率、群众满意度等量化指标，用数据说话，为采购人决策提供科学依据。
4.  **闭环管理，狠抓落实。** 坚持“凡事有交代，件件有着落，事事有回音”，会议决议必须形成督办清单，实行销号管理。

**（二） 组织架构与人员配置**

为确保联席会议的高规格与实效性，我方将成立由项目总监领衔的专项对接工作组，实行“三级对接”机制：

1.  **决策层对接：** 由我方公司分管副总或项目总监出席，主要对接采购人分管领导及科室负责人。负责重大事项决策、突发事件处置及季度/年度工作汇报。
2.  **管理层对接：** 由我方驻场项目经理、运营主管出席，主要对接采购人具体业务经办人员。负责日常工作汇报、月度数据分析、重难点个案研判。
3.  **执行层对接：** 由我方各街道/乡镇核查组长、数据专员出席，根据会议议题需要列席。负责具体核查情况反馈、系统数据录入问题沟通。

**二、 常态化汇报机制（定期汇报）**

我方将建立“周-月-季”的时间维度汇报体系，确保采购人随时掌握项目动态。

**（一） 周例会制度：紧盯进度，快速响应**

1.  **会议时间：** 原则上定为每周一上午（或根据采购人方便时间确定）。
2.  **参会人员：** 我方项目经理、各片区组长；采购人科室经办人员。
3.  **汇报重点：**
    （1）**上周工作复盘：** 详细汇报上周入户核查的具体户数、人数，包括低保、特困、低保边缘家庭及临时救助等各类型的分类统计；汇报上周系统数据维护情况及档案整理进度。
    （2）**本周工作计划：** 明确本周的核查路线图、重点攻坚对象及预估工作量，确保任务分解到人、责任落实到岗。
    （3）**异常情况预警：** 针对上周核查中发现的“疑似骗保”、“人户分离失联”、“家庭财产状况存疑”等异常个案进行即时通报，申请相关部门协助或联合执法。
4.  **输出成果：** 《周工作简报》，包含核心数据图表及本周重点待办事项清单。

**（二） 月度分析会制度：深度复盘，质量把控**

1.  **会议时间：** 每月第4个工作日（结合月度报表报送时间）。
2.  **参会人员：** 我方项目总监、项目经理、质控专员；采购人科室负责人及经办人员。
3.  **汇报重点：**
    （1）**月度数据分析：** 运用大数据思维，对当月救助对象的增减趋势、资金发放规模、致贫原因分布（因病、因残、因学等）进行多维度分析，生成可视化图表，为采购人掌握辖区困难群众底数提供直观参考。
    （2）**质量控制报告：** 我方质控专员汇报当月内部抽查情况，包括入户调查表的规范性、佐证材料的完整性、系统录入的准确性等，敢于“自揭家丑”，并提出整改措施。
    （3）**重难点个案会商：** 选取当月遇到的3-5个典型疑难个案（如赡养费计算争议、车辆价值评估争议等），提交会议进行集体会商，依据政策条款通过“一事一议”确定认定口径。
4.  **输出成果：** 《月度项目运营报告》，包含详尽的数据分析、质控报告及下月改进方案。

**（三） 季度总结会制度：绩效评估，策略调整**

1.  **会议时间：** 每季度末或下季度初。
2.  **参会人员：** 我方公司高层领导、项目总监、专家顾问；采购人分管领导、科室负责人。
3.  **汇报重点：**
    （1）**阶段性绩效自评：** 对照招标文件及合同约定的服务指标（如核查时效性、对象准确率、群众满意度等）进行全面自评，查找差距与不足。
    （2）**服务模式优化：** 结合季度执行情况，对我方的服务流程、人员配置、技术手段进行反思与优化。例如，若发现某街道申请量激增，建议增加驻点人员力量；若发现核查效率低，建议优化入户路线算法。
    （3）**外部环境分析：** 汇报在服务过程中收集到的街道、社区及群众的意见反馈，分析社会舆情风险点，提出针对性的维稳建议。
4.  **输出成果：** 《季度绩效评估报告》及《下一阶段工作实施方案》。

**三、 政策研讨与智库支持机制**

社会救助工作政策性强、敏感度高，且随着“分层分类”社会救助体系建设的推进，政策更新迭代快。为此，菲尔德咨询将把“政策研讨”作为联席会议的重要组成部分，发挥“外脑”与“智库”作用。

**（一） 政策学习与解读机制**

1.  **新政速递与解读：** 每当民政部、省民政厅或市民政局出台新的社会救助文件（如低保扩围增效、特困人员认定办法修订等），我方政策研究组将在24小时内完成文件研读，并编制《政策解读要点》，在联席会议上进行专题汇报。
2.  **执行口径统一：** 针对政策文件中可能存在的模糊地带或自由裁量权空间，通过联席会议与采购人共同探讨，结合本地实际情况，制定统一的《实务操作手册》或《疑难问题解释口径》，确保全区（县）救助认定标准的一致性，避免出现“同城不同遇”现象。
3.  **案例教学与复盘：** 收集整理国家及省市发布的社会救助典型案例（含审计署查出的违规案例及优秀创新案例），在会议上进行剖析，以案说法，提升双方人员的政策运用能力和风险防范意识。

**（二） 专题研讨会制度**

针对特定时期或特定任务，不定期召开专题研讨会：

1.  **年度核查方案研讨：** 在年度复核或专项治理行动开始前，专门召开会议，研讨核查范围、时间节点、力量部署及应急预案，确保大规模行动有序开展。
2.  **扩围增效专题研讨：** 针对低保边缘家庭认定、支出型贫困救助等政策扩面难点，通过会议研讨如何利用“大数据比对+铁脚板摸排”建立主动发现机制，实现“人找政策”向“政策找人”转变。
3.  **审计迎检专题研讨：** 在面临上级绩效评价或审计检查前，召开迎检准备会，对照审计重点对档案资料、资金台账进行模拟预检，查漏补缺。

**四、 会议流程标准化管理**

为确保联席会议的高效性，我方制定了严格的“会前-会中-会后”全流程管理规范。

**（一） 会前准备：充分详实**

1.  **议题征集：** 会前2个工作日，我方项目经理向采购人及我方执行团队征集会议议题，确保问题聚焦。
2.  **材料编制：** 会前1个工作日，我方负责编制汇报PPT、数据报表、案例材料及政策汇编，并发送给采购人预审，确保会议材料内容详实、数据准确。
3.  **日程确认：** 提前确认参会领导及人员行程，发送会议通知（含时间、地点、议程），确保关键人员到场。

**（二） 会中组织：严谨高效**

1.  **主持人机制：** 常规例会由我方项目经理主持，重大节点会议由我方项目总监或邀请采购人领导主持。
2.  **限时汇报：** 坚持“开短会、讲干货”，汇报工作直奔主题，讲问题不遮掩，讲对策有条理，避免长篇大论。
3.  **充分讨论：** 预留充足时间给采购人进行点评和提问，对提出的每一个疑问进行现场解答，不能现场解答的承诺反馈时限。
4.  **会议记录：** 我方指派专人担任会议记录员，对会议发言、讨论过程、决策事项进行全过程详细记录，必要时进行录音备查。

**（三） 会后落实：闭环销号**

1.  **纪要整理：** 会后24小时内，我方整理出《会议纪要》，明确会议决议、待办事项、责任人及完成时限，经双方签字确认后归档。
2.  **督办执行：** 建立《联席会议决议督办台账》，我方项目经理负责跟踪每一项决议的落实情况。
    *   对于一般性事务（如补充某户资料），要求3个工作日内完成；
    *   对于系统性调整（如优化核查流程），要求1周内出具方案并试行；
    *   对于政策性难题，持续跟踪直至解决。
3.  **结果反馈：** 在下一次联席会议上，首先通报上一次会议决议的落实情况，形成“布置-执行-反馈-考核”的完整闭环。

**五、 应急沟通与特殊情况处置机制**

除了常规的联席会议，针对社会救助工作中可能出现的突发紧急情况，我方建立“24小时响应”的应急沟通通道。

1.  **突发事件直报：** 若在服务过程中遇到救助对象发生重大变故（如突发重病、意外死亡、极端个案威胁工作人员等）或涉及媒体舆情的敏感事件，我方承诺在30分钟内通过电话、微信等方式向采购人分管领导口头汇报，2小时内提交书面《突发事件情况说明》。
2.  **临时调度会议：** 针对突发事件，随时响应采购人召唤，召开临时调度会，启动应急预案，协助采购人做好现场安抚、调查取证及舆情应对工作，将负面影响降至最低。

**六、 档案管理与保密要求**

联席会议的所有资料均属于项目重要档案，我方将严格按照档案管理和保密规定执行。

1.  **分类归档：** 建立专门的“联席会议档案盒”，按照时间顺序，将历次会议的通知、签到表、汇报PPT、会议纪要、督办单及整改报告分类装订，做到“一会一档”。
2.  **数字化留存：** 所有纸质材料同步扫描电子版，存储于加密硬盘中，作为项目验收及后续审计的重要依据。
3.  **严格保密：** 会议内容涉及困难群众隐私及政府内部决策信息的，参会人员均需签署保密承诺书。严禁在非工作场合讨论会议内容，严禁将会议材料外泄给第三方。

**七、 预期成效**

通过实施上述联席会议制度，菲尔德咨询承诺达到以下预期成效：

1.  **信息零时差：** 确保采购人对项目进度的知晓率达到100%，消除信息不对称带来的管理盲区。
2.  **政策零偏差：** 通过高频次的政策研讨，确保所有入户核查与审批辅助工作完全符合国家及省市最新政策要求，合规率达到100%。
3.  **问题零积压：** 实现执行层面的问题“早发现、早汇报、早解决”，疑难个案处置及时率显著提升。
4.  **配合零缝隙：** 构建起“政企一家人、工作一条心”的良好合作氛围，形成强大的工作合力，共同推动本地区社会救助事业的高质量发展。

综上所述，菲尔德咨询制定的联席会议制度，不仅仅是一个沟通形式，更是一套集“进度管控、质量监督、政策智库、风险防范”于一体的综合管理体系。我们将以高度的责任感和专业度，严格执行该制度，确保向采购人交出一份满意的答卷。
