# 5.4.3 离职审批与工作交接清单

#### 5.4.3 离职审批与工作交接清单

在社会救助服务项目中，人员的稳定性直接关系到对困难群众服务的连续性与政策执行的精准度。鉴于民政工作的特殊性——涉及大量困难家庭隐私数据、复杂的个案历史记录以及严格的财政资金审核要求，杭州菲尔德经济信息咨询有限公司（以下简称“菲尔德咨询”）制定了极为严苛且周密的《离职审批与工作交接管理制度》。本制度旨在通过标准化的审批流程、清单化的交接内容以及闭环式的责任追溯机制，确保在人员变动时实现“服务不中断、档案不缺失、数据不泄露、资产不流失”，保障项目平稳运行。

**一、 离职审批管理总则与流程规范**

菲尔德咨询坚持“依法合规、预警在前、有序流动”的原则，严格遵循《中华人民共和国劳动合同法》及公司内部管理规定，实施分级审批与分类管理。

**（一） 离职类型的界定与预告期管理**

为确保项目组有充足的时间进行人员替补与业务过渡，公司对不同类型的离职实行差异化管理：

1.  **正常辞职：** 处于试用期的员工，需提前3日提交书面申请；正式员工需提前30日以书面形式通知公司。此预告期用于启动招聘替补程序及整理交接清单。
2.  **协商解除：** 经公司与员工双方协商一致解除劳动合同的，根据协商确定的日期办理离职手续，但原则上交接期不得少于5个工作日，以确保核心数据的完整移交。
3.  **即时解除（辞退）：** 对于严重违反公司规章制度或造成重大工作失误（如造成严重舆情、重大数据泄露）的员工，公司有权即时解除劳动合同。此类情况启动“紧急交接程序”，由项目负责人与行政人事部直接介入，当日完成权限冻结与资产回收。

**（二） 标准化审批流程**

离职审批并非简单的签字过程，而是对员工在职期间工作状态的终结确认与风险评估过程。

1.  **提交申请：** 员工填写《员工离职申请表》，注明离职原因、拟离职日期，并附简要的未结事项说明。
2.  **直属上级面谈与初审：** 项目主管（或街道/乡镇服务站站长）在收到申请后24小时内进行离职面谈。面谈重点在于了解真实离职原因，评估该员工离职对辖区低保核查、特困探访等工作的影响，并尝试挽留核心骨干。
3.  **项目经理复核：** 项目经理根据项目整体人员配置情况，审核离职日期的合理性。对于关键岗位（如数据核对专员、重难点个案社工），项目经理有权要求延长交接期（在法律允许范围内协商），直至替补人员到岗。
4.  **行政人事部审核：** 核查员工考勤、假期余额、合同期限及是否存在未结的培训协议或竞业限制条款。
5.  **总经理/分管副总审批：** 对于项目核心管理人员或涉及敏感数据的岗位，需经公司高层最终审批。

**二、 离职面谈与风险防控机制**

离职面谈是菲尔德咨询完善管理、规避风险的重要环节，也是体现人文关怀的最后窗口。

**（一） 面谈内容的深度与广度**

1.  **原因探究：** 深入了解员工离职的真实动因（如薪资、职业发展、家庭原因或工作压力），为公司优化薪酬体系和管理制度提供数据支持。
2.  **合规宣导：** 再次向员工宣讲保密义务，特别是关于民政服务对象（低保、特困、低保边缘户等）的个人隐私信息，明确告知离职后泄露数据的法律责任。
3.  **情绪疏导：** 对于因工作压力或被动离职的员工，进行必要的情绪安抚，防止负面情绪带入交接工作或在服务对象中散布不当言论，维护政府购买服务项目的形象。

**（二） 离职风险评估**

在审批过程中，行政人事部将会同项目经理进行风险评估，形成《离职风险评估报告》：

1.  **业务中断风险：** 评估该员工负责的片区是否存在正在进行的入户核查任务、急难型临时救助个案，确保这些紧急事项优先交接。
2.  **数据安全风险：** 评估该员工掌握的系统账号权限级别（如省/市大救助信息系统、核对系统），确定是否需要技术部门提前介入进行操作日志审计。
3.  **劳动纠纷风险：** 梳理加班费、年休假、社保缴纳等情况，确保结算合法合规，避免后续法律纠纷影响项目声誉。

**三、 岗位工作交接清单详解**

这是本制度的核心部分。针对社会救助项目的特殊性，菲尔德咨询设计了“四维立体交接清单”，涵盖实物资产、纸质档案、电子数据及业务进度，实行“清单式管理，签字背书，终身追责”。

**（一） 维度一：实物资产与办公设备交接**

社会救助工作涉及大量入户调查设备，必须确保完好归还。

1.  **移动终端设备：** 包括入户核查专用的平板电脑（Pad）、录音笔、执法记录仪（如有）、照相机等。交接时需现场开机测试，检查设备功能是否正常，屏幕、外壳有无破损，并确认设备内存储的业务照片已导出并清空。
2.  **办公用品与耗材：** 办公电脑、打印机、扫描仪、U盘、移动硬盘等。重点检查移动存储介质是否含有非工作内容，严禁私自格式化需归档的硬盘。
3.  **证照与印鉴：** 回收员工工牌、入户调查证、各类系统登录U-Key（数字证书）、项目组公章（如由其保管）、介绍信存根等。U-Key的回收必须即时进行注销或密码重置处理。
4.  **其他物资：** 办公桌椅钥匙、文件柜钥匙、项目配备的电动车（如有）、劳保用品等。

**（二） 维度二：纸质档案与文书资料交接**

民政工作讲究“痕迹管理”，纸质档案是审计检查的重中之重。交接必须做到“一户一档，卷宗齐备”。

1.  **救助对象档案：** 移交其负责片区内所有在册低保、特困、低边家庭的“一户一档”资料。包括申请书、授权书、核对报告、入户调查表、民主评议记录、公示照片、审批表等。交接时需按照档案目录逐页核对，确保无缺项、无漏页。
2.  **工作台账：** 移交日常走访记录本、探访关爱台账、信访接待记录、政策宣传记录等过程性资料。
3.  **政策文件汇编：** 移交由公司或民政局下发的各类政策汇编、操作手册、会议纪要等内部资料，严禁带离公司。
4.  **未归档资料：** 对于尚未完成归档流程的零散材料，需分类整理，装入专门的文件袋，并贴上标签注明材料性质和所属对象，防止混淆丢失。

**（三） 维度三：电子数据与系统权限交接**

在数字化改革背景下，电子数据的交接与权限管控是防范信息泄露的关键。

1.  **业务数据移交：** 员工电脑硬盘中的所有工作相关文档（Word、Excel、PPT、PDF等）必须按文件夹结构完整拷贝至公司指定的服务器或移动硬盘。严禁离职员工保留备份。
2.  **系统账号移交：** 涉及“省/市社会救助管理信息系统”、“居民家庭经济状况核对系统”等政务内网账号，交接当日必须由项目管理员进行密码修改或账号注销操作。
3.  **社交账号与群组：** 如员工使用工作微信、QQ或钉钉号对接街道、社区或服务对象，需将账号管理权移交；若是个人账号加入的工作群（如“XX街道救助工作群”），必须在完成工作交接并经主管确认后，主动退出所有工作群组。
4.  **电子地图与数据库：** 移交其绘制的片区困难群众分布图、电子台账数据库等，确保数据的连续性和可编辑性。

**（四） 维度四：待办事项与业务进度交接**

为防止“人走事停”，必须对正在进行的工作进行详细说明。

1.  **在办个案清单：** 列出正在受理中、核对中、公示中的救助申请个案。详细注明每个个案当前的进度、存在的疑点、下一步需跟进的措施及截止时间。
2.  **重点关注对象清单：** 列出辖区内信访高发户、人户分离户、重病重残户等特殊困难群体，说明其家庭最新状况、主要诉求及应对策略。
3.  **未结费用的说明：** 涉及入户调查交通费、通讯费等未报销款项，需整理好票据，填写报销单，作为交接的一部分。
4.  **外部联络人清单：** 提供对接的社区民政专干、网格员、志愿者骨干的联系方式及沟通注意事项。

**四、 交接监查与责任认定流程**

为确保交接清单的真实性和有效性，菲尔德咨询实施严格的监交制度。

**（一） 三方现场监交**

交接过程必须由移交人（离职员工）、接收人（替补员工或临时接管人）和监交人（项目主管或行政人事专员）三方同时在场进行。

1.  **逐项核对：** 对照《员工离职工作交接清单》，逐一清点实物、核查档案、验证账号。
2.  **现场演示：** 对于复杂的电子表格或系统操作，移交人需现场演示，确保接收人听懂、会用。
3.  **签字确认：** 核对无误后，三方在交接清单上签字并按手印。签字即代表责任转移。接收人签字后，即对接收的资产和档案负责；监交人签字代表确认流程合规。

**（二） 异常情况处理**

1.  **资产缺失赔偿：** 如发现实物资产丢失或人为损坏，由离职员工照价赔偿，赔偿款从其末月工资中扣除。
2.  **档案混乱整改：** 如发现档案整理不规范、缺件严重，监交人有权暂停交接，责令离职员工在规定时间内整改完毕方可办理后续手续。
3.  **拒不交接应对：** 对于恶意拒绝交接或交接不清即擅自离岗的员工，公司将暂缓发放其末月工资及离职证明，并保留追究其法律责任的权利。

**五、 针对社会救助项目的保密与脱密管理**

鉴于社会救助工作的敏感性，离职并不意味着保密义务的终结。

**（一） 签署《离职保密承诺书》**

所有离职员工在办理手续时，必须签署《离职保密承诺书》，承诺：
1.  **永久保密：** 对在职期间接触到的所有服务对象隐私信息（姓名、身份证号、家庭住址、病历、银行账号等）承担无限期保密义务。
2.  **数据销毁：** 承诺个人私人设备（手机、家用电脑）中存储的所有工作数据已彻底删除，无留存、无备份。
3.  **竞业限制（特定岗位）：** 对于掌握公司核心技术秘密（如独创的评估模型、数据分析算法）的高级管理人员，需遵守竞业限制协议。

**（二） 脱密期管理**

对于核心涉密岗位（如核对中心驻点人员），实行脱密期管理。
1.  **提前调岗：** 在正式离职前1-3个月，将其调离涉密岗位，从事非密级工作。
2.  **权限回收：** 提前收回核心系统访问权限，切断其接触新增敏感数据的途径。
3.  **审计追踪：** 对其离职前3个月内的系统操作日志、文件下载记录进行专项审计，确保无数据窃取行为。

**六、 财务结算与离职手续终结**

只有在上述所有交接工作全部完成，且经各环节负责人签字确认后，方可进入财务结算环节。

**（一） 财务审核与结算**

1.  **借款清算：** 财务部核查员工是否有备用金借款，必须在离职前全额还清。
2.  **报销冲抵：** 处理员工未报销的合规业务费用。
3.  **薪资核算：** 根据考勤记录、绩效考核结果及是否有扣款项（如资产赔偿），核算末月工资及经济补偿金（如有）。

**（二） 离职证明与档案转移**

1.  **开具证明：** 财务结算无误后，公司出具《离职证明》，客观记录员工在职时间及担任职务，不随意添加主观评价，但也不隐瞒严重违纪事实。
2.  **社保减员：** 在规定时间内办理社保、公积金减员手续。
3.  **档案转出：** 协助员工将个人人事档案转至新单位或人才交流中心。

**七、 离职后续跟踪与人才回流机制**

菲尔德咨询视离职员工为潜在的资源，建立离职员工关系管理体系。

**（一） 离职后30日回访**

由项目主管在员工离职后30日内进行电话回访，询问交接遗留问题是否已解决，了解其新工作适应情况，传递公司关怀，同时再次提醒保密义务。

**（二） 建立“菲尔德校友录”**

对于表现优秀、正常离职的员工，纳入公司人才库。
1.  **返聘机制：** 当公司有新项目启动或紧急用人需求时，优先联系有经验的离职员工返聘，减少培训成本。
2.  **行业合作：** 保持与在民政领域其他单位就职的离职员工的良性互动，促进业务交流与合作。

**八、 突发性离职应急预案**

针对项目中可能出现的关键岗位突发性离职（如生病、事故、失联），菲尔德咨询建立了“AB角互备与应急替补机制”。

**（一） AB角互备制度**

在日常管理中，每个关键岗位均设置A角（主责）和B角（备岗）。
1.  **日常同步：** A角与B角每周进行一次工作通气，共享关键信息和进度。
2.  **无缝顶替：** 当A角突发离职无法进行正常交接时，B角能立即顶上，确保业务不停摆。

**（二） 应急突击队介入**

公司总部设有“机动社工突击队”。一旦项目发生突发性离职且B角无法完全覆盖时，总部在24小时内派遣资深社工进驻项目点，接管离职员工的工作，直至招聘到新的专职人员。

**（三） 强制接管程序**

对于失联或拒绝配合的员工，公司将启动强制接管程序：
1.  **权限冻结：** 立即通知IT部门和民政局相关科室，强制注销其所有系统账号。
2.  **资产封存：** 在第三方见证下（如街道办工作人员或律师），强行打开其办公柜，清点并封存相关资料，全程录像留证。
3.  **法律追偿：** 发送律师函，追究其因未履行交接义务给项目造成的损失。

**九、 附则**

本制度作为菲尔德咨询《企业管理制度汇编》的重要组成部分，适用于公司承接的所有社会救助服务项目。

**（一） 制度培训**
新员工入职培训时，必须学习本制度，并签字确认知晓离职交接的责任与义务。

**（二） 持续优化**
公司将根据劳动法律法规的更新及项目执行中的实际案例，每年对本制度进行一次修订与完善，确保其时效性与适用性。
