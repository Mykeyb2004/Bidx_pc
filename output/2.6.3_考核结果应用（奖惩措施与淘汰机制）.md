# 2.6.3 考核结果应用（奖惩措施与淘汰机制）

#### 2.6.3 考核结果应用（奖惩措施与淘汰机制）

杭州菲尔德经济信息咨询有限公司（以下简称“菲尔德咨询”或“我司”）深知，在民政社会救助服务项目中，科学、严谨的考核体系是保障服务质量的“生命线”，而考核结果的有效应用则是落实这一生命线的“关键一环”。为了确保本项目“精准救助、温情服务、合规高效”的目标达成，我司将建立一套“奖惩分明、优胜劣汰、双向激励、闭环提升”的考核结果应用体系。本体系不仅关注对服务人员的约束与激励，更关注服务效能的持续改进，确保每一笔财政资金都花在刀刃上，每一项救助政策都落实到困难群众的心坎里。

**一、 考核结果应用总体原则**

我司在执行本项目过程中，对考核结果的应用将严格遵循以下四大原则，确保管理机制的公信力与执行力：

**（一） 刚性兑现与柔性关怀相结合**
考核结果一经确认，必须严格按照制度执行，绝不搞“人情分”、绝不搞“下不为例”，确保制度的刚性与权威。同时，在处理末位人员或违规人员时，注重沟通方式与帮扶指导，体现民政工作的“柔性关怀”，帮助员工查找不足，提供改进机会。

**（二） 物质激励与精神激励相结合**
在考核结果应用中，既要有直接的薪酬绩效体现，发挥经济杠杆的调节作用；又要有荣誉表彰、职业晋升等精神层面的激励，增强项目团队的职业荣誉感和归属感，特别是针对社会救助领域工作强度大、情绪压力大的特点，精神激励尤为重要。

**（三） 个人绩效与团队绩效相结合**
考核结果不仅应用于个人，也应用于项目组、小组等团队单位。通过设定团队目标考核，促进项目内部的协作互助，避免出现“各扫门前雪”的现象，形成“比学赶超”的良性竞争氛围。

**（四） 过程控制与结果导向相结合**
考核结果的应用不仅针对最终的服务产出（如核查报告准确率），也针对服务过程（如入户态度、着装规范、访谈技巧）。通过全过程的考核应用，确保服务质量的每一个环节都处于受控状态。

**二、 考核结果与薪酬绩效挂钩机制**

我司将推行“全面绩效管理”模式，将项目人员的月度薪酬结构划分为“基本工资（70%）+ 绩效工资（30%）”，对于核心管理岗位，绩效工资占比可提升至40%，真正实现“干多干少不一样，干好干坏不一样”。

**（一） 月度绩效考核应用**
根据每月度进行的“服务质量量化考核”得分（百分制），对当月绩效工资进行核算发放：
1. **优秀等级（95分及以上）：** 全额发放绩效工资，并根据项目预算结余情况，额外发放当月绩效基数10%-20%的“服务之星”奖金。
2. **良好等级（85分-94分）：** 全额发放绩效工资。
3. **合格等级（70分-84分）：** 发放绩效工资的80%，扣除部分纳入项目组“奖励基金池”，用于奖励优秀员工。
4. **不合格等级（70分以下）：** 扣发当月全部绩效工资，并由项目督导进行诫勉谈话，签署《绩效改进承诺书》。

**（二） 年度绩效考核应用**
年度考核结果由12个月度考核成绩加权平均得出，结合年度民主评议和采购方（民政局/街道）满意度评价，作为年终奖金分配的核心依据：
1. **年度A级（前15%）：** 年终奖金系数为1.5，优先推荐参与公司年度“金牌社工”、“优秀项目经理”评选，并作为下一年度薪酬职级晋升的必要条件。
2. **年度B级（中间70%）：** 年终奖金系数为1.0，享受正常的年度工龄工资普调。
3. **年度C级（后15%）：** 年终奖金系数为0.5，取消下一年度薪酬晋升资格，并列入“重点观察名单”。

**三、 奖励措施：多维激励体系**

为了激发项目团队的主观能动性，特别是在入户核查任务繁重、面对困难群众诉求复杂的环境下保持工作热情，菲尔德咨询设计了“三位一体”的奖励体系。

**（一） 专项业务奖励（物质层面）**
针对社会救助服务的核心痛点与难点，设立专项奖金，精准激励：
1. **“火眼金睛”奖（核查精准奖）：** 凡在入户核查或信息比对中，通过细致观察、深入访谈，成功发现隐性收入、隐性财产（如车产、房产、经商办企业等关键线索），避免了“错保”发生的，每核实一户，给予核查员现金奖励。此举旨在鼓励工作人员从“走过场”向“实质核查”转变。
2. **“主动发现”奖：** 在日常巡访或接访中，主动发现符合救助条件但未申请的困难群众（即“漏保”群体），并协助其成功纳入救助范围的，每协助一户，给予专项奖励。这体现了从“人找政策”到“政策找人”的服务转型。
3. **“金笔杆”奖（优秀文书奖）：** 针对入户调查报告、个案服务记录撰写质量极高、逻辑严密、证据链完整的，被采购方作为“样板档案”展示的，给予季度奖励。

**（二） 职业发展奖励（成长层面）**
我司为本项目员工打通了“管理+专业”双通道晋升路径，考核结果是晋升的唯一“通行证”：
1. **职级晋升：** 连续两年年度考核为A级的初级核查员，可破格晋升为高级核查员或项目主管，享受相应职级待遇。
2. **培训深造：** 考核优秀的员工，将由公司全额资助参加社会工作师考试培训、民政部/省厅举办的社会救助业务骨干培训班，或前往先发地区进行考察交流。
3. **人才库入选：** 考核前10%的员工，自动纳入菲尔德咨询“储备干部人才库”，未来公司承接的其他大型民政项目，优先从该库中选拔项目负责人。

**（三） 荣誉表彰奖励（精神层面）**
1. **月度评优：** 每月评选“服务之星”、“最美核查员”，照片及事迹在项目驻点办公室及公司公众号展示。
2. **年度盛典：** 在公司年会上，邀请采购方领导为年度优秀员工颁奖，颁发荣誉证书及奖杯，并邀请家属出席，增强员工家属对民政工作的理解与支持。
3. **典型选树：** 挖掘考核优秀员工的感人服务案例，整理成册，向省、市级媒体推介，树立行业标杆，提升员工的社会认可度。

**四、 惩罚措施：分级预警与问责**

为了严肃工作纪律，规范服务行为，防范廉政风险，我司制定了严格的分级惩罚机制。惩罚不是目的，而是为了及时纠偏，防止小错酿成大祸。

**（一） 黄色预警（轻微违规）**
**适用情形：**
1. 上班迟到早退月累计3次以内；
2. 着装不规范、未佩戴工牌；
3. 入户核查表格填写有非关键性缺项或字迹潦草；
4. 档案整理归档不及时，滞后时间在2个工作日以内；
5. 服务态度一般，被服务对象口头投诉（经查实情节轻微）。

**处理措施：**
1. **口头警告：** 由项目经理进行口头批评，并记录在案。
2. **限期整改：** 要求当事人即时纠正错误行为。
3. **绩效扣分：** 每次扣除当月绩效分值1-2分。

**（二） 橙色预警（一般违规）**
**适用情形：**
1. 无故缺勤或擅离职守；
2. 入户核查存在“蜻蜓点水”，未按规定程序询问关键信息；
3. 调查报告逻辑混乱，关键数据（如收入、财产）录入错误；
4. 档案丢失非关键性材料；
5. 对服务对象态度生硬、推诿扯皮，引发有效投诉；
6. 未经批准，延误采购方交办的紧急任务。

**处理措施：**
1. **通报批评：** 在项目组内部发布通报批评文件。
2. **书面检讨：** 当事人需提交不少于1000字的情况说明及整改检查。
3. **绩效扣除：** 扣除当月绩效工资的30%-50%。
4. **暂停工作：** 视情节轻重，责令停职反省1-3天，停职期间无薪资。

**（三） 红色预警（严重违规）**
**适用情形：**
1. 弄虚作假，编造入户核查记录、伪造当事人签字、伪造佐证材料（这是民政核查工作的底线）；
2. 泄露服务对象隐私信息，造成不良社会影响；
3. 利用工作之便，索取、收受服务对象财物，或接受吃请；
4. 与服务对象发生肢体冲突或严重言语辱骂，引发舆情；
5. 连续两个月绩效考核不合格。

**处理措施：**
1. **立即停职：** 立即停止当事人一切工作权限，封存其经手的档案资料。
2. **全额扣款：** 扣发当月全部薪资及年度奖金。
3. **行政降级/解除合同：** 视情节严重程度，给予降级降薪处理或直接解除劳动合同（详见淘汰机制）。
4. **法律追责：** 若涉及违法犯罪（如贪污侵占救助金、出售公民个人信息），依法移送司法机关处理。

**五、 熔断与淘汰机制**

在社会救助招投标服务中，人员的稳定性固然重要，但对于不胜任、不合规的人员，必须建立畅通的退出机制，以保障团队的整体战斗力和纯洁性。菲尔德咨询承诺执行以下“末位淘汰”与“红线熔断”机制。

**（一） 末位淘汰机制（能力不足型）**
针对工作态度尚可但业务能力长期无法满足岗位需求的员工，启动末位淘汰程序：
1. **认定标准：** 年度绩效考核排名倒数第一，或连续三个月月度考核排名倒数第一且得分低于70分的员工。
2. **PIP改进计划：** 在直接淘汰前，给予最后一次机会。由公司资深督导制定为期1个月的“绩效改进计划”（PIP），进行一对一辅导。
3. **考核验收：** PIP期满后进行专项考核。若考核通过，留岗试用；若仍不合格，则依据《劳动合同法》相关规定，进行劝退或调岗处理，调离本项目组。
4. **人员替补：** 一旦启动淘汰程序，我司承诺在3个工作日内从公司人才储备库中调派同等资质或更高资质的人员到岗接替，确保工作不断档。

**（二） 红线熔断机制（严重过失型）**
针对触犯法律法规、职业道德底线或严重违反采购方规定的行为，实行“一票否决”，立即启动熔断淘汰：
1. **触发条件：**
    *   发生“人情保”、“关系保”等严重违规违纪行为；
    *   因工作失职导致重大信访事故或省级以上媒体负面曝光；
    *   严重违反保密协议，造成数据外泄；
    *   无正当理由拒不执行采购方或公司重大决策部署，造成严重后果。
2. **执行流程：**
    *   **取证锁定（2小时内）：** 项目经理联合公司质控部，迅速锁定违规证据，控制事态发展。
    *   **通报除名（24小时内）：** 下达解除劳动合同通知书，并在全公司范围内通报，列入行业黑名单。
    *   **责任追究（后续）：** 依据造成的经济损失或声誉损失，向当事人发起民事索赔。
    *   **紧急补位（24小时内）：** 按照“AB角”配置原则，由备选人员立即顶岗，并向采购方提交事故处理报告及整改措施。

**六、 考核结果的闭环管理与申诉机制**

考核不是为了“整人”，而是为了“正事”。为了确保考核结果应用的公正性与建设性，我司构建了完整的闭环管理流程。

**（一） 绩效面谈与改进计划**
每月考核结果出炉后，项目经理必须与所有扣分员工进行“一对一”绩效面谈。
1. **谈话内容：** 指出具体扣分点，分析问题产生的根源（是态度问题、能力问题还是流程问题），听取员工的解释与困难。
2. **制定对策：** 双方共同制定下个月的改进目标和具体措施，形成《月度绩效改进表》，双方签字确认，作为下月考核的对照依据。
3. **培训跟进：** 针对共性扣分问题（如政策理解偏差），由公司培训部组织专项补课，实现“考核-反馈-培训-提升”的良性循环。

**（二） 申诉处理机制**
为了防止考核中出现主观偏见或误判，保障员工合法权益，设立畅通的申诉渠道：
1. **申诉受理：** 员工对考核结果有异议的，可在收到结果后的3个工作日内，向公司总部的“质量控制部”或“工会委员会”提起书面申诉。
2. **独立调查：** 质控部在接到申诉后，成立独立调查小组（避开项目直接负责人），通过复查档案、调阅录音、回访服务对象等方式进行核实。
3. **结果裁定：** 调查小组在5个工作日内出具复核意见。若申诉属实，立即纠正考核结果，补发扣除的绩效工资，并对考核执行人进行问责；若申诉不实，维持原判，并向员工做好解释工作。

**（三） 采购方反馈联动**
我司将定期（每季度）向采购方提交《项目人员绩效考核与团队优化报告》。
1. **信息透明：** 向采购方通报项目组人员的考核情况、奖惩情况及人员变动情况，确保采购方对服务团队状态的知情权。
2. **接受监督：** 采购方有权建议更换考核不合格或服务态度恶劣的工作人员，我司承诺无条件配合，并在规定时间内完成人员更换。
3. **政策建议：** 结合考核中发现的共性业务难点（如某类政策界定模糊导致核查难），向采购方提出政策优化的合理化建议，体现智库型服务机构的价值。

**七、 总结**

综上所述，杭州菲尔德经济信息咨询有限公司通过构建这一套严密、科学、人性化的考核结果应用体系，旨在打造一支“政治过硬、业务精通、作风优良”的社会救助专业服务铁军。我们将以考核为指挥棒，以奖惩为助推器，坚决剔除“南郭先生”，大力培育“行业工匠”，确保本项目在严格的质量控制下高效运行，为辖区内的困难群众提供最有温度、最有力度、最精准的救助服务，向采购方交出一份满意的答卷。
