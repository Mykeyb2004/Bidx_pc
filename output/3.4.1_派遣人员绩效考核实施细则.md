# 3.4.1 派遣人员绩效考核实施细则

#### 3.4.1 派遣人员绩效考核实施细则

为确保杭州菲尔德经济信息咨询有限公司（以下简称“菲尔德咨询”或“我司”）承接的民政社会救助服务项目能够高效、规范、廉洁运行，切实提升派遣人员的专业素养与服务质量，根据《社会救助暂行办法》及采购单位相关管理规定，结合本项目“政策性强、敏感度高、服务面广”的特点，特制定本绩效考核实施细则。

本细则旨在构建一套“导向清晰、指标量化、程序公正、奖惩分明”的闭环考核体系，将考核结果作为派遣人员薪酬分配、岗位调整、评优评先及续聘解聘的根本依据，从而实现社会救助服务从“粗放式管理”向“精细化绩效治理”的转变。

一、 考核原则与目标

（一） 考核原则
1. **客观公正原则：** 坚持以事实为依据，以数据为支撑，杜绝主观臆断与人情分，确保每一项扣分都有据可查，每一项加分都实至名归。
2. **多维立体原则：** 采取“定性与定量相结合、过程与结果相结合、内部评价与外部反馈相结合”的方式，实现对派遣人员全方位、全周期的精准画像。
3. **责权利对等原则：** 考核结果与个人切身利益直接挂钩，打破“大锅饭”，真正体现“多劳多得、优劳优得、少劳少得、不劳不得”。
4. **动态优化原则：** 根据民政社会救助政策的调整及项目实际运行情况，适时调整考核指标权重，确保持续适应基层救助工作的新要求。

（二） 考核目标
1. **提升业务规范性：** 确保低保、特困、临时救助等各类对象的受理、核查、审批流程符合法定程序，档案合格率达到100%。
2. **增强服务精准度：** 通过严格的入户核查考核，确保“不错保、不漏保、不骗保”，精准认定救助对象。
3. **强化廉洁自律性：** 筑牢廉政防火墙，杜绝吃拿卡要、优亲厚友等违规违纪行为，维护民政工作公信力。
4. **提高群众满意度：** 优化服务态度，提升困难群众的获得感与幸福感，确保零有效投诉。

二、 考核组织架构

为保障考核工作的权威性与执行力，菲尔德咨询将成立“项目绩效考核领导小组”，全面负责考核制度的顶层设计、统筹协调与监督落实。

（一） 组织构成
1. **组长：** 由项目总负责人担任，对考核工作负总责，拥有最终裁定权。
2. **副组长：** 由项目督导及人事经理担任，负责具体考核方案的执行与复核。
3. **组员：** 由各街道/片区组长、质控专员及财务人员组成，负责数据的收集、汇总与初评。

（二） 职责分工
1. **项目督导：** 负责日常业务巡查，记录派遣人员的工作纪律与业务操作情况，每月出具业务考核报表。
2. **质控专员：** 负责按比例抽查入户调查表、系统录入数据及纸质档案，对发现的逻辑错误、缺项漏项进行量化扣分。
3. **片区组长：** 负责收集服务对象（困难群众）及采购单位（民政局/街道办）的反馈意见，作为第三方评价依据。

三、 考核指标体系

本细则采用百分制考核法，结合KPI（关键绩效指标）与360度评估模型，构建包含“德、能、勤、绩、廉”五个维度的指标体系。其中，月度考核侧重于执行力与工作量，年度考核侧重于综合素质与成长性。

（一） 关键绩效指标（KPI）分解（总分100分）

**1. 业务绩效维度（权重40分）**
此维度为考核核心，重点评估派遣人员在社会救助经办服务中的产出质量与效率。
（1） **入户核查率（10分）：** 要求对新增申请对象100%入户，对在册对象按规定频次复核。
*   考核标准：每遗漏一户扣5分；入户轨迹（GPS打卡）与记录不符，视为虚假入户，该项不得分并通报批评。
（2） **档案规范性（10分）：** 重点考核“一户一档”的完整性、逻辑性。
*   考核标准：档案内申请书、身份证复印件、核对报告、入户调查表、公示照片、会议记录等材料缺一不可。每发现一处缺失或逻辑错误（如收入计算错误、家庭人口与户口本不符）扣2分。
（3） **系统录入准确率（10分）：** 省/市社会救助信息系统数据录入情况。
*   考核标准：录入信息与纸质档案不一致，每处扣1分；未在规定时限内完成录入导致审批延误，每例扣5分。
（4） **时效性管理（10分）：** 各类救助业务办理的时限控制。
*   考核标准：低保审核确认未在30个工作日内完成（特殊情况除外），临时救助未在规定时间内办结，每逾期一天扣2分。

**2. 工作能力维度（权重20分）**
重点评估派遣人员的政策掌握程度与实务操作能力。
（1） **政策知晓度（10分）：** 熟悉低保、特困、低保边缘家庭、支出型贫困等认定条件及计算方法。
*   考核方式：每月组织一次闭卷业务测试，卷面分即为该项得分（折算后）；或在日常抽查中现场提问，回答不上来每次扣2分。
（2） **沟通协调能力（5分）：** 在入户调查及接待群众过程中的沟通技巧。
*   考核标准：能够准确解释政策，安抚情绪激动群众。若因解释不清导致群众误解或引发矛盾，每次扣2分。
（3） **文书撰写能力（5分）：** 入户调查报告、民主评议记录等文书的质量。
*   考核标准：描述客观、详实，能够反映家庭真实生活状况。若描述过于简单（如仅写“生活困难”四个字）或存在明显复制粘贴痕迹，每次扣1分。

**3. 工作态度与纪律（权重20分）**
（1） **考勤管理（10分）：** 上下班打卡及在岗情况。
*   考核标准：迟到、早退一次扣1分；无故旷工一次扣5分；工作时间从事与工作无关事务（如玩游戏、网购）发现一次扣2分。
（2） **服从意识（5分）：** 对公司及采购单位工作安排的执行情况。
*   考核标准：无正当理由拒绝执行上级或采购单位交办的任务，该项不得分；执行拖沓、推诿扯皮，每次扣2分。
（3） **团队协作（5分）：** 与同事配合情况及经验分享。
*   考核标准：制造团队矛盾、破坏团结者扣3-5分；积极参与团队疑难个案研讨并提供有效建议者，酌情加分。

**4. 服务满意度（权重10分）**
（1） **群众评价（5分）：** 通过电话回访或意见箱收集群众反馈。
*   考核标准：收到群众表扬信或锦旗，每面加2分；因服务态度恶劣被群众投诉并查实，该项不得分，并追加扣分。
（2） **业主评价（5分）：** 街道/乡镇民政科室负责人的评价。
*   考核标准：由业主方按月打分，直接计入考核成绩。

**5. 廉洁自律（权重10分，一票否决项）**
（1） **信息保密（5分）：** 严格保护救助对象隐私。
*   考核标准：泄露救助对象个人信息（如在微信群转发含身份证号的表格），一次扣5分，造成严重后果的解除劳动合同。
（2） **廉洁从业（5分）：** 严禁收受礼品礼金。
*   考核标准：收受申请人香烟、红包、土特产等，一经查实，考核分归零，直接辞退并移交相关部门处理。

（二） 加分项（上限10分）
1. **主动发现：** 在日常巡查中主动发现“沉默的少数”困难群众，并成功协助其获得救助，每例加2分。
2. **宣传报道：** 撰写的救助案例或工作经验被区级以上媒体或部门录用，每篇加3分。
3. **突发贡献：** 在抗击疫情、防汛抗旱等突发事件中表现突出，获得通报表扬，加5分。

四、 考核实施周期与流程

菲尔德咨询实施“月度考核、季度评议、年度总评”的三级考核机制，确保考核工作的常态化与长效化。

（一） 月度考核（挂钩绩效工资）
1. **数据采集（每月25日前）：** 由质控专员汇总当月考勤记录、业务系统数据、档案抽查记录及投诉记录。
2. **自评与互评（每月26-27日）：** 派遣人员填写《月度绩效自评表》，并在小组内进行简短互评。
3. **主管评分（每月28日）：** 片区组长/督导结合数据与日常表现，进行打分。
4. **结果公示与反馈（每月30日前）：** 考核结果在内部进行“点对点”反馈。员工对结果有异议的，可在2个工作日内向项目经理申诉。

（二） 季度评议（挂钩季度奖金/评优）
1. **业主满意度调查：** 每季度末，由项目经理拜访各街道民政科室，发放《派遣人员服务质量征询单》，听取采购单位意见。
2. **业务能力测试：** 组织一次全员业务知识竞赛或实操演练，检验专业技能提升情况。
3. **综合评定：** 结合三个月的月度考核平均分与季度专项测评，形成季度绩效等级。

（三） 年度总评（挂钩续聘/晋升）
1. **述职报告：** 派遣人员提交年度个人工作总结，进行现场述职。
2. **360度评估：** 邀请采购单位代表、服务对象代表、公司管理层共同参与评估。
3. **结果汇总：** 年度考核分 = 月度考核平均分×60% + 季度考核平均分×20% + 年终评估×20%。

五、 考核结果分级与应用

考核结果不仅仅是一个分数，更是菲尔德咨询进行人力资源优化的核心依据。我们将考核结果划分为四个等级，并实施差异化的激励与约束措施。

（一） 等级划分
1. **A级（优秀）：** 考核得分≥95分，且无任何违规记录。比例原则上不超过团队总人数的20%。
2. **B级（良好）：** 95分＞考核得分≥80分。
3. **C级（合格）：** 80分＞考核得分≥60分。
4. **D级（不合格）：** 考核得分＜60分，或触犯“一票否决”红线。

（二） 结果应用细则

**1. 薪酬激励机制**
（1） **绩效工资发放：**
*   A级：全额发放当月绩效工资，并额外发放绩效系数1.2倍的奖励。
*   B级：全额发放当月绩效工资。
*   C级：发放当月绩效工资的80%，剩余20%作为观察期保证金。
*   D级：扣发当月全部绩效工资。
（2） **年终奖金分配：** 根据年度总评得分，A级员工年终奖系数为1.5，B级为1.0，C级为0.5，D级不享受年终奖。

**2. 职业发展机制**
（1） **晋升通道：** 连续两个季度考核为A级，或年度考核为A级的派遣人员，列入“储备干部库”，优先推荐晋升为片区组长、质控专员或项目助理。
（2） **评优评先：** 年度“优秀社工”、“服务之星”等荣誉称号，仅从A级员工中产生。

**3. 培训与改进机制**
（1） **针对性培训：** 对考核中发现的业务短板（如文书写作弱、政策理解偏），由公司安排专项补强培训。
（2） **绩效面谈：** 对月度考核为C级的员工，由项目督导进行“一对一”绩效改进面谈，制定《绩效改进计划书》（PIP），明确改进目标与期限（通常为1个月）。

**4. 退出与淘汰机制**
（1） **末位淘汰：** 连续两个月考核为D级，或连续三个月考核为C级且经培训无明显改善者，公司有权将其退回或解除劳动合同。
（2） **红线退出：** 凡触犯以下“红线”之一者，直接评定为D级并立即终止服务，且永不录用：
*   利用职务之便索取、收受服务对象财物的；
*   弄虚作假，协助不符合条件人员骗取救助资金的；
*   因严重失职导致救助资金重大流失或造成重大社会负面舆情的；
*   无故旷工连续3天或一年内累计旷工7天以上的。

六、 申诉与监督机制

为防止考核过程中出现偏颇、报复或误判，保障每一位派遣人员的合法权益，菲尔德咨询设立独立的申诉与监督渠道。

（一） 申诉渠道
派遣人员如对考核结果、扣分依据或评价内容有异议，可在收到考核结果通知后的3个工作日内，通过书面形式或公司内部OA系统向“项目绩效考核领导小组”提起申诉。申诉材料应包含申诉理由及相关证明材料（如工作记录截图、通话录音、服务对象证言等）。

（二） 复核流程
1. **受理：** 考核小组收到申诉后，应在1个工作日内予以受理，并指定非原考核人的第三方人员（如公司总部人事专员）介入调查。
2. **调查：** 调查人员通过调阅原始档案、核查系统日志、访谈相关人员等方式，对争议事项进行复核。
3. **裁决：** 调查完成后，考核小组召开复议会议，根据调查事实做出维持原判、调整分数或撤销处罚的决定。
4. **反馈：** 复核决定将在受理后5个工作日内书面反馈给申诉人。该决定为最终裁决。

（三） 监督管理
公司总部将不定期对项目组的考核工作进行飞行检查，重点核查考核记录是否真实、评分是否合规、奖惩是否兑现。对于在考核工作中徇私舞弊、打击报复的管理人员，一经发现，严厉追责，确保考核机制的纯洁性与公正性。

七、 附则

1. **档案管理：** 所有绩效考核原始数据、评分表、面谈记录、改进计划及申诉材料，均由公司人事行政部统一归档保存，保存期限不少于3年，以备采购单位及审计部门查阅。
2. **制度修订：** 本细则将根据本项目实际运行情况及采购单位的最新要求进行动态修订。修订后的版本经公示后生效。
3. **解释权：** 本细则由杭州菲尔德经济信息咨询有限公司负责解释。
