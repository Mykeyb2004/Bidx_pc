# 7.1.1 保密工作领导小组

为深入贯彻落实《中华人民共和国网络安全法》、《中华人民共和国数据安全法》、《中华人民共和国个人信息保护法》等国家法律法规，以及民政部、省、市关于加强社会救助领域数据安全和个人信息保护的系列文件精神，深刻认识到本项目所涉社会救助对象信息的极端敏感性与保密工作的极端重要性，我司承诺，将构建一个权责清晰、运转高效、保障有力的保密工作领导小组，作为本项目数据安全与保密工作的最高决策与管理机构，确保项目全生命周期内所有数据的机密性、完整性与可用性，为项目顺利实施提供最坚实的组织保障。

**一、 成立宗旨与工作原则**

（一）成立宗旨
保密工作领导小组的成立，旨在从组织架构的顶层设计上，将数据安全与保密责任提升至公司战略高度。其核心宗旨是：统一领导、统一部署、统一协调本项目所有涉及敏感信息的采集、存储、处理、流转、使用和销毁等环节的保密工作；建立并完善覆盖全员、全过程、全方位的保密管理体系；有效预防、及时发现、妥善处置各类数据安全风险与泄密事件；确保服务对象的个人隐私和信息安全得到最高级别的保护，维护政府公信力与我司专业声誉，坚决杜绝因我方原因导致任何救助对象信息泄露事件的发生。

（二）工作原则
领导小组的工作将严格遵循以下四大原则：
1.  **合法合规，依法治理原则。** 严格遵守国家及地方关于数据安全、个人信息保护及政府信息公开等相关法律法规，所有保密制度、流程和技术措施的设计与执行均以法律为准绳，确保项目运作的每一个环节都在法治轨道内进行。
2.  **预防为主，综合防范原则。** 坚持“安全始于设计，贯穿于运营”的理念，将保密要求融入项目方案设计、系统开发、人员管理和业务流程的各个方面，构建“事前预防、事中监控、事后追溯”的纵深防御体系，从源头上降低安全风险。
3.  **分级管理，突出重点原则。** 根据社会救助数据的敏感程度，对数据进行科学的分类分级管理。对涉及救助对象身份、财产、健康、家庭关系等核心敏感信息，实行最高等级的保护措施，明确接触权限，严控知悉范围，确保“非授权、不访问”。
4.  **明确责任，严肃追责原则。** 建立健全的保密工作责任制，将保密责任层层分解，落实到岗、落实到人。小组组长为第一责任人，各成员分工负责，签订保密责任书。对于违反保密规定的行为，无论职位高低、主观故意或过失，均启动问责程序，严肃处理，绝不姑息。

**二、 组织架构与成员组成**

为确保领导小组的权威性、专业性与执行力，我司将组建一个“一长两副多成员、下设办公室”的矩阵式管理架构。

（一）领导小组架构
1.  **组长（1名）：** 由公司总经理或主管该业务线的副总经理担任。作为保密工作的第一责任人，对本项目的整体数据安全与保密工作负总责，拥有最终决策权和否决权。
2.  **常务副组长（1名）：** 由本项目总负责人担任。作为保密工作的总执行官，全面负责领导小组的日常运作，统筹协调各方资源，组织实施领导小组的各项决议，直接对组长负责。
3.  **技术副组长（1名）：** 由公司信息技术总监或资深技术专家担任。作为技术安全的主要负责人，专注于技术层面的安全防护体系建设、运维和应急处置，为保密工作提供强大的技术支撑。
4.  **成员（若干名）：** 由项目核心部门负责人和关键岗位人员组成，包括但不限于：
    *   项目执行部经理
    *   数据管理与分析专员
    *   一线服务团队督导
    *   法务与合规专员
    *   人力资源部培训专员
    *   客户服务与投诉处理专员

（二）领导小组办公室
在领导小组下设立“保密工作领导小组办公室”（以下简称“保密办”），挂靠在项目管理部，作为领导小组的常设办事机构和执行枢纽。办公室主任由项目执行部经理兼任，并配备1-2名专职或兼职保密干事，负责处理日常事务，确保领导小组各项工作指令的上传下达和闭环落实。

**三、 主要职责与任务分工**

领导小组及其成员的职责分工明确，各司其职，协同作战，形成强大的管理合力。

（一）领导小组整体职责
1.  **战略规划与决策：** 审定本项目整体的保密工作战略、方针和中长期规划；对涉及数据安全的重大事项、重大投资和重大风险处置方案进行集体研究并作出最终决策。
2.  **制度体系建设：** 批准并颁布本项目的数据安全与保密管理系列制度，包括但不限于《项目保密管理办法》、《员工保密行为准则》、《数据分类分级标准》、《涉密信息处理流程》、《应急响应预案》等，并监督其执行情况。
3.  **资源统筹与保障：** 审批保密工作所需的专项预算，确保在人员配备、技术设备、专业培训、安全审计等方面的投入充足、到位，为保密工作提供坚实的物质基础。
4.  **监督检查与考核：** 定期听取保密办和各专项负责人的工作汇报，组织开展全面的保密工作检查和风险评估，审定年度保密工作考核结果，并将考核结果与相关部门及人员的绩效挂钩。
5.  **应急指挥：** 在发生重大或特别重大数据安全事件时，立即启动最高级别应急响应，担任应急指挥部总指挥，统一指挥、协调各方力量进行应急处置，最大限度降低事件带来的损失和负面影响。

（二）组长职责
1.  全面领导本项目的保密工作，确保公司对项目数据安全的承诺得以兑现。
2.  主持召开领导小组全体会议，听取汇报，研究问题，部署工作。
3.  签发领导小组的重要文件、制度和决议。
4.  在重大、紧急情况下，代表领导小组行使决策权。
5.  承担保密工作第一责任人的政治责任、领导责任和法律责任。

（三）常务副组长职责
1.  组织起草、修订项目保密工作各项规章制度和工作计划，并推动落实。
2.  主持召开领导小组办公室会议和专题协调会，解决日常工作中遇到的具体问题。
3.  负责组织对全体项目人员的保密教育和培训工作，提升全员保密意识。
4.  负责组织开展定期的保密自查、风险排查和内部审计工作。
5.  负责与采购人（招标方）就保密工作进行日常沟通、协调与汇报。

（四）技术副组长职责
1.  负责规划、设计、建设和维护本项目的技术安全防护体系，确保网络、系统、数据库等基础设施的安全可靠。
2.  负责制定并实施数据加密、访问控制、安全审计、漏洞扫描、入侵检测等技术安全策略。
3.  负责对新技术、新应用在项目中的引入进行安全评估。
4.  负责数据安全事件的技术分析、溯源和处置，组织技术力量进行系统修复和加固。
5.  跟踪业界最新的安全威胁和技术动态，持续优化项目技术防护能力。

（五）小组成员职责
1.  **项目执行部经理：** 负责将保密要求融入一线业务流程，监督入户核查、信息录入、档案管理等环节的保密措施执行。
2.  **数据管理与分析专员：** 严格按照分类分级标准管理数据资产，执行数据的脱敏、备份、恢复和销毁操作，监控数据访问日志。
3.  **一线服务团队督导：** 负责对一线社工、调查员进行日常保密监督和提醒，及时发现并纠正不规范操作，收集一线保密风险点并上报。
4.  **法务与合规专员：** 负责审查所有涉密合同、协议，提供法律咨询，确保各项保密措施符合法律法规要求，协助处理法律纠纷。
5.  **人力资源部培训专员：** 负责制定并实施全员保密培训计划，组织新员工岗前保密培训、在岗人员定期轮训，管理员工保密协议的签署与存档。
6.  **客户服务与投诉处理专员：** 负责在处理服务对象咨询和投诉时，严格遵守保密规定，妥善处理涉及个人信息的问询，并记录相关情况。

**四、 工作机制与运行保障**

为确保领导小组高效、规范、持续地发挥作用，我司将建立一套完善的运行保障机制。

（一）会议制度
1.  **领导小组全体会议：** 每季度召开一次，由组长主持。主要内容包括听取季度保密工作总结和下季度计划，审议重要制度，研究重大问题。
2.  **保密办工作例会：** 每月召开一次，由常务副组长主持。主要内容包括通报月度安全态势，协调解决日常工作难题，布置具体工作任务。
3.  **专题协调会议：** 根据工作需要不定期召开，由常务副组长或其委托人主持，针对特定问题（如制度修订、技术升级、事件复盘等）进行深入研究。
4.  **紧急会议：** 发生或可能发生数据安全事件时，立即启动，由组长或其授权的副组长主持，启动应急响应程序。

（二）报告与沟通机制
建立“纵向贯通、横向协同”的信息沟通网络。保密办作为枢纽，负责收集、汇总各成员单位的保密工作信息，形成《保密工作月报》、《风险评估报告》等，定期向领导小组汇报。同时，建立与采购人（招标方）的定期沟通机制，主动汇报项目保密工作进展、成效与存在问题，积极接受监督指导。

（三）督导检查机制
采取“定期自查、随机抽查、专项检查”相结合的方式。领导小组每年至少组织一次全面的保密工作大检查。保密办每季度组织一次覆盖全流程的专项检查。一线督导每日对团队成员的操作行为进行随机抽查。检查结果将以书面形式通报，对发现的问题建立台账，明确整改责任人与时限，并进行跟踪验证，形成闭环管理。

（四）教育培训长效机制
将保密教育贯穿于员工职业生涯全过程。新员工入职，必须接受不少于4小时的保密专题培训，考试合格并签署保密协议后方可上岗。在岗员工每年接受不少于2次的保密再教育和警示教育。针对关键涉密岗位人员，定期组织参加外部专业机构的高级培训。培训内容覆盖法律法规、公司制度、操作规程、技术防范知识和泄密案例分析，确保保密意识入脑入心。

（五）考核与问责机制
制定《项目保密工作考核评价办法》，将保密工作成效纳入部门和个人的年度绩效考核体系，权重不低于15%。对在保密工作中做出突出贡献的集体和个人予以表彰奖励。对违反保密规定，造成信息泄露或安全事件的，严格按照《员工奖惩条例》和保密责任书的约定进行处理，包括但不限于警告、降级、经济处罚、解除劳动合同等；情节严重，构成犯罪的，将依法移交司法机关追究其法律责任。

综上所述，我司设立的“保密工作领导小组”不仅是一个形式上的组织，更是一个实体化运作、权责分明、机制健全的战斗堡垒。我们将以最坚决的态度、最严格的制度、最有力的措施，筑牢社会救助数据安全防线，确保项目在绝对安全的环境下平稳、高效运行，不负采购人的信任与重托，切实维护好每一位困难群众的合法权益与个人尊严。
